### 1. 实现搜索过滤（让搜索框生效）

在 Vue 3 中，我们要“盯着” `filterText` 的变化，一旦用户打字，就让树组件执行过滤逻辑。

**操作：** 在 `<script setup>` 中补充 `watch` 和完善 `filterNode`。

JavaScript

```
import { ref, onMounted, watch } from "vue"; // 记得补充引入 watch

// ... 之前的代码 ...

// 1. 监控搜索框文字变化
watch(filterText, (val) => {
  // treeRef 是你在 <el-tree> 上写的那个 ref 属性
  treeRef.value?.filter(val);
});

// 2. 编写具体的过滤规则 (Java 里的 boolean 判断)
const filterNode = (value, data) => {
  if (!value) return true; // 如果搜索框是空的，显示所有节点
  return data.label.includes(value); // 如果包含搜索词，返回 true
};
```

### 2. 实现双击交互（让 Pinia “接招”）

当用户双击表名时，我们需要调用你之前写好的 `useWorkspaceStore` 里的 `addTab` 方法。

**操作：** 补充 `handleNodeDblClick` 函数。

JavaScript

```
// 双击打开表数据
const handleNodeDblClick = (data) => {
  // 只有双击“表”类型的节点才触发（过滤掉数据库节点）
  if (data.type === 'table') {
    store.addTab({
      id: data.id,
      title: data.label,
      type: 'table',
      params: { tableName: data.label } // 传给右侧，告诉它要查哪张表
    });
  }
};
```

### 3. 实现工具栏按钮（新建控制台）

你的“+”号图标现在点了没反应。我们需要给它加上点击事件。

**操作：**

1. 在模板里给图标加上 `@click="handleNewConsole"`。
    
2. 在脚本里写这个函数。
    

HTML

```
<el-icon class="action-icon" @click="handleNewConsole"><Plus/></el-icon>
```

JavaScript

```
// 脚本部分
const handleNewConsole = () => {
  store.addTab({
    title: 'Console',
    type: 'console'
  });
};
```

### 4. 视觉美化：穿上“IDEA 的西装”

你现在的 CSS 只有一行。为了让它看起来像个专业软件，我们需要处理 Element Plus 的默认样式，把背景调深，文字调淡。

**操作：** 完善 `<style>` 块（建议使用 `:deep` 穿透）。

CSS

```
<style scoped>
/* 整个侧边栏容器 */
.db-sidebar {
  display: flex;
  flex-direction: column;
  height: 100%;
  background-color: #313335; /* IDEA 侧边栏色号 */
}

/* 工具栏样式 */

  display: flex;
  padding: 5px 10px;
  gap: 10px;
  border-bottom: 1px solid #2b2b2b;

.action-icon {
  cursor: pointer;
  color: #afb1b3;
  font-size: 16px;
}
.action-icon:hover { color: #fff; } /* 鼠标悬停变白 */

/* 搜索框容器 */
.sidebar-search { padding: 8px; }

/* 深度修改 el-tree 内部样式 */
:deep(.el-tree) {
  background-color: transparent;
  color: #afb1b3;
}

/* 鼠标划过节点背景色 */

  color: #409eff;
}

  display: flex;
  align-items: center;
  gap: 6px;
  font-size: 13px;
}
</style>
```
