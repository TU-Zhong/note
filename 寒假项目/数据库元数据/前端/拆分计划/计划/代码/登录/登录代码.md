实现登录功能是前后端分离项目中**最经典、最重要**的环节。它不仅仅是一个页面，而是贯穿整个系统的**“通行证机制”**。

考虑到你的毕设背景，我推荐使用 **“Token（令牌）机制”**。这是目前最主流的方案。

我们需要分三步走：**后端造证（生成 Token）** -> **前端领证（存储 Token）** -> **后端查证（拦截器验证）**。

#### 1. 创建简单的 LoginController

为了方便演示，我们暂时用一个 `static Map` 来模拟 Redis 存储 Token（重启后端会失效，但毕设足够了）。

`src/main/java/com/example/dbdesign/controller/LoginController.java`:

Java

```java
package com.example.dbdesign.controller;

import com.example.dbdesign.common.Result;
import org.springframework.web.bind.annotation.*;
import java.util.HashMap;
import java.util.Map;
import java.util.UUID;

@RestController
@RequestMapping("/api")
public class LoginController {

    // ⚡️ 简易版 Token 存储池 (K=Token, V=Username)
    // 真正上线通常用 Redis
    public static Map<String, String> tokenStore = new HashMap<>();

    @PostMapping("/login")
    public Result login(@RequestBody Map<String, String> user) {
        String username = user.get("username");
        String password = user.get("password");

        // 1. 简单校验 (你可以去数据库查 sys_user 表，这里先写死逻辑演示流程)
        // 建议你把这里改成调用 userService.login(username, password)
        if ("admin".equals(username) && "123456".equals(password)) {
            
            // 2. 生成一个令牌 (UUID)
            String token = UUID.randomUUID().toString();
            
            // 3. 存入服务器内存 (记录这个 token 属于 admin)
            tokenStore.put(token, username);

            // 4. 把 token 返给前端
            return Result.success(token);
        } else {
            return Result.error("用户名或密码错误");
        }
    }
    
    @PostMapping("/logout")
    public Result logout(@RequestHeader("Authorization") String token) {
        tokenStore.remove(token);
        return Result.success("退出成功");
    }
}
```

#### 2. 创建“安保人员” (拦截器 Interceptor)

```java
package com.example.dbdesign.config;

import com.example.dbdesign.controller.LoginController;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.springframework.web.servlet.HandlerInterceptor;

public class AuthInterceptor implements HandlerInterceptor {

    @Override
    public boolean preHandle(HttpServletRequest request, HttpServletResponse response, Object handler) throws Exception {
        // 放行 OPTIONS 请求 (跨域预检)
        if ("OPTIONS".equalsIgnoreCase(request.getMethod())) {
            return true;
        }

        // 1. 从 Header 拿 token
        String token = request.getHeader("Authorization");

        // 2. 检查 token 是否有效
        if (token != null && LoginController.tokenStore.containsKey(token)) {
            return true; // ✅ 有证，放行
        }

        // 3. 无证，直接驳回
        response.setStatus(401); // 401 未授权
        response.setContentType("application/json;charset=utf-8");
        response.getWriter().write("{\"code\": 401, \"msg\": \"未登录或登录过期\"}");
        return false;
    }
}
```

#### 3. 注册拦截器 (配置类)

`src/main/java/com/example/dbdesign/config/WebConfig.java`:

Java

```java
package com.example.dbdesign.config;

import org.springframework.context.annotation.Configuration;
import org.springframework.web.servlet.config.annotation.InterceptorRegistry;
import org.springframework.web.servlet.config.annotation.WebMvcConfigurer;

@Configuration
public class WebConfig implements WebMvcConfigurer {

    @Override
    public void addInterceptors(InterceptorRegistry registry) {
        registry.addInterceptor(new AuthInterceptor())
                .addPathPatterns("/**")       // 拦截所有路径
                .excludePathPatterns(         // 排除这些路径 (白名单)
                        "/api/login", 
                        "/error"
                );
    }
}
```

#### 2. 配置路由守卫 (`src/router/index.js`)

你需要告诉 Vue Router：**“没拿到 Token 之前，除了登录页，哪儿也不许去。”**
[[路由]]
JavaScript


```js                                      
import { createRouter, createWebHistory } from 'vue-router'
import Login from '../views/Login.vue'
import MainLayout from '../views/MainLayout.vue' // 假设你的主页叫这个

const router = createRouter({
  history: createWebHistory(),
  routes: [
    {
      path: '/login',
      name: 'Login',
      component: Login
    },
    {
      path: '/',
      name: 'Home',
      component: MainLayout
    }
  ]
})

// 🚧 路由守卫：检查是否有 Token
router.beforeEach((to, from, next) => {
  const token = localStorage.getItem('token')
  
  // 如果要去的地方不是登录页，且没有 token
  if (to.path !== '/login' && !token) {
    next('/login') // 强制踢回登录页
  } else {
    next() // 放行
  }
})

export default router
```
