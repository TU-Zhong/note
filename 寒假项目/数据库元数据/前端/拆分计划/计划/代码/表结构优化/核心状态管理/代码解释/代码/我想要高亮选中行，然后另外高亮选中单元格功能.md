这是一个非常高级且极具 IDE 质感的需求！

在 DataGrip 或 Navicat 中，当你点击一个格子时，**整行会有一个浅色的高亮（表示选中了这行），而你点击的那个具体单元格会有一个更深的边框或底色（表示你的光标在这个格子上）**。

Element Plus 默认只提供了“整行高亮”（`highlight-current-row`），没有提供“单元格高亮”。但我们可以利用 Vue 的动态 Class 机制，自己写一个完美的！

这里有个前端**黄金小技巧**：给单元格加边框时，千万不能用 `border`，否则表格的线会因为多出 1-2 像素而产生剧烈的“抖动”。我们要用 **内阴影 (`box-shadow: inset`)** 来模拟边框。

请按照以下 3 步来升级你的代码：

### 1. 增加选中单元格的状态记录

在 `<script setup>` 中，之前我们写了 `editingCell`（用于记录双击编辑），现在我们再加一个 `selectedCell`（用于记录单击选中），并写好对应的处理函数。

JavaScript

```js
// 记录当前【单击选中】的单元格
const selectedCell = ref({
  rowId: null,
  colKey: null
})

// 单击单元格事件
const handleCellClick = (row, column, cell, event) => {
  selectedCell.value = {
    rowId: row.__row_id,
    colKey: column.property
  }
}

// 动态给选中的单元格加上专用的 class
const getCellClassName = ({ row, column }) => {
  if (selectedCell.value.rowId === row.__row_id && selectedCell.value.colKey === column.property) {
    return 'cell-selected-highlight' // 触发咱们自定义的 CSS
  }
  return ''
}
```

### 2. 将事件和属性绑定到表格上

在你的 `<template>` 中，找到 `<el-table>` 标签，把刚刚写好的单击事件 `@cell-click` 和单元格样式控制 `:cell-class-name` 绑定上去。

HTML

```html
      <el-table
          :data="tableData"
          style="width: 100%;"
          border
          height="100%"
          :row-class-name="getRowClassName"
          highlight-current-row
          @current-change="handleCurrentRowChange"
          
          :cell-class-name="getCellClassName"
          @cell-click="handleCellClick"
          
          @cell-dblclick="handleCellDbClick"
      >
```

### 3. 加入高逼格的 CSS 样式

在最底部的 `<style scoped>` 中，加入下面这段 CSS。

CSS

```
/* 选中行的整行浅色高亮 (覆盖 Element Plus 默认的颜色，让它更像 IDE) */
:deep(.el-table__body tr.current-row > td.el-table__cell) {
  background-color: #e6f1fc !important;
}

/* ✨ 核心魔法：选中单元格的深色描边高亮 */
:deep(.el-table .cell-selected-highlight) {
  /* 使用 inset 内阴影模拟边框，绝对不会导致表格抖动变形 */
  box-shadow: inset 0 0 0 2px #409eff !important;
  /* 单元格的背景色稍微加深一点点，与整行高亮区分开 */
  background-color: #dbf0ff !important;
}

/* 输入框出现时，让它完美贴合单元格，去掉默认边框 */
:deep(.el-table .el-input__wrapper) {
  box-shadow: none !important;
  padding: 0 8px;
  border-radius: 0;
}
```

1. 你点击的那一行会变成浅蓝色。
    
2. 你点击的那个具体的格子，会被一个清晰的蓝色边框（内阴影）死死框住，底色也更深一点！
    
3. 然后你再**双击**它，边框里就会丝滑地出现输入框。
