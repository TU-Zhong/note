这一步是把你之前写的“大脑”（Pinia）装进“身体”的关键。我们要利用 **Element Plus** 提供的布局组件，复刻出类似 IntelliJ IDEA 的专业界面骨架。

---

### 1. 核心概念：`el-container` 的嵌套逻辑

在 Element Plus 中，布局是靠这几个组件实现的：

- **`el-container`**：外层容器。当子元素包含 `el-header` 或 `el-footer` 时，它会自动垂直排列；否则水平排列。
    
- **`el-header`**：顶栏（放置标题、连接状态）。
    
- **`el-aside`**：侧边栏（放置数据库树）。
    
- **`el-main`**：主要区域（放置 SQL 编辑器和结果表）。
    

---

### 2. 创建布局组件：`src/layout/MainLayout.vue`

这个文件将定义你的“领土划分”。建议你按照下面的代码手敲一遍，重点关注 **CSS 的高度设置**（这是很多毕设项目布局崩掉的主因）。

Code snippet
[[wrapper 类]]


```vue
<template>
  <el-container class="app-wrapper">
    
    <el-header height="40px" class="ide-header">
      <div class="logo">DB-Metadata-Browser</div>
      <div class="user-info">何翊 - 22级软件工程三班</div>
    </el-header>

    <el-container class="main-body">
      
      <el-aside width="260px" class="ide-aside">
        <div class="placeholder-text">Database Explorer</div>
      </el-aside>

      <el-main class="ide-main">
        <div class="placeholder-text">Workspace (Tabs)</div>
      </el-main>

    </el-container>
  </el-container>
</template>

<style scoped>
/* 关键：让整个页面高度等于浏览器视口高度，不出现多余滚动条 */
.app-wrapper {
  height: 100vh;
  display: flex;
  flex-direction: column;
}

/* 顶部样式：深灰色，IDEA 风格 */
.ide-header {
  background-color: #3c3f41;
  color: #afb1b3;
  display: flex;
  justify-content: space-between;
  align-items: center;
  border-bottom: 1px solid #2b2b2b;
  font-size: 13px;
}

/* 侧边栏样式 */
.ide-aside {
  background-color: #313335;
  border-right: 1px solid #2b2b2b;
  color: #afb1b3;
}

/* 主区域样式 */
.ide-main {
  background-color: #2b2b2b;
  padding: 0; /* 必须设为0，否则标签页上方会有白边 */
  color: #cfd3dc;
}

.placeholder-text {
  text-align: center;
  margin-top: 50px;
  color: #606266;
  font-style: italic;
}
</style>
```

### 3. App.vue 瘦身：连接“灵魂”与“身体”

修改你的 `src/App.vue`。现在的它不再负责具体业务，仅仅是一个**挂载点**。

Code snippet

```vue
<template>
  <MainLayout />
</template>

<script setup>
// 引入布局组件
import MainLayout from './layout/MainLayout.vue'
</script>

<style>
/* 整个项目的全局初始化，消除默认边距 */
body {
  margin: 0;
  padding: 0;
  overflow: hidden; /* 防止窗口出现双重滚动条 */
  font-family: "Segoe UI", Tahoma, Geneva, Verdana, sans-serif;
}
</style>
```

---

### 💡 为什么这样拆分？（软件工程思维）

1. **单一职责原则**：`App.vue` 只管应用入口，`MainLayout.vue` 只管页面结构。
    
2. **解耦布局与业务**：如果你以后想把侧边栏挪到右边，你只需要改 `MainLayout` 的 CSS，而不需要动任何业务逻辑代码。
    
3. **适配性**：使用 `100vh` 确保了你的项目在任何分辨率的显示器上，都能像真正的桌面软件一样撑满屏幕。
    

---

### 🛠️ 手动实现时的小技巧

- **观察 `flex` 方向**：外层的 `el-container` 包含 `el-header`，所以它默认是 `direction="vertical"`（垂直排列）；里面的 `el-container` 包含 `el-aside` 和 `el-main`，所以它默认是水平排列。
    
- **配色方案**：我给出的配色（`#3c3f41`, `#313335`, `#2b2b2b`）是参考 JetBrains 的 **Darcula 主题**，这会让你的毕设看起来非常专业。
