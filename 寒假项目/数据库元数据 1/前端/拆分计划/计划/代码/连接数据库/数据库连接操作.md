你的后端代码使用的是 `@RequestParam`，这意味着它**不接收 JSON 格式的 Body**，而是接收 `application/x-www-form-urlencoded` 格式的数据（或者直接拼在 URL 后面）。

**前端 Axios 调用时必须特殊处理**，不能直接传 JSON 对象，否则后端会报 `Missing ServletRequestParameterException`。

为了保持代码整洁，我们依然采用 **“新建一个弹窗组件，在侧边栏引用”** 的方案。

### 第一步：新建弹窗组件 `ConnectionDialog.vue`

在 `src/components` 目录下新建 `ConnectionDialog.vue`。

这个组件做了两件事：

1. **自动拼凑 JDBC URL**：用户填 IP 和端口，代码自动生成 `jdbc:mysql://...`，不用用户手写。
    
2. **格式转换**：把数据转成 `URLSearchParams` 发给后端。
    

Code snippet

```vue
<template>
  <el-dialog
    v-model="visible"
    title="新建数据库连接"
    width="600px"
    :close-on-click-modal="false"
    @close="handleClose"
  >
    <el-form :model="form" ref="formRef" :rules="rules" label-width="120px">
      
      <el-form-item label="连接名称" prop="name">
        <el-input v-model="form.name" placeholder="例如：本地MySQL" />
      </el-form-item>

      <el-form-item label="驱动Jar路径" prop="jarPath">
        <el-input v-model="form.jarPath" placeholder="D:/drivers/mysql-connector-j-8.0.33.jar">
           <template #append>
             <el-tooltip content="后端动态加载驱动需要绝对路径" placement="top">
               <el-icon><QuestionFilled /></el-icon>
             </el-tooltip>
           </template>
        </el-input>
      </el-form-item>

      <el-divider content-position="left">数据库详情</el-divider>

      <el-row :gutter="20">
        <el-col :span="16">
          <el-form-item label="主机 (Host)" prop="host">
            <el-input v-model="form.host" @input="updateUrl"/>
          </el-form-item>
        </el-col>
        <el-col :span="8">
          <el-form-item label="端口" prop="port">
            <el-input v-model="form.port" @input="updateUrl"/>
          </el-form-item>
        </el-col>
      </el-row>

      <el-form-item label="数据库名" prop="dbName">
        <el-input v-model="form.dbName" placeholder="mysql" @input="updateUrl"/>
      </el-form-item>

      <el-form-item label="JDBC URL" prop="url">
        <el-input v-model="form.url" type="textarea" :rows="2" />
      </el-form-item>

      <el-form-item label="用户名" prop="userName">
        <el-input v-model="form.userName" />
      </el-form-item>

      <el-form-item label="密码" prop="passWord">
        <el-input v-model="form.passWord" type="password" show-password />
      </el-form-item>

    </el-form>

    <template #footer>
      <span class="dialog-footer">
        <el-button @click="visible = false">取消</el-button>
        <el-button type="primary" :loading="loading" @click="handleConnect">
          测试并连接
        </el-button>
      </span>
    </template>
  </el-dialog>
</template>

<script setup>
import { ref, watch, reactive } from 'vue'
import axios from 'axios'
import { ElMessage } from 'element-plus'
import { QuestionFilled } from '@element-plus/icons-vue'

const props = defineProps(['modelValue'])
const emit = defineEmits(['update:modelValue', 'success'])

const visible = ref(false)
const loading = ref(false)
const formRef = ref()

const form = reactive({
  name: 'Local-MySQL',
  // 这里的默认路径你需要改成你自己电脑上真实的 jar 包路径方便测试
  jarPath: 'D:\\maven_repo\\mysql\\mysql-connector-j\\8.0.33\\mysql-connector-j-8.0.33.jar', 
  host: 'localhost',
  port: '3306',
  dbName: 'mysql', // 默认连接系统库
  url: 'jdbc:mysql://localhost:3306/mysql?serverTimezone=UTC',
  userName: 'root',
  passWord: ''
})

const rules = {
  name: [{ required: true, message: '请输入连接名', trigger: 'blur' }],
  jarPath: [{ required: true, message: 'Jar包路径不能为空', trigger: 'blur' }],
  url: [{ required: true, message: 'URL不能为空', trigger: 'blur' }],
  userName: [{ required: true, message: '请输入用户名', trigger: 'blur' }]
}

// 自动生成 JDBC URL 的小助手
const updateUrl = () => {
  form.url = `jdbc:mysql://${form.host}:${form.port}/${form.dbName}?serverTimezone=UTC&useSSL=false`
}

watch(() => props.modelValue, (val) => visible.value = val)
watch(visible, (val) => emit('update:modelValue', val))

const handleConnect = async () => {
  await formRef.value.validate(async (valid) => {
    if (!valid) return

    loading.value = true
    try {
      // 🔥 核心点：因为后端用的是 @RequestParam，所以前端必须用 URLSearchParams
      const params = new URLSearchParams()
      params.append('jarPath', form.jarPath)
      params.append('url', form.url)
      params.append('userName', form.userName)
      params.append('passWord', form.passWord)

      // 发送 POST 请求
      const res = await axios.post('http://localhost:8080/connect', params)

      if (res.data.code === 200) {
        ElMessage.success(res.data.msg)
        
        // 构造一个新的树节点对象
        const newConnectionNode = {
          id: res.data.data.token, // 这是后端返回的 UUID connectionId
          label: form.name,        // 这是用户起的名字
          type: 'database',        // 节点类型
          children: []             // 准备放表
        }
        
        // 通知父组件
        emit('success', newConnectionNode)
        visible.value = false
      } else {
        ElMessage.error(res.data.msg || '连接失败')
      }
    } catch (e) {
      console.error(e)
      ElMessage.error('无法连接到后端服务器')
    } finally {
      loading.value = false
    }
  })
}

const handleClose = () => {
  // 不重置 form，保留上次输入方便调试
}
</script>
```

### 总结要点

1. **参数传递方式**：这是最容易报错的地方。后端用 `@RequestParam`，前端必须用 `new URLSearchParams()` 或者 `FormData`。如果用普通的 JSON 对象 `{ key: value }`，后端会接收不到参数。
    
2. **数据流向**：
    
    - 用户填表 -> 点击连接 -> `ConnectionDialog` 发请求。
        
    - 请求成功 -> 拿到 `token` (UUID)。
        
    - `ConnectionDialog` 触发 `emit('success', node)`。
        
    - `Sidebar` 监听到事件 -> 把 `node` push 到 `treeData`。
        
    - 左侧树列表刷新，出现新的数据库图标。
        

现在，你可以点击侧边栏的 **+** 号，输入你的本地数据库信息（注意 Jar 包路径要填对），点击连接，应该就能看到提示“数据库连接成功”，并且左侧树上多了一项
