

用小区大门和快递柜的关系来比喻，保证楼下遛弯的大爷都能听懂：

🔑 **ReentrantLock（普通门锁）**
- 就像你家大门挂的链条锁
- 你回家开门后：可以反复进出（可重入）
- 但每次只能有一个人在家（同一时间只能一个线程进）
- 适合：厕所单间、保险箱操作

📬 **ReentrantReadWriteLock（智能快递柜）**
- 左边是透明展示柜（读锁），右边是储物箱（写锁）
- **看快递**：所有人可以同时扒着展示柜看（多个线程同时读）
- **存快递**：如果有人要放新快递，就要：①清空看快递的人 ②独自使用整个柜子（写锁独占）
- 适合：公告栏查看（多人读）和更新（单人写）

🍵 生活场景对比：
1. 老王修水管（写操作）：用普通门锁，全家清场
2. 邻居串门喝茶（读操作）：用智能柜，可以同时来好几家人唠嗑
3. 物业换公告（先读后写）：先让大家看完旧公告，再锁柜子换新公告

就像小区里的快递站，平时随便看（读锁），但更新货架时就要包场（写锁）～ 🚚
