    private void m1(){}

    void m2(){}

    protected void m3(){}

官方解析：这道题目考察Java中方法的访问权限修饰符和方法继承、覆盖的相关知识。C选项是正确的,因为protected修饰的方法m3()对子类是可见的,而且非static方法是可以被子类覆盖的。  
  
分析其他选项:  
  
A错误:m1()是private方法,private修饰的方法对子类是不可见的,因此子类无法继承和覆盖私有方法。  
  
B错误:m2()使用默认访问权限(package-private),只对同一包内的类可见。如果子类与父类不在同一个包中,则无法继承和覆盖该方法。所以不能说"一定"能够继承和覆盖。  
  
D错误:m4()是static方法(静态方法)。静态方法可以被子类继承,但不能被覆盖(重写)。子类中定义相同签名的静态方法,这种行为称为隐藏(hide),而不是覆盖(override)。  
  
补充说明:protected修饰的方法m3()具有以下特点:  
1. 对同一包内的其他类可见  
2. 对不同包中的子类也可见  
3. 是非静态方法,可以被子类覆盖  
因此无论子类在哪个包中,都一定能够继承和覆盖m3()方法。
