```java
CompletableFuture<Void> future = CompletableFuture.runAsync(() -> System.out.println("7. CompletableFuture创建线程：" + Thread.currentThread().getName()));
```
CompletableFuture 的输出没有显示的原因是主线程可能在 CompletableFuture 任务执行完成之前就结束了。CompletableFuture.runAsync() 是异步执行的，它不会阻塞主线程。

让我们修改代码来确保能看到输出：
```java
// 7.使用CompletableFuture
CompletableFuture<Void> future = CompletableFuture.runAsync(
    () -> System.out.println("7. CompletableFuture创建线程：" + Thread.currentThread().getName())
);
// 等待CompletableFuture执行完成
future.join();
```
或
```java
// 7.使用CompletableFuture

CompletableFuture<Void> future = CompletableFuture.runAsync(

    () -> System.out.println("7. CompletableFuture创建线程：" + Thread.currentThread().getName())

);

// 等待一小段时间确保任务执行完成

Thread.sleep(100);
```
这两种方式都可以确保我们能看到 CompletableFuture 的输出。`join()` 方法会等待任务完成，而 `Thread.sleep()` 则是给任务一个执行的时间窗口。在实际应用中，推荐使用 `join()` 或 `get()` 方法来等待异步任务完成。
