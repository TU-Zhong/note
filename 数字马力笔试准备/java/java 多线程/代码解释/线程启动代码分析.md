```java
public class MyThread extends Thread{  
public static void main(String args[]){  
MyThread thread = new MyThread();  
thread.run();  
}  
public void start(){  
for(int i = 0; i < 5; i++) {  
System.out.print(i);  
}  
}  
}
```

MyThread继承Thread类，且MyThread没有重写run()方法。因此调用MyThread的run方法时，就是调用父类Thread的run方法。

`main` 方法中调用的是 `thread.run()` 而非 `thread.start()`

- `run()` 方法只是普通方法调用，不会启动新线程
- `Thread` 类的原始 `run()` 方法是空实现
