### **一、基础食材包（核心类）**
```java
// 字节流：适合搬运所有类型文件（像万能三轮车）
InputStream  // 进货用（读数据）
OutputStream // 出货用（写数据）

// 字符流：专门处理文本（像专用煎饼纸袋）
Reader  // 读文字
Writer  // 写文字
```

---

### **二、常用工具套餐**
#### **1. 文件读写（煎饼配方存储）**
```java
// 字节流读写文件
try (FileInputStream in = new FileInputStream("配方.txt");
     FileOutputStream out = new FileOutputStream("备份.txt")) {
    int data;
    while ((data = in.read()) != -1) {
        out.write(data); // 一个字节一个字节复制
    }
}

// 字符流读写文件（更高效读文字）
try (BufferedReader reader = new BufferedReader(new FileReader("订单.txt"));
     BufferedWriter writer = new BufferedWriter(new FileWriter("总订单.txt"))) {
    String line;
    while ((line = reader.readLine()) != null) {
        writer.write(line + "\n"); // 整行读取写入
    }
}
```

#### **2. 缓冲流（加速小推车）**
```java
// 比普通流快N倍！
try (BufferedInputStream bis = new BufferedInputStream(new FileInputStream("大视频.mp4"));
     BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream("副本.mp4"))) {
    byte[] buffer = new byte[1024];
    int len;
    while ((len = bis.read(buffer)) != -1) {
        bos.write(buffer, 0, len); // 批量搬运数据
    }
}
```

#### **3. 对象序列化（煎饼外卖打包）**
```java
// 先让类实现Serializable接口
class 煎饼订单 implements Serializable {
    String 口味;
    int 数量;
}

// 保存订单
try (ObjectOutputStream oos = new ObjectOutputStream(new FileOutputStream("订单.data"))) {
    oos.writeObject(new 煎饼订单("加蛋", 3)); 
}

// 读取订单
try (ObjectInputStream ois = new ObjectInputStream(new FileInputStream("订单.data"))) {
    煎饼订单 订单 = (煎饼订单) ois.readObject();
    System.out.println(订单.口味); // 输出：加蛋
}
```

#### **4. 标准输入输出（摊主和顾客对话）**
```java
// 从键盘读取输入
Scanner scanner = new Scanner(System.in);
System.out.print("要几个煎饼？");
int num = scanner.nextInt();

// 向屏幕输出
System.out.println("马上做好" + num + "个煎饼！");

// 错误信息输出
System.err.println("火太大煎糊了！");
```

#### **5. NIO新武器（高效传送带）**
```java
// 使用Path和Files工具类
Path 源文件 = Paths.get("原始配方.txt");
Path 目标文件 = Paths.get("加密配方.txt");

// 快速复制文件
Files.copy(源文件, 目标文件, StandardCopyOption.REPLACE_EXISTING);

// 读取全部内容
String 内容 = new String(Files.readAllBytes(源文件));
```

### **三、重点技巧备忘录**
1. **记得关流**：用`try-with-resources`自动关（就像收摊要关火）
2. **选对工具**：
   - 文本文件 → 字符流（Reader/Writer）
   - 图片/视频 → 字节流（InputStream/OutputStream）
3. **缓冲必用**：Buffered系列性能提升明显
4. **异常处理**：所有IO操作都要处理IOException
