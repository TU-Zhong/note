### 问题一：堆有哪几种？

从基本功能上讲，堆最常见的分类有两种：

#### 1. 最小堆 (Min-Heap)

这是我们一直在讨论的类型，也是 `heapq` 的默认实现。

- **规则**：父节点的值 **小于或等于** 其子节点的值。
    
- **结果**：根节点永远是整个堆中**最小值**。
    

#### 2. 最大堆 (Max-Heap)

这与最小堆的概念正好相反。

- **规则**：父节点的值 **大于或等于** 其子节点的值。
    
- **结果**：根节点永远是整个堆中**最大值**。
    

最小堆和最大堆是算法问题中最常用到的两种堆。除此之外，还有一些为了在特定操作（如合并两个堆）上获得更好性能而设计的更复杂的堆，比如：

- 二项堆 (Binomial Heap)
    
- 斐波那契堆 (Fibonacci Heap)
    
- 配对堆 (Pairing Heap)
    

不过在日常的算法解题和常规应用中，你只需要掌握最小堆和最大堆就足够了。

### 问题二：`heapq` 的操作只能是最小堆吗？

**是的，直接来说，Python 的 `heapq` 模块本身只支持最小堆。**

它所有的函数（`heappush`, `heappop` 等）都是按照最小堆的规则来设计的。没有一个叫做 `max_heappush` 的内建函数。

**但是，这并不意味着我们无法用 `heapq` 实现一个最大堆。** 我们可以用一个非常简单且高效的技巧来“欺骗”最小堆，让它为我们实现最大堆的功能。

#### 模拟最大堆的技巧：存入相反数

这个技巧的原理是： **一个数集合中的最大值，取反后就是这个新集合中的最小值。**

例如，对于集合 `[1, 9, 5]`：

- 最大值是 `9`。
    
- 把它们全部取反，得到 `[-1, -9, -5]`。
    
- 在这个新集合中，最小值是 `-9`。
    

我们可以利用这个特性：

1. **存入 (Push)**：当你想要往最大堆里存入一个数 `x` 时，你实际存入 `heapq` 的是它的相反数 `-x`。
    
2. **取出 (Pop)**：当你从 `heapq` 里取出一个最小值时（比如 `-9`），你再把它取反一次（`-(-9)`），就得到了我们真正想要的原始集合中的最大值 `9`。
    

#### 示例代码：用 `heapq` 实现最大堆

Python

```
import heapq

# 假设我们想找到 [1, 10, 3, 7, 2] 中的最大值
nums = [1, 10, 3, 7, 2]
max_heap = []

# 1. 存入时，存入它们的相反数
for num in nums:
    heapq.heappush(max_heap, -num)
    
# 看看 max_heap 列表现在的内容
# 它是一个最小堆，但存的是负数
print(f"内部存储的最小堆: {max_heap}") 
# 输出可能类似: [-10, -7, -3, -1, -2]
# 注意：第一个元素 -10 是最小值

# 2. 查看最大值（不取出）
# 堆顶是-10，取反后就是10
largest_num = -max_heap[0]
print(f"当前的最大值是: {largest_num}") # 输出: 10

# 3. 依次取出最大值
print("\n按从大到小的顺序取出元素:")
while max_heap:
    # 每次从最小堆中弹出最小值（一个负数）
    popped_val = heapq.heappop(max_heap)
    # 再取反，就得到了我们想要的最大值
    original_val = -popped_val
    print(original_val)
```

输出结果：

```
内部存储的最小堆: [-10, -7, -3, -1, -2]
当前的最大值是: 10

按从大到小的顺序取出元素:
10
7
3
2
1
```

**对于元组 (Tuple) 也是同理**，我们只需要对元组中用于比较的第一个元素取反即可： `heapq.heappush(max_heap, (-priority, task_name))`

### 总结

| 功能     | 直接实现 (最小堆)                       | 模拟实现 (最大堆)                       |
| ------ | -------------------------------- | -------------------------------- |
| **存储** | `heapq.heappush(heap, item)`     | `heapq.heappush(heap, -item)`    |
| **取出** | `smallest = heapq.heappop(heap)` | `largest = -heapq.heappop(heap)` |
| **查看** | `smallest = heap[0]`             | `largest = -heap[0]`             |
| **原理** | 直接利用 `heapq` 的默认行为               | 存入相反数，利用最小堆找到“最小的负数”（即绝对值最大的数）   |
