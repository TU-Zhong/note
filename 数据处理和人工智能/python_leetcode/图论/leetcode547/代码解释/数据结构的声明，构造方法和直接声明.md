**总的来说，Python 中绝大多数内置数据结构的声明和初始化，就是通过调用其类型的构造函数（如 `set()`, `list()`, `dict()`）或者使用更简洁的“字面量语法”（Literal Syntax）来完成的。**

下面我们来详细看一下几种最核心的内置数据结构，您会发现它们的规律是一致的。

---

### 核心内置数据结构的声明方式

这里有四种最基本、最常用的数据结构：列表（List）、元组（Tuple）、集合（Set）、字典（Dictionary）。

|数据结构|功能简介|构造函数方式 (通用)|**字面量语法 (更常用、更推荐)**|Java 类比|
|:--|:--|:--|:--|:--|
|**列表 (list)**|有序、可变、可重复的集合。|`my_list = list()`|`my_list = []`|`ArrayList`|
|**元组 (tuple)**|有序、**不可变**、可重复的集合。|`my_tuple = tuple()`|`my_tuple = ()`|（类似 `final List`）|
|**集合 (set)**|无序、可变、**不重复**的集合。|`my_set = set()`|`my_set = {1, 2, 3}` _(注意)_|`HashSet`|
|**字典 (dict)**|无序的键值对 (key-value) 集合。|`my_dict = dict()`|`my_dict = {}`|`HashMap`|

### 详细说明与示例

#### 1. 列表 (list) - 最常用的“动态数组”

列表用来存放一系列有序的元素，可以随时添加、删除和修改。

Python

```
# 方式一：使用构造函数（不常用）
empty_list_1 = list()

# 方式二：使用字面量语法（最常用）
empty_list_2 = []

# 创建时直接初始化
fruits = ["apple", "banana", "cherry", "apple"]
print(fruits)  # 输出: ['apple', 'banana', 'cherry', 'apple']
print(fruits[0]) # 输出: 'apple'

# 添加元素
fruits.append("orange")
print(fruits)  # 输出: ['apple', 'banana', 'cherry', 'apple', 'orange']
```

#### 2. 元组 (tuple) - 不可变的列表

元组就像一个被“锁定”的列表，一旦创建就不能修改。这让它在某些场景下更安全、更高效。

Python

```
# 方式一：使用构造函数
empty_tuple_1 = tuple()

# 方式二：使用字面量语法
empty_tuple_2 = ()

# 创建时直接初始化
coordinates = (120.3, 23.5)
print(coordinates)  # 输出: (120.3, 23.5)

# 尝试修改会报错 TypeError
# coordinates[0] = 121.0  # 这行代码会引发错误！
```

#### 3. 集合 (set) - 高效去重与关系测试

就像您之前看到的，用于存储不重复的元素，并能快速检查成员是否存在。

Python

```
# 方式一：使用构造函数（创建空集合的唯一方式）
empty_set = set()

# 方式二：使用字面量语法（用于创建非空集合）
unique_numbers = {1, 2, 3, 4, 4, 5}
print(unique_numbers)  # 输出: {1, 2, 3, 4, 5} (自动去重)

# 添加元素
unique_numbers.add(6)
unique_numbers.add(1) # 再次添加1，但集合内容不变
print(unique_numbers)  # 输出: {1, 2, 3, 4, 5, 6}
```

**特别注意**： `my_empty_set = {}` **不会**创建一个空集合，它会创建一个**空字典**！这是一个非常重要的区别。要创建空集合，必须使用 `set()`。

#### 4. 字典 (dict) - 键值对存储

用来存储一系列的键(key)和值(value)的对应关系。

Python

```
# 方式一：使用构造函数
empty_dict_1 = dict()

# 方式二：使用字面量语法（最常用）
empty_dict_2 = {}

# 创建时直接初始化
person = {
    "name": "张三",
    "age": 30,
    "city": "台北"
}
print(person)  # 输出: {'name': '张三', 'age': 30, 'city': '台北'}

# 通过键访问值
print(person["name"])  # 输出: '张三'

# 添加或修改键值对
person["job"] = "工程师"
person["age"] = 31
print(person) # 输出: {'name': '张三', 'age': 31, 'city': '台北', 'job': '工程师'}
```

### 总结与规律

是的，您的理解完全正确。在 Python 中声明和初始化数据结构的核心规律就是：

1. **构造函数方式**：`变量名 = 类型名()`，例如 `my_list = list()`。这种方式通用且清晰。
2. **字面量方式**：`变量名 = 符号`，例如 `my_list = []`。这是 Python 推崇的更简洁、更高效的方式，在实际编码中最为常见。

除了这些内置类型，很多来自标准库或第三方库的复杂数据结构也遵循类似的初始化方式，例如：

Python

```
from collections import deque

# 创建一个双端队列
my_deque = deque()
```

这种直观、一致的设计，正是 Python 语言魅力的重要组成部分。
