### Java 集合的四种排序方式

在 Java 中，对集合（Collection）或数组（Array）进行排序是非常常见的操作。主要可以通过以下四种方式实现：

1. **自然排序 (Natural Ordering)**：让集合元素的类实现 `Comparable` 接口。
    
2. **比较器排序 (Comparator Ordering)**：创建一个实现了 `Comparator` 接口的比较器类。
    
3. **流式排序 (Stream Sorting)**：使用 Java 8 引入的 Stream API 的 `sorted()` 方法。
    
4. **工具类排序 (Utility Class Sorting)**：使用 `Collections` 或 `Arrays` 工具类提供的 `sort()` 方法。
    

---

### 1. 自然排序：实现 `Comparable` 接口

当一个类的对象具有内在的、唯一的排序逻辑时，可以让该类实现 `Comparable` 接口。这被称为“自然排序”。例如，数字的大小、字符串的字典序就是一种自然排序。

**核心方法**：`int compareTo(T o)`

- 返回**负整数**：表示当前对象 (`this`) 小于参数对象 (`o`)。
    
- 返回**零**：表示当前对象等于参数对象。
    
- 返回**正整数**：表示当前对象大于参数对象。
    

**使用场景**：

- 当排序逻辑是该类的主要和固定逻辑时。
    
- 希望将排序逻辑封装在对象本身中。
    

示例代码：

假设我们有一个 Student 类，我们希望默认按照学生的年龄进行升序排序。

Java

```
import java.util.ArrayList;
import java.util.Collections;
import java.util.List;

// 1. 让 Student 类实现 Comparable 接口
class Student implements Comparable<Student> {
    private String name;
    private int age;

    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    public String getName() {
        return name;
    }

    public int getAge() {
        return age;
    }

    @Override
    public String toString() {
        return "Student{" + "name='" + name + '\'' + ", age=" + age + '}';
    }

    // 2. 重写 compareTo 方法，定义自然排序规则（按年龄升序）
    @Override
    public int compareTo(Student other) {
        // this.age > other.age, 返回正数
        // this.age < other.age, 返回负数
        // this.age == other.age, 返回 0
        return this.age - other.age;
    }
}

public class NaturalSortExample {
    public static void main(String[] args) {
        List<Student> students = new ArrayList<>();
        students.add(new Student("Alice", 23));
        students.add(new Student("Bob", 20));
        students.add(new Student("Charlie", 25));

        System.out.println("排序前: " + students);

        // 3. 直接调用 Collections.sort()，它会自动使用 compareTo 方法
        Collections.sort(students);

        System.out.println("按年龄自然排序后: " + students);
    }
}
```

**输出**：

```
排序前: [Student{name='Alice', age=23}, Student{name='Bob', age=20}, Student{name='Charlie', age=25}]
按年龄自然排序后: [Student{name='Bob', age=20}, Student{name='Alice', age=23}, Student{name='Charlie', age=25}]
```

### 2. 比较器排序：使用 `Comparator` 接口

如果一个类没有实现 `Comparable` 接口，或者你想根据多种不同的规则进行排序（例如，有时按年龄排序，有时按名字排序），则可以使用 `Comparator`。它将排序逻辑从业务类中分离出来。

**核心方法**：`int compare(T o1, T o2)`

- 返回**负整数**：表示 `o1` 小于 `o2`。
    
- 返回**零**：表示 `o1` 等于 `o2`。
    
- 返回**正整数**：表示 `o1` 大于 `o2`。
    

**使用场景**：

- 排序的类无法修改（例如，来自第三方库）。
    
- 需要提供多种排序策略。
    
- 排序逻辑是临时的或与业务逻辑无关。
    

示例代码：

我们继续使用 Student 类（这次可以不实现 Comparable），但我们想按照名字的字典序来排序。

Java

```
import java.util.ArrayList;
import java.util.Collections;
import java.util.Comparator;
import java.util.List;

// Student 类可以不实现 Comparable 接口
// ...

// 1. 创建一个或多个实现了 Comparator 接口的比较器类
class SortByName implements Comparator<Student> {
    @Override
    public int compare(Student s1, Student s2) {
        return s1.getName().compareTo(s2.getName());
    }
}

public class ComparatorSortExample {
    public static void main(String[] args) {
        List<Student> students = new ArrayList<>();
        students.add(new Student("Charlie", 25));
        students.add(new Student("Alice", 23));
        students.add(new Student("Bob", 20));

        System.out.println("排序前: " + students);

        // 2. 将比较器实例作为参数传递给 Collections.sort()
        Collections.sort(students, new SortByName());

        System.out.println("按名字排序后: " + students);

        // 也可以使用 Lambda 表达式（更简洁）
        // 按年龄降序排序
        Comparator<Student> sortByAgeDesc = (s1, s2) -> s2.getAge() - s1.getAge();
        Collections.sort(students, sortByAgeDesc);
        System.out.println("按年龄降序排序后: " + students);
    }
}
```

**输出**：

```
排序前: [Student{name='Charlie', age=25}, Student{name='Alice', age=23}, Student{name='Bob', age=20}]
按名字排序后: [Student{name='Alice', age=23}, Student{name='Bob', age=20}, Student{name='Charlie', age=25}]
按年龄降序排序后: [Student{name='Charlie', age=25}, Student{name='Alice', age=23}, Student{name='Bob', age=20}]
```

### 3. 流式排序：Java 8 Stream API

Java 8 引入的 Stream API 提供了一种声明式、函数式的编程方式来处理集合。`sorted()` 方法是其核心排序操作。

**核心方法**：

- `Stream<T> sorted()`: 根据元素的自然顺序进行排序（要求元素实现 `Comparable`）。
    
- `Stream<T> sorted(Comparator<? super T> comparator)`: 根据提供的比较器进行排序。
    

**特点**：

- **非侵入式**：不修改原始集合，而是返回一个排序后的新流。
    
- **链式操作**：可以与其他流操作（如 `filter`, `map`）组合使用。
    
- **代码简洁**：语法非常优雅。
    

**示例代码**：

Java

```
import java.util.ArrayList;
import java.util.Comparator;
import java.util.List;
import java.util.stream.Collectors;

public class StreamSortExample {
    public static void main(String[] args) {
        List<Student> students = new ArrayList<>();
        students.add(new Student("Charlie", 25));
        students.add(new Student("Alice", 23));
        students.add(new Student("Bob", 20));

        System.out.println("原始列表: " + students);

        // 1. 使用 sorted() 和 Comparator 按年龄升序排序
        List<Student> sortedByAge = students.stream()
                .sorted(Comparator.comparingInt(Student::getAge))
                .collect(Collectors.toList());

        System.out.println("按年龄流式排序后: " + sortedByAge);

        // 2. 按名字降序排序
        List<Student> sortedByNameDesc = students.stream()
                .sorted(Comparator.comparing(Student::getName).reversed())
                .collect(Collectors.toList());

        System.out.println("按名字降序流式排序后: " + sortedByNameDesc);
        
        // 注意：原始列表 students 并未被修改
        System.out.println("排序后原始列表: " + students);
    }
}
```

**输出**：

```
原始列表: [Student{name='Charlie', age=25}, Student{name='Alice', age=23}, Student{name='Bob', age=20}]
按年龄流式排序后: [Student{name='Bob', age=20}, Student{name='Alice', age=23}, Student{name='Charlie', age=25}]
按名字降序流式排序后: [Student{name='Charlie', age=25}, Student{name='Bob', age=20}, Student{name='Alice', age=23}]
排序后原始列表: [Student{name='Charlie', age=25}, Student{name='Alice', age=23}, Student{name='Bob', age=20}]
```

### 4. 工具类排序：`Collections` 和 `Arrays`

这是最基础和直接的排序方式。`java.util.Collections` 用于对 `List` 集合进行排序，而 `java.util.Arrays` 用于对数组进行排序。

**核心方法**：

- `Collections.sort(List<T> list)`: 对列表进行自然排序。
    
- `Collections.sort(List<T> list, Comparator<? super T> c)`: 使用比较器对列表排序。
    
- `Arrays.sort(T[] a)`: 对数组进行自然排序。
    
- `Arrays.sort(T[] a, Comparator<? super T> c)`: 使用比较器对数组排序。
    

**特点**：

- **原地排序（In-place）**：直接修改原始的列表或数组。
    
- **适用性广**：是前两种方法（`Comparable` 和 `Comparator`）的具体应用场景。
    

**示例代码**：

Java

```
import java.util.Arrays;
import java.util.List;
import java.util.ArrayList;
import java.util.Collections;

public class UtilitySortExample {
    public static void main(String[] args) {
        // --- 使用 Collections 对 List 排序 ---
        List<String> names = new ArrayList<>(Arrays.asList("David", "Alice", "Charlie", "Bob"));
        System.out.println("List 排序前: " + names);
        
        // 自然排序（字典序）
        Collections.sort(names);
        System.out.println("List 自然排序后: " + names);

        // --- 使用 Arrays 对数组排序 ---
        int[] numbers = {5, 2, 8, 1, 9};
        System.out.println("Array 排序前: " + Arrays.toString(numbers));
        
        // 自然排序（数值大小）
        Arrays.sort(numbers);
        System.out.println("Array 自然排序后: " + Arrays.toString(numbers));
    }
}
```

**输出**：

```
List 排序前: [David, Alice, Charlie, Bob]
List 自然排序后: [Alice, Bob, Charlie, David]
Array 排序前: [5, 2, 8, 1, 9]
Array 自然排序后: [1, 2, 5, 8, 9]
```

### 总结

| **方式**       | **核心接口/类**                | **主要特点**            | **优点**                             | **缺点**             |
| ------------ | ------------------------- | ------------------- | ---------------------------------- | ------------------ |
| **1. 自然排序**  | `Comparable`              | 排序逻辑与类本身绑定          | 简单直接，符合单一职责原则                      | 排序逻辑固定，不够灵活        |
| **2. 比较器排序** | `Comparator`              | 排序逻辑与类分离，作为外部策略     | 灵活性高，可实现多种排序规则                     | 需要额外创建比较器类或Lambda  |
| **3. 流式排序**  | `java.util.stream.Stream` | 返回新流，不修改原集合，支持链式操作  | 代码优雅，函数式编程，易于组合                    | 会创建新集合，可能产生额外开销    |
| **4. 工具类排序** | `Collections`, `Arrays`   | 直接对原集合/数组进行修改（原地排序） | 高效，是`Comparable`和`Comparator`的直接应用 | 修改原始数据结构，不适用于不可变场景 |
