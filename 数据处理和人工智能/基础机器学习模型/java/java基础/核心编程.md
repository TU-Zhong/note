- **动手编码**: 这份题单的核心是“动手”。不要只看不练，务必将每个题目都在 IDE 中实现一遍。
    
- **先思考再实现**: 在写代码前，先思考题目考察的知识点是什么，有哪些实现方案，各自的优缺点是什么。
