- **（澄清问题，展现专业性）** “您指的是我们系统的核心接口QPS（每秒请求数）吗？还是指能同时承载的在线用户数？”

- **（展示技术细节，体现落地能力）** “为了支撑这个QPS，我们在技术层面做了一些调优。比如，我们将内嵌Tomcat的**最大线程数设置为200**，并且为了与之匹配，我们将**数据库连接池（HikariCP）的最大连接数也设置为200**。此外，对于一些非核心的、耗时较长的写操作，我们引入了**消息队列（如RabbitMQ）进行异步化处理**，这样可以快速响应用户请求，把耗时操作交由后台消费者慢慢处理，起到了‘削峰填谷’的作用，进一步提升了主流程的QPS。”
