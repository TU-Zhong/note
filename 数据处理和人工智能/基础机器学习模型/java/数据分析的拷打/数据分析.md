“我认为这是整个项目中最能体现数据价值的一步。基于EDA的洞察，我使用`Pandas`构建了一个以‘客户ID’为唯一索引的特征宽表。我精心设计了四类特征：”

2. “**近期行为特征：** 为了捕捉客户的动态变化，我构造了更能体现近期活力的特征，比如`最近3个月的下单次数`、`最近一次下单距今天数`、`近期订单额增长率`等。”
    
3. “**客户偏好特征：** 为了量化客户对我们产品和工艺的偏好，我构造了`采购布料数量`、`最常采购的纺织工艺`、`是否采购过当季新品`等特征。”
    
4. “**业务属性特征：** 将`客户所属行业`、`销售区域`这类文本信息，通过**独热编码（One-Hot Encoding）**转化为模型可以识别的0/1特征。”

### **第三阶段：机器学习建模与敏捷迭代**

“数据准备好后，我将问题定义为一个**二分类问题**：预测一个纯纺织客户，未来是否会转化为综合服务客户。我采取了敏捷迭代的方式进行建模：”

#### **模型 V1：基线模型（逻辑回归）**

- “**模型拟合：** 我首先选择**逻辑回归**作为基线模型。因为它速度快，结果的**可解释性强**，便于我们快速验证特征的有效性。”
    
- “**模型评估：** 我们的业务目标是给销售团队一个**精准的**、小而美的目标清单，避免浪费销售资源。因此，在评估模型时，我没有单纯地看准确率（Accuracy），而是选择了**精确率（Precision）**作为核心评估指标，同时辅以**AUC值**来衡量模型的整体排序能力。V1模型的精确率大约是70%，AUC为0.78，证明了特征的有效性，但还有提升空间。”
    

#### **模型 V2：迭代优化（随机森林）**

- “**迭代思路：** 为了提升模型性能，我分析了V1模型的错误案例，发现它对于一些交易模式比较复杂的客户，预测效果不佳。我判断这可能是因为简单的线性模型无法捕捉特征间的非线性关系。”
    
- “**模型拟合：** 因此，我尝试了更复杂的**随机森林（Random Forest）**模型。同时，我利用随机森林自带的**特征重要性（Feature Importance）**功能，对我的特征进行了筛选和权重分析，剔除了一些噪声特征。”
    
- “**评估与调优：** 经过使用`GridSearchCV`进行参数调优后，V2模型的**精确率提升到了85%**，AUC值也达到了0.91。这个结果显著优于基线模型，我们判断它已经具备了投入实际应用的价值。”
