3. 1. RIP(Routing Information Protocol)确实是一个基于距离矢量的路由协议:  
	- RIP使用跳数作为度量标准来选择路由  
	- 路由器只知道到达目的网络的下一跳和总跳数  
	- 每30秒与相邻路由器交换完整的路由表  
	- 适用于小型网络,最大支持15跳  
  
	2. OSPF(Open Shortest Path First)确实是一个基于链路状态的路由协议:  
	- 路由器掌握整个网络的拓扑信息  
	- 使用SPF算法计算最短路径  
	- 只在网络拓扑发生变化时更新路由信息  
	- 适用于大型网络,无跳数限制  
	- 收敛速度快,开销值作为度量标准
