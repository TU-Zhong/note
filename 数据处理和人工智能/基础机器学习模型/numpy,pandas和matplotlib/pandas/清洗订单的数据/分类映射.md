### 代码实现

下面是如何在 Python (Pandas) 中实现这个分类逻辑。我们创建一个“规则字典”，然后编写一个函数来应用这些规则。

Python
==apply(func),给函数func传入的是一个数据不是Series,然后`if keyword in clean_category:`是通过字符串的子字符串查找功能看k有没有在clean_category中==
```python
import pandas as pd
import io

# 假设 df 是您之前创建的测试数据集
# ... (此处省略加载数据的代码) ...
# df = pd.read_csv(io.StringIO(csv_data))

# 1. 定义映射规则字典
# 键(key)是标准分类，值(value)是所有可以映射到该分类的关键词列表
mapping_rules = {
    'T恤': ['t恤', 't-shirt', '体恤', 'polo'],
    '裤子': ['裤', 'jeans'], # “裤”字可以匹配所有类型的裤子
    '外套': ['外套', 'jacket', '羽绒服', '夹克', '风衣'],
    '衬衫': ['衬衫', 'shirt', '衬衣'],
    '袜子': ['袜'], # “袜”字可以匹配所有类型的袜子
    '帽子': ['帽']  # “帽”字可以匹配所有类型的帽子
}

# 2. 创建一个函数来应用这些规则
def clean_and_map_category(raw_category):
    # 确保输入是字符串
    if not isinstance(raw_category, str):
        return '未分类'

    # 基础清洗：转为小写，去除首尾空格
    clean_category = raw_category.lower().strip()

    # 应用映射规则
    for standard_category, keywords in mapping_rules.items():
        for keyword in keywords:
            if keyword in clean_category:
                return standard_category # 找到匹配的关键词，立即返回标准分类
    
    # 如果所有规则都匹配不上，返回“未分类”
    return '未分类'

# 3. 将函数应用到 '产品类别' 列，并创建新的一列
df['标准类别'] = df['产品类别'].apply(clean_and_map_category)

# 4. 检验清洗结果
print("清洗和分类后的结果统计:")
print(df['标准类别'].value_counts())

# 查看包含原始类别和标准类别的对比
print("\n部分数据清洗前后对比:")
print(df[['产品类别', '标准类别']].head(15))
```

### 代码解释

1. **`mapping_rules` 字典**：这是我们清洗逻辑的核心。我们把同一大类的关键词都放在一起，这样非常容易维护和扩展。例如，所有和“裤子”相关的词都检查是否存在“裤”或“jeans”即可。
    
2. **`clean_and_map_category` 函数**：
    
    - 首先进行基础的清洗（转小写、去空格），这能解决很多大小写和空格不统一的问题。
        
    - 然后，它会遍历我们的规则字典，检查输入的原始类别中是否**包含**任何一个关键词。
        
    - 一旦找到匹配，它就立刻返回对应的标准分类（例如，只要发现 `t恤衫` 中包含 `t恤`，就立刻返回 `'T恤'`）。
        
    - 如果循环结束都没有找到任何匹配的关键词，就返回 `'未分类'`，方便我们后续对这些“硬骨头”进行排查。
        
3. **`.apply()` 方法**：这是 Pandas 中将一个函数应用到一整列数据上的标准方法，非常高效。
    

通过这个流程，您就可以将原来混乱的几十种类别，清晰地归类到我们预先定义的6个标准类别中。
