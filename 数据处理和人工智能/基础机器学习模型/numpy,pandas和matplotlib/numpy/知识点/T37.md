这个问题和上一题非常相似，但我们需要的不再是最大值本身，而是最大值所在的位置（索引）。NumPy为此提供了一个专门的函数：`.argmax()`。

---

### 核心函数：`.argmax()`

`argmax` 的意思是 “**arg**ument of the **max**imum”，即“最大值的**参数（索引）**”。它返回的是最大值**第一次出现位置的索引**，而不是最大值本身。

它同样拥有 `axis` 参数，并且其工作原理与 `.max()` 完全一样：

- **`axis=1`**: 找出每一行中最大值的**列索引**。
    
- **`axis=0`**: 找出每一列中最大值的**行索引**。
    
- **不指定 `axis`**: 找出整个数组（扁平化后）最大值的索引。
    

---

### 解决方案：找出每行最大值的索引 (`axis=1`)

要找出每一行的最大值所在的位置，我们只需将 `.max(axis=1)` 换成 `.argmax(axis=1)` 即可。

**代码如下：**

Python

```
import numpy as np

# 重新创建 sales_data 数组
np.random.seed(42)
sales_data = np.random.randint(50, 200, size=(12, 12))

# 使用 .argmax() 方法并指定 axis=1
indices_of_max_in_each_row = sales_data.argmax(axis=1)

# 为了方便验证，我们打印一下原始数组和结果
print("原始数组 (为了简洁，只显示前5行):")
print(sales_data[:5])
print("-" * 50)
print("每行最大值的位置 (列索引):")
print(indices_of_max_in_each_row)
```

**输出结果：**

```
原始数组 (为了简洁，只显示前5行):
[[102 173 131 112 107  80 134 185 102 119 162 181]
 [185 174 180 113 195  98 102 152 119 193 159 101]
 [119 109 112  84 159  60 123 113 183 115 117 103]
 [175 192 108 195 119  88 121 160 170 161 166 189]
 [132 110 115 175 183 199 152 181 187 194 172 154]]
--------------------------------------------------
每行最大值的位置 (列索引):
[ 7  4  8  3  5  5  6  8  4  5  4  7]
```

**结果解释：**

- 返回的数组 `[ 7 4 8 ...]` 包含了12个元素，每个元素对应一行的结果。
    
- 第一个值 `7` 表示：在原始数组的第1行（索引0）中，最大值出现在第8个位置（**列索引为7**）。
    
- 第二个值 `4` 表示：在原始数组的第2行（索引1）中，最大值出现在第5个位置（**列索引为4**）。
    
- 以此类推。
    

---

### 如何验证结果？

我们可以利用得到的索引，回头去原始数组中把这些最大值取出来，看看是否和上一题用 `.max(axis=1)` 得到的结果一致。这是一个非常好的练习，可以组合运用我们之前学过的索引知识！

Python

```
# 验证第一行的结果
first_row = sales_data[0]
max_index_in_first_row = indices_of_max_in_each_row[0] # 值为 7
value_at_that_index = first_row[max_index_in_first_row]
actual_max_in_first_row = sales_data[0].max()

print(f"\n验证第一行：")
print(f"argmax() 找到的最大值索引是: {max_index_in_first_row}")
print(f"该索引 {max_index_in_first_row} 对应的值是: {value_at_that_index}")
print(f"直接用 .max() 计算的最大值是: {actual_max_in_first_row}")

# 更进一步：使用花式索引一次性取出所有行的最大值
all_max_values_from_argmax = sales_data[np.arange(12), indices_of_max_in_each_row]
print("\n使用 argmax 结果一次性提取所有最大值:\n", all_max_values_from_argmax)

# 对比上一题的结果
print("\n直接使用 .max(axis=1) 计算的最大值:\n", sales_data.max(axis=1))

print("\n两个结果是否完全一致:", np.array_equal(all_max_values_from_argmax, sales_data.max(axis=1)))
```

**输出结果：**

```
验证第一行：
argmax() 找到的最大值索引是: 7
该索引 7 对应的值是: 185
直接用 .max() 计算的最大值是: 185

使用 argmax 结果一次性提取所有最大值:
 [185 195 183 195 199 198 180 191 182 198 194 188]

直接使用 .max(axis=1) 计算的最大值:
 [185 195 183 195 199 198 180 191 182 198 194 188]

两个结果是否完全一致: True
```

验证成功！

**总结：**

- `.max()` 返回**最大值**本身。
    
- `.argmax()` 返回最大值所在的**索引/位置**。
    

这两个函数通常成对使用，是数据分析中非常高频的操作。
