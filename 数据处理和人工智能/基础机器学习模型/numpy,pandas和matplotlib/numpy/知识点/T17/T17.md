好的，这是一个非常重要的数据清洗环节！找出缺失值（`np.nan`）的位置是后续进行填充或删除操作的第一步。

要解决这个问题，我们需要组合使用两个核心的 NumPy 函数：

1. `np.isnan()`: 用于判断数组中的每个元素是否为 `np.nan`。
    
2. `np.where()`: 用于找出满足条件的元素的索引。
    

---

### 方法一：使用 `np.isnan()` 配合 `np.where()` (最标准、最常用)

这个方法分为两步，逻辑非常清晰：首先判断哪些是 `nan`，然后找出它们的位置。

**代码如下：**

Python

```python
import numpy as np

# 首先，我们重新创建上一题的数组，确保结果可复现
# 我们使用一个固定的随机种子
np.random.seed(0) 
arr = np.arange(10, dtype=float)
nan_indices = np.random.choice(np.arange(10), size=5, replace=False)
arr[nan_indices] = np.nan

print("带有 NaN 的数组:", arr)
print("-" * 30)

# 步骤 1: 使用 np.isnan() 创建一个布尔掩码 (boolean mask)
#          NaN 的位置为 True，其他位置为 False
is_nan_mask = np.isnan(arr)
print("是否为 NaN 的布尔掩码:", is_nan_mask)

# 步骤 2: 使用 np.where() 找出掩码中为 True 的所有索引
nan_positions = np.where(is_nan_mask)
print("NaN 的位置 (使用 np.where):", nan_positions)

```

**输出结果：**

```
带有 NaN 的数组: [ 0. nan  2.  3. nan nan nan  7. nan  9.]
------------------------------
是否为 NaN 的布尔掩码: [False  True False False  True  True  True False  True False]
NaN 的位置 (使用 np.where): (array([1, 4, 5, 6, 8]),)
```

**结果解释：**

- `np.where()` 的返回结果是一个**元组 (tuple)**。
    
- 因为我们的 `arr` 是一个一维数组，所以这个元组里只有一个元素，这个元素就是包含所有 `NaN` 位置索引的数组 `array([1, 4, 5, 6, 8])`。
    
- 如果是二维数组，元组里就会有两个元素，分别对应行索引和列索引。
    

---

### 补充说明：布尔索引的作用

虽然 `np.where()` 是专门用来找索引的，但这里得到的布尔掩码 `is_nan_mask` 本身也非常强大，可以直接用于数据筛选。

Python

```
# is_nan_mask 是: [False  True False False ... ]
# 我们可以直接用它来筛选原数组，得到所有为 nan 的值
print("\n直接使用布尔掩码筛选出的元素:", arr[is_nan_mask])
```

**输出结果：**

```
直接使用布尔掩码筛选出的元素: [nan nan nan nan nan]
```

这证明了 `is_nan_mask` 準确地标记了所有 `NaN` 值。

---

### 总结

要找出 `np.nan` 在数组中的位置，最标准、最推荐的写法就是：

Python

```
nan_positions = np.where(np.isnan(arr))
```

这行代码简洁、高效且语义清晰，是您务必要掌握的组合技巧。
