创建一个三维的随机数组非常简单。根据您的需求，您可以创建浮点数（小数）或整数类型的随机数组。

---

### 方法一：创建浮点数随机数组 (数值在 0 到 1 之间)

这是最常见的做法，使用 `np.random.rand()` 函数。它会生成一个指定形状的数组，并用 `[0, 1)` 区间内的均匀分布随机浮点数填充。

**代码如下：**

Python

```python
import numpy as np

# 直接将维度作为参数传入 rand() 函数
# np.random.rand(d0, d1, d2, ...)
random_float_array = np.random.rand(3, 3, 3)

print("3x3x3 的浮点数随机数组：")
print(random_float_array)
```

**代码解释：**

- `np.random.rand(3, 3, 3)`: `rand` 函数接收维度作为独立的参数，这里 `3, 3, 3` 分别代表了三个轴的长度，NumPy 随即创建了一个 3x3x3 的数组。
    

---

### 方法二：创建整数随机数组 (在指定范围内)

如果您需要的是随机整数，可以使用 `np.random.randint()` 函数。您需要指定整数的范围和数组的形状。

**代码如下：**

Python

```python
import numpy as np

# 创建一个 3x3x3 的数组，数值为 [0, 100) 之间的随机整数
# np.random.randint(low, high, size)
random_int_array = np.random.randint(0, 100, size=(3, 3, 3))

print("3x3x3 的整数随机数组：")
print(random_int_array)
```

**代码解释：**

- `0`: 随机整数的下限（包含）。
    
- `100`: 随机整数的上限（**不包含**）。
    
- `size=(3, 3, 3)`: 数组的形状，需要以元组（tuple）的形式传入。
    

---

### 总结

- 如果需要 **0到1之间的随机小数**，使用 `np.random.rand(3, 3, 3)`。
    
- 如果需要 **指定范围内的随机整数**，使用 `np.random.randint(0, 100, size=(3, 3, 3))`。
    

您可以根据自己的具体需求选择使用。
