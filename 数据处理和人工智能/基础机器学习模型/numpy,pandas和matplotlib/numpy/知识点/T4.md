在 Python 中，提到创建矩阵 $A, b, C, D$，这就完全是 **NumPy** 的主场了！

在数学和编程的习惯里，大写字母（如 $A, C, D$）通常用来代表**二维矩阵**，而小写字母（如 $b$）通常用来代表**一维向量**。

在 NumPy 中创建矩阵有几种非常常用的方式，你可以直接在 Jupyter 中运行下面的代码试试看：

Python

```
import numpy as np

# ---------------------------------------------------
# 方式 1：手动输入具体数值（使用 np.array）
# ---------------------------------------------------

# 创建矩阵 A (一个 2行2列 的矩阵)
# 写法：在最外层方括号内，再嵌套方括号，里面的每一个方括号代表矩阵的一行
A = np.array([[1, 2],
              [3, 4]])

# 创建向量 b (一个一维数组，通常用小写)
b = np.array([5, 6])


# ---------------------------------------------------
# 方式 2：使用快捷函数生成特定矩阵（非常适合初始化）
# ---------------------------------------------------

# 创建矩阵 C (例如：一个 3行3列 的全零矩阵)
# 注意里面的括号是两层：np.zeros((行数, 列数))
C = np.zeros((3, 3))

# 补充：如果你想要全 1 矩阵，可以用 np.ones((3, 3))


# ---------------------------------------------------
# 方式 3：结合你刚才学过的随机数生成矩阵
# ---------------------------------------------------

np.random.seed(42)
# 创建矩阵 D (例如：一个 2行3列 的随机整数矩阵，数值在 1 到 10 之间)
# 秘诀在于加上 size=(行, 列) 这个参数
D = np.random.randint(low=1, high=10, size=(2, 3))


# 打印出来看看它们的庐山真面目
print("=== 矩阵 A ===")
print(A)

print("\n=== 向量 b ===")
print(b)

print("\n=== 矩阵 C ===")
print(C)

print("\n=== 矩阵 D ===")
print(D)
```

### 💡 进阶小贴士：矩阵乘法

既然创建了矩阵，最常见的操作可能就是**矩阵相乘**了。在 NumPy 中，千万不要直接用 `*` 号去乘两个矩阵（那是对应位置元素相乘），而是应该使用 `@` 符号或者 `np.dot()`。

比如，计算矩阵 $A$ 乘以向量 $b$：

Python

```
# 矩阵 A 乘以向量 b
result = A @ b
print("A 乘以 b 的结果:", result)
```
