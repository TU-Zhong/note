要解决这个问题，我们需要使用 NumPy 中一个极其强大和核心的功能，叫做**布尔索引 (Boolean Indexing)** 或 **布尔掩码 (Boolean Masking)**。

这个过程在逻辑上分为两步，但写起来可以非常简洁。

---

### 分步理解布尔索引

#### 第一步：创建条件，生成布尔掩码

首先，我们不对数组进行任何操作，只是“质问”它：`sales_data > 150`。这个操作会返回一个和 `sales_data` **形状完全相同**的布尔数组（只包含 `True` 和 `False`）。

- 如果原始位置的元素大于150，那么新数组对应位置就是 `True`。
    
- 如果原始位置的元素不大于150，那么新数组对应位置就是 `False`。
    

这个布尔数组，我们就称之为**掩码 (mask)**。

**代码如下：**

Python

```
import numpy as np

# 重新创建 sales_data 数组
np.random.seed(42)
sales_data = np.random.randint(50, 200, size=(12, 12))

# 步骤 1: 创建条件，生成一个布尔掩码
mask = sales_data > 150

print("原始数组 (为了简洁，只显示前5行前5列):")
print(sales_data[:5, :5])
print("-" * 40)
print("生成的布尔掩码 (同样只显示前5行前5列):")
print(mask[:5, :5])
```

**输出结果：**

```
原始数组 (为了简洁，只显示前5行前5列):
[[102 173 131 112 107]
 [185 174 180 113 195]
 [119 109 112  84 159]
 [175 192 108 195 119]
 [132 110 115 175 183]]
----------------------------------------
生成的布尔掩码 (同样只显示前5行前5列):
[[False  True False False False]
 [ True  True  True False  True]
 [False False False False  True]
 [ True  True False  True False]
 [False False False  True  True]]
```

您可以看到，掩码的形状和原数组完全对应，`True` 的位置就是原数组中数值大于150的地方。

#### 第二步：将掩码应用到原数组上

现在，我们把这个“面具”（掩码）盖在原始数据上，只让 `True` 位置的数字“露”出来。语法就是 `sales_data[mask]`。

**代码如下：**

Python

```
# 步骤 2: 将掩码应用到原数组上，提取所有为 True 位置的元素
high_sales = sales_data[mask]

print("\n所有大于150的销售数据:")
print(high_sales)
```

**输出结果：**

```
所有大于150的销售数据:
[173 185 174 180 195 159 175 192 195 175 183 152 152 182 191 193 183
 160 170 161 166 189 199 181 187 194 172 154 180 151 182 161 178 188
 198 162 171 188 191 151 179 174 194 151 168 155 198 196 153]
```

**注意**：返回的结果是一个**一维数组**，因为它只是把所有满足条件的元素收集起来，不再保留原始的二维形状。

---

### 合二为一：简洁的写法

在实际编码中，我们通常会把上面两步合成一步，这也是您最常见到的写法：

Python

```
# 将条件直接写在方括号里
high_sales_concise = sales_data[sales_data > 150]

print("\n使用简洁写法得到的结果:")
print(high_sales_concise)
```

这行代码 `sales_data[sales_data > 150]` 和我们分步操作得到的结果是完全一样的，但更加简洁。

### 补充技巧：不仅能提取，还能修改

布尔索引的强大之处还在于，它不仅能提取数据，还能直接对满足条件的元素进行**批量修改**。

例如，我们可以将所有大于180的“超级销售额”标记为 `-1`：

Python

```
# 创建一个副本进行修改，以免影响原始数据
sales_data_copy = sales_data.copy()

# 将所有大于180的元素赋值为-1
sales_data_copy[sales_data_copy > 180] = -1

print("\n修改后的数组 (部分显示):")
print(sales_data_copy[:5, :5])
```

这个技巧在数据预处理中非常有用。
