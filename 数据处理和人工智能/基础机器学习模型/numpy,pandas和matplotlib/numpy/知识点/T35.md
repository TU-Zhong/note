要实现这个目标，我们需要使用 NumPy 数组的 `.max()` 方法，并配合一个非常核心的参数——`axis`。这个参数能让聚合函数（如 `max`, `min`, `sum`, `mean` 等）沿着指定的轴进行运算。

---

### 核心概念：`axis` 参数

您可以把 `axis` 理解为“**沿着这个轴进行压缩/塌陷**”的方向。对于一个二维数组（比如我们的 `sales_data`，它有2个轴）：

- **`axis=0`** 指的是第一个轴，也就是**行**。当 `axis=0` 时，操作是**沿着行向下**进行的，将所有行“压缩”到一行，最终分别计算出**每一列**的最大值。
    
- **`axis=1`** 指的是第二个轴，也就是**列**。当 `axis=1` 时，操作是**沿着列向右**进行的，将所有列“压缩”到一列，最终分别计算出**每一行**的最大值。
    
- **不指定 `axis`**：默认情况下，函数会作用于整个数组，计算出所有元素中的最大值，返回一个单一的数值。
    

简单来说，对于二维数组，**指定哪个轴，哪个轴就会“消失”**。

---

### 解决方案：找出每行的最大值 (`axis=1`)

根据上面的解释，要找出“每行”的最大值，我们需要沿着列的方向（`axis=1`）进行压缩。

**代码如下：**

Python

```
import numpy as np

# 重新创建 sales_data 数组
np.random.seed(42)
sales_data = np.random.randint(50, 200, size=(12, 12))

# 使用 .max() 方法并指定 axis=1
max_in_each_row = sales_data.max(axis=1)

# 为了方便验证，我们打印一下原始数组和结果
print("原始数组 (为了简洁，只显示前5行):")
print(sales_data[:5])
print("-" * 50)
print("每行的最大值:")
print(max_in_each_row)
print("\n结果的形状:", max_in_each_row.shape)
```

**输出结果：**

```
原始数组 (为了简洁，只显示前5行):
[[102 173 131 112 107  80 134 185 102 119 162 181]
 [185 174 180 113 195  98 102 152 119 193 159 101]
 [119 109 112  84 159  60 123 113 183 115 117 103]
 [175 192 108 195 119  88 121 160 170 161 166 189]
 [132 110 115 175 183 199 152 181 187 194 172 154]]
--------------------------------------------------
每行的最大值:
[185 195 183 195 199 198 180 191 182 198 194 188]

结果的形状: (12,)
```

**结果解释：**

- `.max(axis=1)` 对 `sales_data` 的每一行都进行了计算，找出了该行的最大值。
    
- 最终返回的是一个包含12个元素的一维数组，每个元素分别对应原始数组中每一行的最大值。例如，第一个值 `185` 是原始数组第一行中的最大值。
    

### 补充示例：找出每列的最大值 (`axis=0`)

为了加深您对 `axis` 的理解，我们可以看一下如果设置 `axis=0` 会发生什么。

Python

```
# 使用 .max() 方法并指定 axis=0
max_in_each_col = sales_data.max(axis=0)

print("\n每列的最大值:")
print(max_in_each_col)
print("\n结果的形状:", max_in_each_col.shape)
```

**输出结果：**

```
每列的最大值:
[185 192 180 195 195 199 179 198 191 194 178 196]

结果的形状: (12,)
```

您可以看到，这次它返回的也是一个有12个元素的一维数组，但每个元素对应的是原始数组中**每一列**的最大值。

总结一个简单的规则帮助记忆：

对于我们这个 (行, 列) 形状的二维数组：

- `axis=1` -> 操作**行**（压缩列）
    
- `axis=0` -> 操作**列**（压缩行）
