早安！這是一個很經典的 NumPy 練習題，可以很好地考察對**數組創建和切片（slicing）**的理解。

有幾種方法可以實現，這裡為您介紹兩種最常見的。

---

### 方法一：先創建全1數組，再將內部置零（最直觀）

這個思路非常直接：先創建一個符合外層要求的 10x10 全1數組，然後選取中間 8x8 的區域，把它們全部變成 0。

**代碼如下：**

```Python
import numpy as np

# 1. 創建一個 10x10 的全為 1 的數組
ones_array = np.ones((10, 10), dtype=int) # 使用 dtype=int 讓它顯示為整數 1

# 2. 選取內部區域，並將其賦值為 0
#    內部區域的行索引是從 1 到 8 (不包含10)
#    內部區域的列索引也是從 1 到 8 (不包含10)
ones_array[1:-1, 1:-1] = 0

# 打印結果
print(ones_array)
```
**代碼解釋：**

- `np.ones((10, 10), dtype=int)`: 創建一個 10x10，所有元素都是整數 `1` 的數組。
    
- `ones_array[1:-1, 1:-1] = 0`: 這是最關鍵的一步。
    
    - 切片 `1:-1` 是一個非常 Pythonic 的用法，它表示“從索引1開始，到倒數第一個元素為止（不包含倒數第一個）”。
        
    - `1:-1` 用在行上，就選取了第2行到第9行（索引1到8）。
        
    - `1:-1` 用在列上，就選取了第2列到第9列（索引1到8）。
        
    - `[1:-1, 1:-1]` 合在一起就精確地選中了中間 8x8 的區域。
        

---

### 方法二：先創建全0數組，再填充邊界（較靈活）

這個思路是反過來的：先創建一個 10x10 的全0數組，然後通過切片分別將四條邊賦值為 1。

**代碼如下：**
```python
import numpy as np

# 1. 創建一個 10x10 的全為 0 的數組
zeros_array = np.zeros((10, 10), dtype=int)

# 2. 將四條邊界賦值為 1
zeros_array[0, :] = 1   # 第一行
zeros_array[-1, :] = 1  # 最後一行
zeros_array[:, 0] = 1   # 第一列
zeros_array[:, -1] = 1  # 最後一列

# 打印結果
print(zeros_array)
```
**代碼解釋：**

- `:` 在切片中表示“選取該軸上的所有元素”。
    
- `zeros_array[0, :] = 1` 的意思就是“選取第1行（索引為0）的所有列，並把它們的值設為1”。
    

---

### 方法三：使用 `np.pad()` 函數（最高效）

這是更 "NumPy風格" 的做法，直接表達了「為一個數組添加邊框」的思想。

**代碼如下：**
```python
import numpy as np

# 1. 先創建 8x8 的內部全0數組
inner_zeros = np.zeros((8, 8), dtype=int)

# 2. 使用 np.pad() 在外圍用常數 1 進行填充
#    pad_width=1 表示在每個軸的兩邊都填充1個寬度
padded_array = np.pad(inner_zeros, pad_width=1, mode='constant', constant_values=1)

# 打印結果
print(padded_array)
```
**總結：** 對於初學者，**方法一最為直觀且常用**，能很好地幫您理解切片操作。方法三 `np.pad()` 則在處理更複雜的邊界填充問題時顯得更為強大和方便。
