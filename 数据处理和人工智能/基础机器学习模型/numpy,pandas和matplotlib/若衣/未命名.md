这份400题的题单专为构建**高可用企业级系统、深入理解 RuoYi-Vue 及其增强版 RuoYi-Vue-Plus 源码、应对严苛的后端研发需求**而设计。

题单从基础架构与环境配置入手，深入底层权限模型、数据访问机制（JDBC元数据/动态驱动/锁），再扩展到多数据源架构、高并发中间件，最后落地于容器化部署与线上排查。建议结合源码阅读与本地开发环境实战演练。

### **准备工作：环境与工具**

- **开发环境**: JDK 8/17/21 (视具体版本而定), IntelliJ IDEA, Maven 3.6+。
    
- **部署环境**: 建议在 Ubuntu 或 WSL 2 下结合 Docker / Docker Compose 进行 MySQL、Redis、Nacos 等中间件的快速部署与网络调试。
    
- **必备工具**: Postman/Apifox, DataGrip/Navicat, Git。
