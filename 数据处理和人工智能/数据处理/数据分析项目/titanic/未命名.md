### **方法三：终极挑战 - 用 NumPy 复现 Pandas 的 `groupby` 功能**

这是我最推荐给你的方法，因为它能让你深刻理解 NumPy 和 Pandas 的内在联系。

**目标：** 我们来手动计算一下泰坦尼克号数据集中，**不同性别 (Sex) 和不同船舱等级 (Pclass) 的乘客的平均票价 (Fare)**。这个任务用 Pandas 的 `groupby` 一行就能搞定，但这次我们**只用 NumPy 来完成计算部分**。

**挑战步骤：**

1. **准备数据**：先用 Pandas 加载数据，然后把需要的列抽出来，转换成 NumPy 数组。
    
    Python
    
    ```
    import pandas as pd
    import numpy as np
    
    # 只需用 Pandas 加载数据
    df = pd.read_csv("train.csv")
    
    # 把我们需要的列转换成纯 NumPy 数组
    sex_arr = df['Sex'].values
    pclass_arr = df['Pclass'].values
    fare_arr = df['Fare'].values
    
    print("数据已转换为 NumPy 数组！")
    ```
    
2. **筛选数据 (核心)**：使用 NumPy 的**布尔索引 (Boolean Indexing)** 来筛选出符合条件的数据。
    
    - 找出所有“头等舱的女性” (`Pclass=1`, `Sex='female'`) 的票价。
        
    - **提示**：你需要创建两个布尔数组（一个是判断 `pclass_arr == 1`，另一个是判断 `sex_arr == 'female'`），然后用 `&` (与) 操作符合并它们，生成一个最终的“掩码”(mask)。
        
    
    Python
    
    ```
    # 示例：如何找到头等舱女性的票价
    mask_female = (sex_arr == 'female')
    mask_pclass1 = (pclass_arr == 1)
    
    # & 用于合并两个条件
    final_mask = mask_female & mask_pclass1
    
    # 使用这个最终的 mask 来从 fare_arr 中选取票价
    pclass1_female_fares = fare_arr[final_mask]
    ```
    
3. **计算结果**：对筛选出来的票价数组求平均值。
    
    Python
    
    ```
    # 计算平均票价
    avg_fare = pclass1_female_fares.mean()
    print(f"头等舱女性的平均票价 (NumPy 计算): {avg_fare}")
    ```
    
4. **验证**：用你熟悉的 Pandas `groupby` 来验证你的计算结果是否正确。
    
    Python
    
    ```
    # 用 Pandas 来验证
    pandas_result = df.groupby(['Pclass', 'Sex'])['Fare'].mean()
    print("\n用 Pandas groupby 计算的结果:")
    print(pandas_result)
    ```
