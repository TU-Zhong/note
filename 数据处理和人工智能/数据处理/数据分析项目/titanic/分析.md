### **Titanic 数据分析完整流程**

1. **明确目标**: 我们的核心目标是**预测乘客的生还情况 (`Survived`)**。因此，所有分析都应围绕“什么因素会影响生还率”来展开。

4. **探索性数据分析 (EDA)**: 通过可视化等手段，深入探索特征与“生还”之间的关系。
    
5. **模型训练与预测 (后续步骤)**: 基于分析结果，建立机器学习模型进行预测。

### **步骤 1: 加载与探索数据**

首先，我们需要加载数据并对它有一个宏观的认识。`train.csv` 是我们的主要分析对象，因为它包含了“生还”与否的标签。`test.csv` 用于模型训练好之后提交结果，我们先用`train.csv`来分析。

我将为你编写代码来执行以下操作：

1. 导入必要的库 (`pandas`, `matplotlib`, `seaborn`)。
    
2. 加载 `train.csv` 和 `test.csv` 文件。
    
3. 显示数据的前几行、基本信息（列名、数据类型、缺失值）和数值特征的统计摘要。

### **步骤 2: 探索性数据分析 (EDA) 与数据可视化**

现在，我们要深入挖掘数据，看看哪些特征对生还率有显著影响。可视化是这一步最有力的工具。

#### **分析问题 1: 性别(Sex)与生还率的关系？**

“女士优先”是泰坦尼克号上著名的口号，我们来看看数据是否支持这一点。

Python

```python
if not train_df.empty:
    fig, ax = plt.subplots(1, 2, figsize=(12, 5))

    # 1. 不同性别的生还人数
    sns.countplot(x='Sex', hue='Survived', data=train_df, ax=ax[0])
    ax[0].set_title('Survival Count by Sex')
    ax[0].set_xlabel('Sex')
    ax[0].set_ylabel('Count')
    ax[0].legend(title='Survived', labels=['No', 'Yes'])

    # 2. 不同性别的生还率
    # 使用 groupby 计算每个性别的平均生还率
    sex_survival_rate = train_df.groupby('Sex')['Survived'].mean().reset_index()
    sns.barplot(x='Sex', y='Survived', data=sex_survival_rate, ax=ax[1])
    ax[1].set_title('Survival Rate by Sex')
    ax[1].set_xlabel('Sex')
    ax[1].set_ylabel('Survival Rate')
    # 在柱子上显示百分比
    for index, row in sex_survival_rate.iterrows():
        ax[1].text(row.name, row.Survived, f'{row.Survived:.2%}', color='black', ha="center")

    plt.tight_layout()
    plt.savefig('survival_by_sex.png')
    print("\n生成图像 'survival_by_sex.png'")
```
