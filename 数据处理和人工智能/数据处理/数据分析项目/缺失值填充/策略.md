好的，这是一个非常核心且实用的数据分析问题。处理缺失值（Missing Values）没有一劳永逸的“最佳”方法，正确的策略取决于多种因素，例如：数据的类型、缺失的比例、数据的业务含义以及你最终要使用的模型。

下面我为您列出了一套从简单到复杂的常用缺失值填充策略，并说明了它们各自的适用场景、优缺点和 Python 实现方法。

---

### **1. 删除法 (Deletion)**

最简单直接的方法，但代价是损失数据。

#### **策略 1.1: 删除整行 (Listwise Deletion)**

如果某一行中包含一个或多个缺失值，就将这一整行数据全部删除。

- **适用场景**:
    
    - 缺失值的数量非常少（通常小于总数据量的5%）。
        
    - 缺失是完全随机的，删除后不会对整体数据分布产生大的影响。
        
- **优点**: 实现简单，不会引入虚假的数据。
    
- **缺点**: 如果缺失值很多，会损失大量宝贵的数据。
    
- **Python 实现**:
    
    Python
    
    ```
    # df 是你的 DataFrame
    df_dropped_rows = df.dropna()
    ```
    

#### **策略 1.2: 删除整列 (Dropping Columns)**

如果某一列（特征）的缺失值比例非常高（例如超过50%或60%），说明这个特征本身包含的有效信息就很少，可以考虑直接删除这一整列。

- **适用场景**:
    
    - 某一列的缺失值比例过高。
        
    - 该特征与你要预测的目标关系不大。
        
- **优点**: 可以减少特征数量，简化模型，避免引入大量不确定的填充值。
    
- **缺点**: 可能会丢失一些有用的信息。
    
- **Python 实现**:
    
    Python
    
    ```
    # 删除 'Cabin' 这一列
    df_dropped_cols = df.drop('Cabin', axis=1)
    ```
    

---

### **2. 简单统计量填充法 (Simple Imputation)**

用数据的某个统计特征来填充缺失值，适用于快速建模。

#### **策略 2.1: 均值 / 中位数 / 众数填充**

- **均值填充 (Mean)**:
    
    - **适用场景**: 针对**数值型**数据，且数据分布比较对称（类似正态分布）。
        
    - **缺点**: 对异常值（outliers）非常敏感。一个极端高或低的值会严重影响均值。
        
    - **Python 实现**: `df['Age'].fillna(df['Age'].mean())`
        
- **中位数填充 (Median)**:
    
    - **适用场景**: 针对**数值型**数据，特别是当数据分布倾斜或**存在异常值**时。中位数更能代表数据的集中趋势。
        
    - **缺点**: 可能会丢失一些关于数据分布的信息。
        
    - **Python 实现**: `df['Age'].fillna(df['Age'].median())`
        
- **众数填充 (Mode)**:
    
    - **适用场景**: 针对**分类型**数据（Categorical Data）。
        
    - **缺点**: 如果数据分布很平均，众数可能没有代表性。
        
    - **Python 实现**: `df['Embarked'].fillna(df['Embarked'].mode()[0])` (记得加 `[0]`)
        

#### **策略 2.2: 固定值填充 (Constant Value)**

用一个特定的值来填充所有的缺失。

- **适用场景**:
    
    - **数值型**: 当 `0` 有特殊业务含义时（例如，缺失的销售额可以看作0）。
        
    - **分类型**: 当缺失本身可能是一种信息时，可以用一个特定的字符串（如 "Unknown", "Missing"）来填充。这相当于**创建了一个新的类别**，让模型可以学习“缺失”这个状态所代表的规律。
        
- **优点**: 实现简单，可以保留“缺失”这个信息。
    
- **缺点**: 如果固定值选择不当，可能会对模型产生误导。
    
- **Python 实现**:
    
    Python
    
    ```
    # 数值型
    df['Sales'].fillna(0, inplace=True)
    # 分类型
    df['Color'].fillna('Unknown', inplace=True)
    ```

### **3. 高级填充法 (Advanced Imputation)**

这些方法更复杂，但通常能更好地保留数据的原始结构和关系。

#### **策略 3.1: 时间序列数据的特殊填充**

对于按时间排序的数据，我们可以利用其顺序性。

- **前向填充 (Forward Fill)**: 用上一个有效观测值来填充。
    
    - **适用场景**: 假设观测值在一段时间内保持不变（例如，每日库存量）。
        
    - **Python 实现**: `df['Stock_Price'].fillna(method='ffill')`
        
- **后向填充 (Backward Fill)**: 用下一个有效观测值来填充。
    
    - **适用场景**: 当我们认为未来的值更能代表当前缺失的状态时。
        
    - **Python 实现**: `df['Stock_Price'].fillna(method='bfill')`
        

#### **策略 3.2: 基于模型的填充**

利用其他特征来预测缺失值。

- **K-近邻填充 (KNN Imputation)**:
    
    - **思想**: “物以类聚”。找到与包含缺失值的样本最相似的 K 个样本，用这 K 个样本的均值（数值型）或众数（分类型）来填充。
        
    - **优点**: 比简单的统计量填充更精确，因为它考虑了样本间的相似性。
        
    - **缺点**: 计算量大，特别是对于大数据集。
        
    - **Python 实现 (使用 scikit-learn)**:
        
        Python
        
        ```
        from sklearn.impute import KNNImputer
        imputer = KNNImputer(n_neighbors=5)
        df_filled = imputer.fit_transform(df[['Age', 'Fare', 'Pclass']]) # 只能用于数值列
        ```
        
- **回归/分类模型填充**:
    
    - **思想**: 将含有缺失值的列作为**目标 (y)**，其他列作为**特征 (X)**，训练一个模型（如线性回归、随机森林）来预测缺失值。**我们在泰坦尼克号项目中对年龄的处理就是这种思想的简化版。**
        
    - **优点**: 可能是最准确的填充方法之一，因为它充分利用了特征间的关系。
        
    - **缺点**: 实现复杂，如果多个列都有缺失，需要构建多个模型。
        

#### **策略 3.3: 多重插补法 (Multivariate Imputation)**

这是目前最强大和推荐的高级方法之一。

- **思想**: 比如 MICE (Multivariate Imputation by Chained Equations) 算法。它将含有缺失值的列轮流作为目标，用其他所有列来对其进行回归预测。这个过程会迭代多次，直到所有缺失值都收敛。
    
- **优点**: 能够很好地保留数据的不确定性和变量间的相关性，结果非常稳健。
    
- **缺点**: 计算非常密集，耗时较长。
    
- **Python 实现 (使用 scikit-learn)**:
    
    Python
    
    ```
    from sklearn.experimental import enable_iterative_imputer
    from sklearn.impute import IterativeImputer
    imputer = IterativeImputer(max_iter=10, random_state=0)
    df_filled = imputer.fit_transform(df)
    ```
    

### **总结与选择建议**

|策略类别|何时使用？|
|---|---|
|**删除法**|快速验证想法，或缺失比例极低/极高时。|
|**简单统计量填充**|**最常用的基准方法**。快速、简单，适用于大多数初步分析和建模。|
|**固定值填充**|当“缺失”本身就是一种有意义的状态时。|
|**高级填充法**|当数据质量要求非常高，且计算资源充足时。特别是在最终模型交付阶段。|
