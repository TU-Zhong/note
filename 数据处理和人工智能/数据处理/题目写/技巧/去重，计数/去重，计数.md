好的，这是一个非常核心且常用的 Pandas 函数。

`.nunique()` 的全称是 **"Number of Unique"**，它的核心作用是：

**计算并返回一个序列（Series）或 DataFrame 列中，不重复值的数量。**

简单来说，就是“数一数有多少种不同的值”。

---

### 核心特性

1. **去重 (Unique)**：它只计算每个值一次，无论这个值出现了多少次。
    
2. **计数 (Number)**：它不返回那些不重复的值是什么，而是直接返回它们的**数量**（一个整数）。
    
3. **默认忽略缺失值 (NaN)**：在计算时，它默认会忽略掉 `NaN` (Not a Number) 这种缺失值。
    

---

### 实例说明

我们创建一个简单的 Pandas Series 来演示它的用法：

Python

```
import pandas as pd
import numpy as np # 用来创建 NaN

# 创建一个包含重复值和缺失值的 Series
s = pd.Series(['苹果', '香蕉', '苹果', '橙子', '香蕉', '苹果', np.nan, '菠萝'])

print("原始数据:")
print(s)
```

**原始数据:**

```
0     苹果
1     香蕉
2     苹果
3     橙子
4     香蕉
5     苹果
6    NaN  # 这是一个缺失值
7     菠萝
dtype: object
```

现在，我们对这个 Series 使用 `.nunique()`：

Python

```
# 计算不重复值的数量
unique_count = s.nunique()

print(f"\n不重复值的数量是: {unique_count}")
```

**运行结果:**

```
不重复值的数量是: 4
```

**为什么是 4？** 因为 `.nunique()` 找到了以下 4 种不同的、非空的值：

1. '苹果'
    
2. '香蕉'
    
3. '橙子'
    
4. '菠萝'
    

它自动忽略了重复出现的 '苹果'、'香蕉'，并且也忽略了那个 `NaN` 缺失值。

---

### 与相似函数的对比

为了更好地理解 `.nunique()`，我们把它和几个相似的函数做个比较：

|方法/属性|功能描述|对我们的示例 `s` 的结果|
|---|---|---|
|**`.nunique()`**|**计算不重复且非空值的数量**|`4`|
|**`.unique()`**|**返回不重复且非空值本身（一个数组）**|`['苹果', '香蕉', '橙子', '菠萝', nan]`|
|**`.count()`**|**计算所有非空值的数量（包含重复）**|`7` （6个水果 + 1个菠萝，忽略了NaN）|
|**`.size`**|**返回所有元素的总数量（包含重复和空值）**|`8` (所有行的总数)|

### 如何包含 `NaN` 进行计数？

`.nunique()` 有一个参数 `dropna`，默认是 `True`（丢弃`NaN`）。如果你想把 `NaN` 也当作一种特殊的不重复值来计算，可以设置为 `False`。

Python

```
# 把 NaN 也算作一个不重复的类别
unique_count_with_nan = s.nunique(dropna=False)

print(f"\n包含 NaN 在内的不重复值数量是: {unique_count_with_nan}")
```

**运行结果:**

```
包含 NaN 在内的不重复值数量是: 5
```

现在结果是 5，因为它计算了 '苹果', '香蕉', '橙子', '菠萝' 以及 `NaN` 这五种。

### 总结

在数据分析中，当你需要快速回答“某一列有多少个不同的类别？”、“有多少个不同的用户ID？”、“有多少种不同的产品？”这类问题时，`.nunique()` 就是最直接、最方便的函数。
