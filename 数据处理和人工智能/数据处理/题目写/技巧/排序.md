当然！`.sort_values()` 是 `pandas` 数据处理流程中最常用的函数之一。除了刚刚我们用到的按单列降序排列，它还有很多强大且常见的用法。

下面我为你总结一下，并通过一个统一的例子来展示。

假设我们有这样一个学生成绩的 DataFrame：

Python

```
import pandas as pd
import numpy as np

data = {
    '班级': ['B班', 'A班', 'B班', 'A班', 'B班'],
    '姓名': ['小红', '小明', '小李', '小华', '小刚'],
    '分数': [92, 85, 92, 78, np.nan], # 注意这里有一个缺失值 NaN
    '出勤率': [0.98, 0.95, 0.95, 0.91, 0.99]
}
df = pd.DataFrame(data)
print("原始数据:")
print(df)
```

**原始数据:**

```
   班级  姓名    分数   出勤率
0  B班  小红  92.0  0.98
1  A班  小明  85.0  0.95
2  B班  小李  92.0  0.95
3  A班  小华  78.0  0.91
4  B班  小刚   NaN  0.99
```

---

### 1. 按多列排序（主、次关键字）

这是最常见的进阶用法。当你希望在一个字段排序相同的情况下，再按另一个字段排序时，就可以这么做。

**场景**：先按“班级”升序（A班在前，B班在后），如果班级相同，再按“分数”降序（分数高的在前）。

**用法**：给 `by` 参数传递一个列名的**列表**。

Python

```
# 先按'班级'升序，再按'分数'降序
# 注意 ascending 也需要传递一个对应的布尔值列表
multi_sorted_df = df.sort_values(by=['班级', '分数'], ascending=[True, False])

print("\n1. 按多列排序 (班级升序, 分数降序):")
print(multi_sorted_df)
```

**结果**：

```
   班级  姓名    分数   出勤率
1  A班  小明  85.0  0.95  <-- A班先按分数高低排
3  A班  小华  78.0  0.91
0  B班  小红  92.0  0.98  <-- B班再按分数高低排
2  B班  小李  92.0  0.95
4  B班  小刚   NaN  0.99
```

- `by=['班级', '分数']`：排序的优先级是从左到右。
    
- `ascending=[True, False]`：列表中的 `True` 对应 `班级`（升序），`False` 对应 `分数`（降序）。
    

### 2. 控制缺失值（NaN）的位置

默认情况下，缺失值 `NaN` 会被放在排序的末尾。我们可以通过 `na_position` 参数来控制它。

**场景**：我们希望在按分数排序时，分数缺失的学生（小刚）排在最前面。

**用法**：设置 `na_position='first'`。

Python

```
# 按分数降序，但将缺失值放在最前面
nan_first_df = df.sort_values(by='分数', ascending=False, na_position='first')

print("\n2. 控制缺失值位置 (NaN 在前):")
print(nan_first_df)
```

**结果**：

```
   班级  姓名    分数   出勤率
4  B班  小刚   NaN  0.99  <-- NaN被提到了最前面
0  B班  小红  92.0  0.98
2  B班  小李  92.0  0.95
1  A班  小明  85.0  0.95
3  A班  小华  78.0  0.91
```

- `na_position` 可选值为 `'first'` 或 `'last'` (默认)。
    

### 3. 原地修改（In-place Sort）

通常情况下，`.sort_values()` 会返回一个**新**的排好序的 DataFrame，原始的 `df` 不会改变。如果你想直接在原始的 DataFrame 上进行修改以节省内存，可以使用 `inplace=True`。

**场景**：确定要对 `df` 本身进行排序，并且后续不再需要原始顺序。

**用法**：设置 `inplace=True`。

Python

```
# 复制一份 df 用于演示
df_copy = df.copy()

# 对 df_copy 本身进行排序，这个操作不会返回任何东西 (返回 None)
df_copy.sort_values(by='出勤率', ascending=False, inplace=True)

print("\n3. 原地修改 (df_copy 已被改变):")
print(df_copy)
```

**结果**：

```
   班级  姓名    分数   出勤率
4  B班  小刚   NaN  0.99
0  B班  小红  92.0  0.98
1  A班  小明  85.0  0.95
2  B班  小李  92.0  0.95
3  A班  小华  78.0  0.91
```

**注意**：因为是原地修改，所以这个函数调用本身不返回 DataFrame。在链式调用中要特别小心，因为它会中断链条。

### 4. 按索引（Index）排序

虽然 `.sort_values()` 也可以排索引，但 `pandas` 提供了更专用的 `.sort_index()` 方法，这通常是更好的选择。

**场景**：DataFrame 的行索引被打乱了，希望恢复其原始顺序（或按字母、数字顺序排列）。

**用法**：调用 `.sort_index()`。

Python

```
# 先打乱 df 的索引
shuffled_df = df.sample(frac=1)
print("\n4. 索引被打乱的数据:")
print(shuffled_df)

# 按索引升序排回
index_sorted_df = shuffled_df.sort_index()
print("\n按索引排序后的结果:")
print(index_sorted_df)
```

**结果**：

```
4. 索引被打乱的数据:
   班级  姓名    分数   出勤率
1  A班  小明  85.0  0.95
3  A班  小华  78.0  0.91
0  B班  小红  92.0  0.98
4  B班  小刚   NaN  0.99
2  B班  小李  92.0  0.95

按索引排序后的结果:
   班级  姓名    分数   出勤率
0  B班  小红  92.0  0.98
1  A班  小明  85.0  0.95
2  B班  小李  92.0  0.95
3  A班  小华  78.0  0.91
4  B班  小刚   NaN  0.99
```

### 总结

|主要参数|用途|示例|
|---|---|---|
|`by`|指定排序依据的列名|`by='分数'` 或 `by=['班级', '分数']`|
|`ascending`|指定升序或降序。可以是一个布尔值，也可以是布尔值列表|`ascending=False` 或 `ascending=[True, False]`|
|`na_position`|控制 `NaN` (缺失值) 的位置|`na_position='first'` (在前) 或 `'last'` (在后)|
|`inplace`|是否在原始 DataFrame 上直接修改|`inplace=True` (原地修改) 或 `False` (返回副本)|
|`ignore_index`|`True` 时，排序后的新索引会重置为 `0, 1, 2, ...`|`ignore_index=True`|
