### 步骤 2: 使用 `pivot_table` 进行重塑

`pivot_table` 函数是实现这个需求的核心。

- `index='id'`: 我们希望 `id` 列成为新表格的行索引。
    
- `columns='month'`: 我们希望 `month` 列中的唯一值（'Jan', 'Feb', 'Mar' 等）成为新表格的列。
    
- `values='revenue'`: 我们希望表格中的填充值来自 `revenue` 列。

```python
import pandas as pd
import numpy as np

# 1. 准备数据
data = {
    'id': [1, 2, 3, 1, 1],
    'revenue': [8000, 9000, 10000, 7000, 6000],
    'month': ['Jan', 'Jan', 'Feb', 'Feb', 'Mar']
}
department_df = pd.DataFrame(data)

# 2. 使用 pivot_table 进行重塑
pivot_df = department_df.pivot_table(index='id', columns='month', values='revenue')

# 3. 调整列名
pivot_df = pivot_df.add_suffix('_Revenue')

# 4. 确保所有月份的列都存在
all_months = [
    "Jan", "Feb", "Mar", "Apr", "May", "Jun", 
    "Jul", "Aug", "Sep", "Oct", "Nov", "Dec"
]
all_month_revenue_cols = [f'{month}_Revenue' for month in all_months]

# 使用 .reindex() 重新索引列，缺失的列会用 NaN 填充
final_df = pivot_df.reindex(columns=all_month_revenue_cols)

# 将 id 从索引恢复为普通列
final_df = final_df.reset_index()

# 将 pandas 默认的 NaN（浮点数类型）替换为 None 对象，以完全匹配您的要求
# 注意：这可能会导致列的数据类型变为 'object'
final_df = final_df.astype(object).replace(np.nan, None)


# 打印最终结果
print("最终格式化结果:")
print(final_df)
```
