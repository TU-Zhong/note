好的，当然！

`ndarray` 的属性就像是这个数组的**“身份证”或“规格说明书”**，它们描述了这个数组的基本信息，但不会改变数组本身的内容。通过这些属性，我们可以快速了解一个数组的结构、类型和大小。

这些属性在使用时**后面不加括号**，因为它们是描述信息，而不是一个需要执行的动作（函数/方法）。

---

我们先创建一个“主角”数组，然后用它来演示各个属性：

Python

```
import numpy as np

# 创建一个2行3列的二维数组作为我们的示例
a = np.array([[1, 2, 3],
              [4, 5, 6]])
```

这个数组 `a` 看起来是这样的：

```
[[1 2 3]
 [4 5 6]]
```

下面我们来逐个介绍它最重要的几个属性：

### 1. `.shape` (形状)

- **通俗解释**：这是最重要的属性，告诉你数组**在每个维度上有多少个元素**。可以理解为数组的“规格”或“尺寸”。
    
- **对于数组 `a`**：它有 2 行、3 列。
    
- **代码示例**：
    
    Python
    
    ```
    print(a.shape)
    ```
    
    **输出**：`(2, 3)`
    
    - 这是一个元组（tuple），第一个数字 `2` 代表第一个维度（行数），第二个数字 `3` 代表第二个维度（列数）。
        

### 2. `.ndim` (维度数量)

- **通俗解释**：告诉你这个数组**总共有几个维度**。
    
- **对于数组 `a`**：它有行和列两个维度。
    
- **代码示例**：
    
    Python
    
    ```
    print(a.ndim)
    ```
    
    **输出**：`2`
    
    - 这个数字其实就是 `.shape` 属性返回的元组中元素的个数 (`len(a.shape)`)。
        

### 3. `.size` (总元素数量)

- **通俗解释**：告诉你这个数组里**总共包含多少个元素**。
    
- **对于数组 `a`**：它总共有 2 x 3 = 6 个数字。
    
- **代码示例**：
    
    Python
    
    ```
    print(a.size)
    ```
    
    **输出**：`6`
    
    - 这个数字就是 `.shape` 属性元组中所有数字的乘积。
        

### 4. `.dtype` (数据类型)

- **通俗解释**：告诉你数组里存放的元素的**具体数据类型**。就像我们之前讨论的，是整数 (`int`)、浮点数 (`float`) 还是其他类型。
    
- **对于数组 `a`**：它里面装的都是整数。
    
- **代码示例**：
    
    Python
    
    ```
    print(a.dtype)
    ```
    
    **输出**：`int64` (或者 `int32`，取决于您的操作系统)
    
    - `int64` 表示一个64位的整数。
        

### 5. `.itemsize` (单个元素大小)

- **通俗解释**：告诉你数组中**每一个元素**在内存中占据多少空间，单位是**字节 (bytes)**。
    
- **对于数组 `a`**：它的类型是 `int64`，64位 (bit) 等于 8 字节 (bytes) (因为 1 字节 = 8 位)。
    
- **代码示例**：
    
    Python
    
    ```
    print(a.itemsize)
    ```
    
    **输出**：`8`
    

### 6. `.nbytes` (总字节大小)

- **通俗解释**：告诉你**整个数组**中的所有数据总共占据了多大的内存空间。
    
- **对于数组 `a`**：总共有 6 个元素，每个占 8 字节，所以总共是 6 * 8 = 48 字节。
    
- **代码示例**：
    
    Python
    
    ```
    print(a.nbytes)
    ```
    
    **输出**：`48`
    
    - 这个值恒等于 `.size` * `.itemsize`。
        

### 7. `.T` (转置)

- **通俗解释**：对于二维数组，这个属性可以**把行和列进行翻转**。这在线性代数中非常常用。
    
- **对于数组 `a`**：原来是 2x3 的矩阵，转置后会变成 3x2 的矩阵。
    
- **代码示例**：
    
    Python
    
    ```
    print("原始数组 a:\n", a)
    print("转置后的数组 a.T:\n", a.T)
    ```
    
    **输出**：
    
    ```
    原始数组 a:
     [[1 2 3]
      [4 5 6]]
    转置后的数组 a.T:
     [[1 4]
      [2 5]
      [3 6]]
    ```
    

---

### 总结表格

|属性 (Attribute)|通俗解释|对 `a = np.array([[1,2,3],[4,5,6]])` 的结果|
|---|---|---|
|**`a.shape`**|数组的规格尺寸 (几行几列)|`(2, 3)`|
|**`a.ndim`**|数组的维度数量|`2`|
|**`a.size`**|数组的总元素个数|`6`|
|**`a.dtype`**|数组元素的数据类型|`int64`|
|**`a.itemsize`**|每个元素占用的字节数|`8`|
|**`a.nbytes`**|整个数组占用的字节数|`48`|
|**`a.T`**|翻转行和列 (转置)|`[[1, 4], [2, 5], [3, 6]]`|
