好的，我们来系统地学习一下 NumPy 中的比较函数和相关操作。这部分知识是进行数据筛选和处理的关键，尤其是它会产生布尔数组（Boolean Arrays），这是实现条件索引（即“掩码”，Masking）的基础。

### 核心思想

NumPy 中的比较操作也是 **矢量化** 的。当你比较两个数组或一个数组与一个标量时，它会进行 **元素级 (element-wise)** 的比较，并返回一个与原数组形状相同的 **布尔数组**。数组中的每个元素都是 `True` 或 `False`，表示该位置的元素是否满足比较条件。

---

### 准备工作

和之前一样，我们先准备一些数据。

Python

```
import numpy as np

# 创建两个数组用于相互比较
x = np.array([1, 2, 3, 4, 5])
y = np.array([5, 4, 3, 2, 1])

# 创建一个用于和标量比较的数据
data = np.array([10, 2, 8, 5, 8])
```

---

### 1. 元素级比较 (Element-wise Comparisons)

这是最基础的比较方式，可以使用标准的比较运算符，也可以使用 NumPy 的函数。

#### 知识点

- **比较运算符**: `==` (等于), `!=` (不等于), `>` (大于), `<` (小于), `>=` (大于等于), `<=` (小于等于)。
    
- **NumPy 通用函数 (ufuncs)**: `np.equal`, `np.not_equal`, `np.greater`, `np.less`, `np.greater_equal`, `np.less_equal`。
    

通常直接使用运算符代码更简洁易读。

#### 例子

Python

```
# --- 数组与标量比较 ---
# 找出 data 中所有大于 5 的元素
mask_gt_5 = data > 5
print(f"data > 5  : {mask_gt_5}")
# 输出: [ True False  True False  True]

# 找出 data 中所有等于 8 的元素
mask_eq_8 = data == 8
print(f"data == 8 : {mask_eq_8}")
# 输出: [False False  True False  True]

# --- 数组与数组比较 ---
# 比较 x 和 y 中对应位置的元素
print(f"x > y     : {x > y}")
# 输出: [False False False  True  True]

print(f"x == y    : {x == y}")
# 输出: [False False  True False False]

# --- 使用 NumPy 函数 (效果同上) ---
print(f"np.greater(x, y): {np.greater(x, y)}")
# 输出: [False False False  True  True]
```

---

### 2. 数组整体比较

有时候我们想知道两个数组是否 **完全相同**，而不是逐个元素比较。

|函数 (Function)|功能说明|返回值|
|---|---|---|
|`np.array_equal()`|判断两个数组是否形状和元素都完全相同。|单个 `True` 或 `False`|
|`np.array_equiv()`|判断两个数组是否等效。如果形状不同，会先尝试广播(broadcast)。|单个 `True` 或 `False`|

Export to Sheets

**重要提示**: 你不能直接用 `if x == y:` 来判断两个数组是否相等，因为 `x == y` 返回的是一个布尔数组，在 `if` 语句中会引发 `ValueError`。

#### 例子

Python

```
x_copy = np.array([1, 2, 3, 4, 5])
z = np.array([[5, 5], [5, 5]])

# 检查 x 和 y 是否完全相等
print(f"np.array_equal(x, y): {np.array_equal(x, y)}") # -> False

# 检查 x 和 x_copy 是否完全相等
print(f"np.array_equal(x, x_copy): {np.array_equal(x, x_copy)}") # -> True

# 检查数组 z 和标量 5 是否等效 (5 会被广播成 [[5,5],[5,5]])
print(f"np.array_equiv(z, 5): {np.array_equiv(z, 5)}") # -> True
```

### 3. 逻辑运算 (Logical Operations)

我们可以将多个比较条件组合起来，形成更复杂的查询。

#### 知识点

- **不**能使用 Python 的 `and`, `or`, `not` 关键字，它们无法在 NumPy 数组上进行元素级逻辑运算。
    
- 必须使用位运算符：
    
    - `&` 对应 **与 (and)**
        
    - `|` 对应 **或 (or)**
        
    - `~` 对应 **非 (not)**
        
- **语法要点**: 每个比较条件都必须用括号 `()` 包起来，因为 `&`, `|` 的优先级高于 `>` , `<` 等。
    

#### 例子

Python

```
# --- 找出 data 中所有大于 3 且小于 9 的数 ---
# 错误写法: data > 3 & data < 9
# 正确写法:
mask_complex = (data > 3) & (data < 9)
print(f"(data > 3) & (data < 9): {mask_complex}")
# data: [10, 2, 8, 5, 8]
# data>3: [T, F, T, T, T]
# data<9: [F, T, T, T, T]
#  &结果: [F, F, T, T, T]
# 输出: [False False  True  True  True]

# --- 找出 data 中等于 2 或等于 10 的数 ---
mask_or = (data == 2) | (data == 10)
print(f"(data == 2) | (data == 10): {mask_or}")
# 输出: [ True  True False False False]

# --- 找出 data 中不等于 8 的数 ---
mask_not = ~(data == 8)
print(f"~(data == 8): {mask_not}")
# 输出: [ True  True False  True False]
```

### 4. 实用辅助函数

这些函数作用于布尔数组，用于快速判断数组的整体情况。

|函数 (Function)|功能说明|
|---|---|
|`np.any()`|检查数组中是否存在**至少一个** `True`。|
|`np.all()`|检查数组中是否**所有元素**都为 `True`。|

Export to Sheets

它们也支持 `axis` 参数，可以在多维数组上沿指定轴进行判断。

#### 例子

Python

```
# 创建一个布尔掩码
mask = data > 5 # -> [ True False  True False  True]

# data 中是否存在大于 5 的数?
print(f"np.any(mask): {np.any(mask)}") # -> True

# data 中是否所有数都大于 5?
print(f"np.all(mask): {np.all(mask)}") # -> False

# --- 在多维数组上使用 axis ---
matrix_mask = matrix_data > 5 # matrix_data 是上个话题里的二维数组
# [[F, F, F, F],
#  [F, T, T, T],
#  [T, T, T, T]]

# 检查每一列(axis=0)是否含有 True
print(f"np.any(matrix_mask, axis=0): {np.any(matrix_mask, axis=0)}") # -> [ True  True  True  True]

# 检查每一行(axis=1)是否全部为 True
print(f"np.all(matrix_mask, axis=1): {np.all(matrix_mask, axis=1)}") # -> [False False  True]
```

### 最终应用：布尔索引 (Boolean Indexing)

比较操作的最终目的，通常是为了 **筛选数据**。我们将上面生成的布尔数组（掩码）作为索引，就可以取出所有对应位置为 `True` 的元素。

#### 例子

Python

```
# 1. 找出 data 中所有大于 5 的值
mask = data > 5  # -> [ True False  True False  True]
print(f"大于5的值: {data[mask]}") # -> [10  8  8]

# 2. 一步到位：找出 data 中大于 3 且小于 9 的值
selected_data = data[(data > 3) & (data < 9)]
print(f"大于3且小于9的值: {selected_data}") # -> [8 5 8]

# 3. 也可以用于赋值：将 data 中所有等于 8 的值替换为 -1
data_copy = data.copy()
data_copy[data_copy == 8] = -1
print(f"将8替换为-1后: {data_copy}") # -> [10  2 -1  5 -1]
```

掌握了比较和逻辑运算，你就掌握了 NumPy 数据筛选的精髓。
