好的，我们来详细梳理 NumPy 中关于排序的函数和方法。排序是数据处理中的一个常规操作，无论是为了数据呈现，还是作为某些算法的预处理步骤，都非常重要。

NumPy 提供了几种灵活且高效的排序方法。

### 核心知识点：两种主要的排序方式

在 NumPy 中，你需要明确区分两种排序操作，这非常关键：

1. **`np.sort()` - 函数，返回副本**
    
    - 这是一个 NumPy 的顶级函数。
        
    - 它会返回一个 **新** 的、排好序的数组副本，**原始数组保持不变**。
        
2. **`ndarray.sort()` - 方法，原地修改**
    
    - 这是一个数组对象自身的方法。
        
    - 它会 **直接修改原始数组**，使其有序。
        
    - 这个方法的返回值为 `None`，所以你不能写 `new_arr = arr.sort()`。
        

---

### 准备工作

我们先创建一些用于演示的未排序数组。

Python

```
import numpy as np

# 一维数组
arr1d = np.array([3, 1, 4, 1, 5, 9, 2, 6])

# 二维数组
arr2d = np.random.randint(10, 100, size=(3, 4))
```

---

### 1. 基础排序 (一维数组)

#### 例子 1: `np.sort()` - 返回副本

Python

```
print(f"原始数组 arr1d: {arr1d}")

# 使用 np.sort() 函数
sorted_copy = np.sort(arr1d)

print(f"排序后的副本: {sorted_copy}")
print(f"原始数组 arr1d 仍然不变: {arr1d}")
```

**输出:**

```
原始数组 arr1d: [3 1 4 1 5 9 2 6]
排序后的副本: [1 1 2 3 4 5 6 9]
原始数组 arr1d 仍然不变: [3 1 4 1 5 9 2 6]
```

#### 例子 2: `ndarray.sort()` - 原地排序

Python

```
# 为了不影响原始 arr1d，我们创建一个副本进行演示
arr1d_copy = arr1d.copy()
print(f"即将被原地排序的数组: {arr1d_copy}")

# 使用 .sort() 方法
result = arr1d_copy.sort()

print(f"调用 .sort() 的返回值: {result}") # 注意返回值是 None
print(f"数组已被原地修改: {arr1d_copy}")
```

**输出:**

```
即将被原地排序的数组: [3 1 4 1 5 9 2 6]
调用 .sort() 的返回值: None
数组已被原地修改: [1 1 2 3 4 5 6 9]
```

### 2. 多维数组排序

对于二维或更高维度的数组，我们可以使用 `axis` 参数来控制排序的方向。

#### 知识点

- `axis=1` (默认值): 沿着**行**的方向，对每一**行**内部的元素进行独立排序。
    
- `axis=0`: 沿着**列**的方向，对每一**列**内部的元素进行独立排序。
    
- `axis=None`: 将数组**扁平化 (flatten)** 为一维数组后再进行排序。
    

#### 例子

Python

```
print("原始二维数组 arr2d:\n", arr2d)

# --- 按行排序 (axis=1) ---
sorted_rows = np.sort(arr2d, axis=1)
print("\n按行排序 (axis=1) 的结果:\n", sorted_rows)

# --- 按列排序 (axis=0) ---
sorted_cols = np.sort(arr2d, axis=0)
print("\n按列排序 (axis=0) 的结果:\n", sorted_cols)

# --- 扁平化后排序 (axis=None) ---
sorted_flat = np.sort(arr2d, axis=None)
print("\n扁平化后排序 (axis=None) 的结果:\n", sorted_flat)
```

**一个可能的输出:**

```
原始二维数组 arr2d:
 [[67 82 78 47]
 [88 73 13 81]
 [74 48 58 29]]

按行排序 (axis=1) 的结果:
 [[47 67 78 82]  # <- [67 82 78 47] 排序后
 [13 73 81 88]  # <- [88 73 13 81] 排序后
 [29 48 58 74]]  # <- [74 48 58 29] 排序后

按列排序 (axis=0) 的结果:
 [[67 48 13 29]  # <- 第1列 [67, 88, 74] 排序后
 [74 73 58 47]  # <- 第2列 [82, 73, 48] 排序后
 [88 82 78 81]]  # <- 第3列 [78, 13, 58] 排序后

扁平化后排序 (axis=None) 的结果:
 [13 29 47 48 58 67 73 74 78 81 82 88]
```

### 3. 间接排序 `argsort`

这是 NumPy 一个非常强大和有用的功能。`np.argsort()` **不返回排序后的值**，而是返回**原始数组中元素的索引**，按照这些索引来取值，就能得到一个有序的数组。

#### 知识点

- 返回值是**索引数组**。
    
- 常用于根据一个数组的顺序去排序另一个（或多个）相关的数组。
    

#### 例子

Python

```
# --- 基本用法 ---
data = np.array([30, 10, 40, 20])
# 索引:         0   1   2   3

# 获取排序后的索引
indices = np.argsort(data)
print(f"原始数据: {data}")
print(f"排序索引 (argsort): {indices}")
# 解释: 最小的数是10(索引1), 第二小是20(索引3), 然后是30(索引0), 最后是40(索引2)

# 使用索引来重建有序数组
print(f"通过索引排序后的数据: {data[indices]}") # -> [10 20 30 40]

# --- 实用案例：根据一个数组排序另一个数组 ---
names = np.array(['张三', '李四', '王五', '赵六'])
ages = np.array([35, 28, 42, 35])

# 我们想根据年龄从小到大排序，同时保持姓名和年龄的对应关系
# 1. 获取年龄的排序索引
age_sort_indices = np.argsort(ages)
print(f"\n年龄排序索引: {age_sort_indices}") # -> [1 0 3 2]

# 2. 使用这个索引来同时排序两个数组
sorted_names = names[age_sort_indices]
sorted_ages = ages[age_sort_indices]

print(f"按年龄排序后的姓名: {sorted_names}")
print(f"按年龄排序后的年龄: {sorted_ages}")
```

**输出:**

```
原始数据: [30 10 40 20]
排序索引 (argsort): [1 3 0 2]
通过索引排序后的数据: [10 20 30 40]

年龄排序索引: [1 0 3 2]
按年龄排序后的姓名: ['李四' '张三' '赵六' '王五']
按年龄排序后的年龄: [28 35 35 42]
```

注意，`王五` (42岁)排在了最后，`李四` (28岁)排在了最前。`张三`和`赵六`都35岁，他们的相对顺序可能因排序算法而异（见下文）。

### 4. 其他相关知识点

#### 降序排序

NumPy 的排序函数没有像 Python 列表那样的 `reverse=True` 参数。要实现降序排序，标准的做法是 **先升序排序，再反转数组**。

Python

```
# 降序排序
desc_sorted_arr = np.sort(arr1d)[::-1]
print(f"\n降序排序结果: {desc_sorted_arr}")
# 输出: [9 6 5 4 3 2 1 1]
```

#### `kind` 参数 - 选择排序算法

`np.sort` 和 `ndarray.sort` 都有一个 `kind` 参数，可以指定使用的排序算法。

- `'quicksort'` (默认): 快速排序，速度通常最快，但不稳定。
    
- `'mergesort'` / `'stable'`: 归并排序，是 **稳定排序**。
    
- `'heapsort'`: 堆排序。
    

**稳定排序 (Stable Sort)** 指的是，如果两个元素的值相等，那么在排序后，它们原来的相对前后顺序保持不变。这在 `argsort` 的例子中很重要，如果使用 `kind='stable'`，`张三`会稳定地排在`赵六`前面，因为它在原始数组中就靠前。
