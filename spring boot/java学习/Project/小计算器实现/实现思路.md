1. 首先使用JFromDesigner设计好面板
2. 在每个按钮上绑定事件，按下就会在上面的textField1中增加对应的字符或者函数字符串,并对当前的textField1中的字符串进行是否违法的检查，如果违法，就在textField2中输出错误信息；如果不违法，就计算结果然后输出到textField2中。
3. 分解textField1成数字，运算符，和函数名
4. 将3步骤得到的List进行处理，得到逆波兰表达式
	[[逆波兰表达式的规则]]
5. 对逆波兰表达式进行处理，得到最终结果，输出到textField2。
	非常重要的是划分两个参数的运算符和一个参数的运算符，分别进行处理。然后还要维护一个栈来进行运算。
		<mark style="background: #FFF3A3A6;">这个两个参数的，先弹出的在后面，后弹出的在前面，不要弄混。</mark>
	比如2/3,那么栈中2在前面，3在后面，使用时要注意位置。
