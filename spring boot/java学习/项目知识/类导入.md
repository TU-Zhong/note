在Java中，有一个基本规则：**命名包中的类无法导入或直接引用默认包（无包名）中的类**。这是语言设计的限制，不能通过导入语句解决。

要解决这个问题，你有两个选择：

## 选项1（推荐）：将Apple类移到适当的包中

例如，创建一个名为`model`的包，并将Apple类移动到该包中：

1. 在src目录下创建`model`包
2. 将Apple类移到该包中并添加包声明
3. 然后在当前类中导入：

<!-- replace lines 1 to 3 -->
```java
package ApplePredicate;

import model.Apple;

```
