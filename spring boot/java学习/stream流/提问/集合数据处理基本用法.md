**Stream API** 提供了丰富的方法来对集合中的数据进行处理，包括 **过滤**、**映射**、**排序**、**去重** 和 **分组**。以下是每种操作的详细说明和具体示例：

---

### 1. **过滤（Filter）**
**用途**：筛选出符合条件的元素。  
**方法**：`filter(Predicate<T> predicate)`  
**示例**：筛选出长度大于 3 的字符串。
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie", "David");

List<String> result = names.stream()
                           .filter(name -> name.length() > 3) // 过滤长度大于 3 的名字
                           .collect(Collectors.toList());

System.out.println(result); // 输出 [Alice, Charlie, David]
```

---

### 2. **映射（Map）**
**用途**：将元素转换为另一种形式。  
**方法**：`map(Function<T, R> mapper)`  
**示例**：将字符串列表转换为它们的长度。
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

List<Integer> lengths = names.stream()
                             .map(String::length) // 将名字映射为长度
                             .collect(Collectors.toList());

System.out.println(lengths); // 输出 [5, 3, 7]
```

---

### 3. **排序（Sort）**
**用途**：对元素进行排序。  
**方法**：`sorted()` 或 `sorted(Comparator<T> comparator)`  
**示例**：对字符串列表按长度排序。
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

List<String> sortedNames = names.stream()
                                .sorted(Comparator.comparingInt(String::length)) // 按长度排序
                                .collect(Collectors.toList());

System.out.println(sortedNames); // 输出 [Bob, Alice, Charlie]
```

---

### 4. **去重（Distinct）**
**用途**：去除重复元素。  
**方法**：`distinct()`  
**示例**：去除列表中的重复元素。
```java
List<String> names = Arrays.asList("Alice", "Bob", "Alice", "Charlie");

List<String> uniqueNames = names.stream()
                                .distinct() // 去重
                                .collect(Collectors.toList());

System.out.println(uniqueNames); // 输出 [Alice, Bob, Charlie]
```

---

### 5. **分组（Grouping）**
**用途**：将元素按条件分组。  
**方法**：`Collectors.groupingBy(Function<T, K> classifier)`  
**示例**：将字符串按长度分组。
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie", "David");

Map<Integer, List<String>> groupedNames = names.stream()
                                               .collect(Collectors.groupingBy(String::length)); // 按长度分组

System.out.println(groupedNames); // 输出 {3=[Bob], 4=[David], 5=[Alice], 7=[Charlie]}
```

---

### 6. **综合示例**
将以上操作结合起来，完成一个复杂的任务：  
**任务**：从列表中筛选出长度大于 3 的字符串，去重，转为大写，按长度分组。
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie", "David", "Alice", "Eve");

Map<Integer, List<String>> result = names.stream()
                                         .filter(name -> name.length() > 3) // 过滤长度大于 3 的名字
                                         .distinct() // 去重
                                         .map(String::toUpperCase) // 转为大写
                                         .collect(Collectors.groupingBy(String::length)); // 按长度分组

System.out.println(result); // 输出 {4=[DAVID, EVE], 5=[ALICE], 7=[CHARLIE]}
```

---

### 7. **其他常用方法**
| 方法名                | 说明                                                                 |
|-----------------------|--------------------------------------------------------------------|
| `limit(long maxSize)` | 截取流中的前 `maxSize` 个元素。                                        |
| `skip(long n)`        | 跳过流中的前 `n` 个元素。                                              |
| `peek(Consumer<T>)`   | 对流中的每个元素执行操作，通常用于调试。                                 |
| `flatMap(Function<T, Stream<R>>)` | 将流中的每个元素转换为一个流，然后将所有流合并为一个流。             |

#### 示例：
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

// 使用 limit 和 skip
List<String> result = names.stream()
                           .skip(1) // 跳过第一个元素
                           .limit(2) // 取前两个元素
                           .collect(Collectors.toList());

System.out.println(result); // 输出 [Bob, Charlie]

// 使用 flatMap
List<List<String>> nestedList = Arrays.asList(
    Arrays.asList("Alice", "Bob"),
    Arrays.asList("Charlie", "David")
);

List<String> flattened = nestedList.stream()
                                   .flatMap(List::stream) // 将嵌套列表扁平化
                                   .collect(Collectors.toList());

System.out.println(flattened); // 输出 [Alice, Bob, Charlie, David]
```

---

### 总结
通过 Stream API 的 **过滤**、**映射**、**排序**、**去重** 和 **分组** 等方法，可以轻松地对集合中的数据进行处理。这些方法的组合使用可以完成复杂的任务，同时让代码更简洁、易读。
