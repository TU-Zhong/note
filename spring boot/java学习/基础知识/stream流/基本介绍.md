`.stream()` 是 Java 8 引入的一个方法，用于将 **集合**（如 `List`、`Set`、`Map` 等）或 **数组** 转换为一个 **流**（`Stream`）。流是一种用于对数据进行 **高效、声明式处理** 的工具，支持链式操作（如过滤、映射、排序等）。

---

### 1. **`.stream()` 的作用**
- **将集合或数组转换为流**：为集合或数组提供流式操作的能力。
- **支持链式调用**：可以对流进行一系列的中间操作（如 `filter`、`map`、`sorted` 等）和终止操作（如 `collect`、`forEach`、`reduce` 等）。

- **延迟执行**：流的中间操作是惰性的，只有在终止操作调用时才会真正执行。

---

### 2. **流的特点**
- **不可重复使用**：流只能被消费一次，一旦终止操作执行完毕，流就会被关闭。
- **高效处理数据**：流支持并行操作（`parallelStream`），可以充分利用多核 CPU 的优势。
- **声明式编程**：只需告诉程序“做什么”，而不是“怎么做”。

---

### 3. **`.stream()` 的用法**
`.stream()` 方法可以从以下数据源创建流：
- **集合**：如 `List`、`Set`、`Map` 等。
- **数组**：通过 `Arrays.stream()` 将数组转换为流。
- **其他**：如文件、随机数生成器等。

#### 示例 1：从集合创建流
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

// 将 List 转换为流
Stream<String> stream = names.stream();

// 对流进行操作
List<String> result = stream.filter(name -> name.length() > 3) // 过滤
                            .map(String::toUpperCase)          // 映射
                            .collect(Collectors.toList());     // 收集为 List

System.out.println(result); // 输出 [ALICE, CHARLIE]
```

#### 示例 2：从数组创建流
```java
int[] numbers = {1, 2, 3, 4, 5};

// 将数组转换为流
IntStream stream = Arrays.stream(numbers);

// 对流进行操作
int sum = stream.filter(x -> x % 2 == 0) // 过滤偶数
                .sum();                  // 求和

System.out.println(sum); // 输出 6
```

---

### 4. **流的操作类型**
流的操作分为两类：
- **中间操作**：返回一个新的流，可以继续操作。例如 `filter`、`map`、`sorted` 等。
- **终止操作**：返回一个非流的结果或执行某些操作。例如 `collect`、`forEach`、`reduce` 等。

#### 示例：
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

// 中间操作：filter, map
// 终止操作：collect
List<String> result = names.stream() // 创建流
                           .filter(name -> name.length() > 3) // 中间操作：过滤
                           .map(String::toUpperCase)          // 中间操作：映射
                           .collect(Collectors.toList());     // 终止操作：收集为 List

System.out.println(result); // 输出 [ALICE, CHARLIE]
```

---

### 5. **流的并行处理**
通过 `.parallelStream()` 方法可以创建并行流，利用多核 CPU 提高处理效率。

#### 示例：
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie", "David", "Eve");

// 并行流处理
List<String> result = names.parallelStream() // 创建并行流
                           .filter(name -> name.length() > 3) // 过滤
                           .collect(Collectors.toList());     // 收集为 List

System.out.println(result); // 输出 [Alice, Charlie, David, Eve]
```

---

### 6. **流的延迟执行**
流的中间操作是惰性的，只有在终止操作调用时才会真正执行。

#### 示例：
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

Stream<String> stream = names.stream() // 创建流
                             .filter(name -> {
                                 System.out.println("过滤: " + name); // 不会立即执行
                                 return name.length() > 3;
                             });

System.out.println("流已创建，但未执行");

// 终止操作触发流的执行
List<String> result = stream.collect(Collectors.toList());

System.out.println(result); // 输出 [Alice, Charlie]
```

---

### 7. **总结**
`.stream()` 的作用是将集合或数组转换为流，从而支持流式操作。它的主要特点包括：
1. **链式调用**：可以流畅地组合多个操作。
2. **延迟执行**：中间操作是惰性的，只有在终止操作调用时才会执行。
3. **高效处理**：支持并行操作，提升性能。
4. **声明式编程**：只需告诉程序“做什么”，而不是“怎么做”。

通过 `.stream()`，可以轻松地对集合或数组进行过滤、映射、排序、去重、分组等操作，让代码更简洁、易读！
