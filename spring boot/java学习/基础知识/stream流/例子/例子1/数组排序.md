在Java中，`sort`方法接受的比较器(Comparator)定义排序顺序是通过`compare`方法的返回值来确定的：

- 返回**负数**：表示第一个参数应排在第二个参数之前
- 返回**零**：表示两个参数相等
- 返回**正数**：表示第一个参数应排在第二个参数之后

排序规则：

1. **从小到大排序(升序)**：
   ```java
   a.sort((a, b) -> a - b);
   ```
   或
   ```java
   a.sort(Integer::compare);
   ```
   
2. **从大到小排序(降序)**：
   ```java
   a.sort((a, b) -> b - a);
   ```
   或
   ```java
   a.sort(Comparator.reverseOrder());
   ```

在你的代码中`a.sort((c,b)->b-c)`，返回的是b-c，所以当b大于c时返回正数，这意味着b会排在c之前，因此实现了从大到小排序。

如果两个参数顺序调换`a.sort((b,c)->b-c)`，那么结果就是从小到大排序。
