**方法引用**（Method Reference）是 Java 8 引入的一种简化 **Lambda 表达式** 的语法。它的作用是直接引用已有的方法，而不是在 Lambda 表达式中重新定义一个方法体。方法引用可以让代码更简洁、易读。

---

### 1. **方法引用的作用**
方法引用用于以下场景：
- 当 Lambda 表达式仅仅是调用一个已有的方法时，可以用方法引用来替代。
- 它是对 Lambda 表达式的进一步简化。

---

### 2. **方法引用的语法**
方法引用的语法是：
```java
类名::方法名
```
或
```java
对象::方法名
```

---

### 3. **方法引用的四种形式**
方法引用有四种形式，分别对应不同的场景：

#### 形式 1：**静态方法引用**
- **语法**：`类名::静态方法名`
- **场景**：引用类的静态方法。

**示例**：
```java
// Lambda 表达式
Function<Integer, String> lambda = num -> String.valueOf(num);

// 方法引用
Function<Integer, String> methodRef = String::valueOf;

System.out.println(methodRef.apply(123)); // 输出 "123"
```

---

#### 形式 2：**实例方法引用（通过对象）**
- **语法**：`对象::实例方法名`
- **场景**：引用某个对象的实例方法。

**示例**：
```java
String str = "Hello";

// Lambda 表达式
Supplier<Integer> lambda = () -> str.length();

// 方法引用
Supplier<Integer> methodRef = str::length;

System.out.println(methodRef.get()); // 输出 5
```

---

#### 形式 3：**实例方法引用（通过类名）**
- **语法**：`类名::实例方法名`
- **场景**：引用某个类的实例方法，第一个参数是方法的调用者。

**示例**：
```java
// Lambda 表达式
BiFunction<String, String, Boolean> lambda = (s1, s2) -> s1.equals(s2);

// 方法引用
BiFunction<String, String, Boolean> methodRef = String::equals;

System.out.println(methodRef.apply("Hello", "Hello")); // 输出 true
```

---

#### 形式 4：**构造方法引用**
- **语法**：`类名::new`
- **场景**：引用类的构造方法。

**示例**：
```java
// Lambda 表达式
Supplier<List<String>> lambda = () -> new ArrayList<>();

// 方法引用
Supplier<List<String>> methodRef = ArrayList::new;

List<String> list = methodRef.get();
list.add("Hello");
System.out.println(list); // 输出 [Hello]
```

---

### 4. **方法引用 vs Lambda 表达式**
方法引用是 Lambda 表达式的简化形式，但它们的功能是等价的。

#### Lambda 表达式：
```java
Function<String, Integer> lambda = s -> s.length();
```

#### 方法引用：
```java
Function<String, Integer> methodRef = String::length;
```

**区别**：
- **Lambda 表达式**：需要在方法体中写具体的逻辑。
- **方法引用**：直接引用已有的方法，代码更简洁。

---

### 5. **方法引用的使用场景**
方法引用通常用于以下场景：
- **Stream API** 中的操作（如 `map`、`filter` 等）。
- **函数式接口**的实现（如 `Function`、`Consumer`、`Supplier` 等）。
- **简化代码**：当 Lambda 表达式仅仅是调用一个方法时。

#### 示例：
```java
List<String> names = Arrays.asList("Alice", "Bob", "Charlie");

// 使用 Lambda 表达式
names.forEach(name -> System.out.println(name));

// 使用方法引用
names.forEach(System.out::println);
```

---

### 6. **总结**
- **方法引用**是 Lambda 表达式的简化形式，用于直接引用已有的方法。
- 它有四种形式：静态方法引用、实例方法引用（通过对象）、实例方法引用（通过类名）和构造方法引用。
- 方法引用让代码更简洁、易读，特别是在 Stream API 和函数式编程中非常有用。

通过使用方法引用，可以减少代码量，同时提高代码的可读性和维护性！
