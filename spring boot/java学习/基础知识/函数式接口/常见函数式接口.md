Java 在 `java.util.function` 包中提供了许多常用的 **函数式接口**，用于支持 Lambda 表达式和函数式编程。以下是这些接口的分类和说明：

---

### 1. **消费型接口（Consumer）**
- **特点**：接受一个参数，不返回值。
- **常用接口**：
  - `Consumer<T>`：接受一个参数，无返回值。
  - `BiConsumer<T, U>`：接受两个参数，无返回值。
- **示例**：
  ```java
  Consumer<String> print = s -> System.out.println(s);
  print.accept("Hello"); // 输出 "Hello"

  BiConsumer<String, Integer> printPair = (s, i) -> System.out.println(s + ": " + i);
  printPair.accept("Age", 25); // 输出 "Age: 25"
  ```

---

### 2. **供给型接口（Supplier）**
- **特点**：不接受参数，返回一个值。
- **常用接口**：
  - `Supplier<T>`：无参数，返回一个值。
- **示例**：
  ```java
  Supplier<Double> random = () -> Math.random();
  System.out.println(random.get()); // 输出一个随机数
  ```

---

### 3. **函数型接口（Function）**
- **特点**：接受一个参数，返回一个值。
- **常用接口**：
  - `Function<T, R>`：接受一个参数，返回一个值。
  - `BiFunction<T, U, R>`：接受两个参数，返回一个值。
  - `UnaryOperator<T>`：接受一个参数，返回同类型的值（`Function` 的特例，`T` 和 `R` 类型相同）。
  - `BinaryOperator<T>`：接受两个参数，返回同类型的值（`BiFunction` 的特例，`T`、`U`、`R` 类型相同）。
- **示例**：
  ```java
  Function<String, Integer> length = s -> s.length();
  System.out.println(length.apply("Hello")); // 输出 5

  BiFunction<String, String, String> concat = (s1, s2) -> s1 + s2;
  System.out.println(concat.apply("Hello", "World")); // 输出 "HelloWorld"

  UnaryOperator<Integer> square = x -> x * x;
  System.out.println(square.apply(5)); // 输出 25

  BinaryOperator<Integer> add = (a, b) -> a + b;
  System.out.println(add.apply(3, 5)); // 输出 8
  ```

---

### 4. **断言型接口（Predicate）**
- **特点**：接受一个参数，返回布尔值。
- **常用接口**：
  - `Predicate<T>`：接受一个参数，返回布尔值。
  - `BiPredicate<T, U>`：接受两个参数，返回布尔值。
- **示例**：
  ```java
  Predicate<Integer> isEven = x -> x % 2 == 0;
  System.out.println(isEven.test(4)); // 输出 true

  BiPredicate<String, Integer> isLengthMatch = (s, len) -> s.length() == len;
  System.out.println(isLengthMatch.test("Hello", 5)); // 输出 true
  ```

---

### 5. **其他常用接口**
| 接口名                  | 说明                                                                 |
|------------------------|--------------------------------------------------------------------|
| `Runnable`             | 无参数，无返回值。常用于线程任务。                                     |
| `Comparator<T>`        | 接受两个参数，返回整数。用于比较两个对象。                               |
| `Callable<V>`          | 无参数，返回一个值。常用于带返回值的任务。                               |
| `ToIntFunction<T>`     | 接受一个参数，返回 `int` 值。                                          |
| `ToDoubleFunction<T>`  | 接受一个参数，返回 `double` 值。                                       |
| `ToLongFunction<T>`    | 接受一个参数，返回 `long` 值。                                         |
| `IntFunction<R>`       | 接受 `int` 参数，返回一个值。                                          |
| `DoubleFunction<R>`    | 接受 `double` 参数，返回一个值。                                       |
| `LongFunction<R>`      | 接受 `long` 参数，返回一个值。                                         |

---

### 6. **原始类型特化接口**
为了减少装箱和拆箱的开销，Java 还提供了针对原始类型（`int`、`double`、`long`）的函数式接口：

| 接口名                  | 说明                                                                 |
|------------------------|--------------------------------------------------------------------|
| `IntConsumer`          | 接受 `int` 参数，无返回值。                                           |
| `DoubleConsumer`       | 接受 `double` 参数，无返回值。                                        |
| `LongConsumer`         | 接受 `long` 参数，无返回值。                                          |
| `IntSupplier`          | 无参数，返回 `int` 值。                                               |
| `DoubleSupplier`       | 无参数，返回 `double` 值。                                            |
| `LongSupplier`         | 无参数，返回 `long` 值。                                              |
| `IntFunction<R>`       | 接受 `int` 参数，返回一个值。                                          |
| `DoubleFunction<R>`    | 接受 `double` 参数，返回一个值。                                       |
| `LongFunction<R>`      | 接受 `long` 参数，返回一个值。                                         |

---

### 7. **示例代码**
```java
import java.util.function.*;

public class FunctionalInterfacesDemo {
    public static void main(String[] args) {
        // Consumer
        Consumer<String> print = s -> System.out.println(s);
        print.accept("Hello");

        // Supplier
        Supplier<Double> random = () -> Math.random();
        System.out.println(random.get());

        // Function
        Function<String, Integer> length = s -> s.length();
        System.out.println(length.apply("Hello"));

        // Predicate
        Predicate<Integer> isEven = x -> x % 2 == 0;
        System.out.println(isEven.test(4));

        // Runnable
        Runnable task = () -> System.out.println("任务完成");
        task.run();
    }
}
```

---

### 总结
Java 提供的常用函数式接口覆盖了大部分场景，包括：
- **消费型**（`Consumer`）
- **供给型**（`Supplier`）
- **函数型**（`Function`）
- **断言型**（`Predicate`）
- **其他场景**（如 `Runnable`、`Comparator` 等）

通过熟悉这些接口，可以更高效地使用 Lambda 表达式和函数式编程！
