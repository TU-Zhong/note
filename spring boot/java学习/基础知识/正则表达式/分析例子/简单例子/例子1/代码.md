

以下是一个简单的正则表达式示例：**验证由字母、数字和下划线组成的用户名，长度4-16个字符**。

---

### **正则表达式**
```java
String usernameRegex = "^[a-zA-Z0-9_]{4,16}$";
```
[[正则表达式分析]]
---

### **代码示例**
```java
import java.util.regex.Pattern;
import java.util.regex.Matcher;

public class SimpleRegexExample {
    public static void main(String[] args) {
        String usernameRegex = "^[a-zA-Z0-9_]{4,16}$";
        String[] testUsernames = {"user123", "john_doe", "a", "invalid@name", "too_long_username_123"};

        for (String username : testUsernames) {
            boolean isValid = Pattern.matches(usernameRegex, username);
            System.out.println(username + " : " + isValid);
        }
    }
}
```

---

### **运行结果**
```
user123               : true
john_doe              : true
a                     : false       （长度不足4）
invalid@name          : false       （包含非法字符@）
too_long_username_123 : false       （长度超过16）
```

---

### **正则解释**
- `^`：匹配字符串开头。
- `[a-zA-Z0-9_]`：允许字母、数字和下划线。
- `{4,16}`：长度至少4个字符，最多16个字符。
- `$`：匹配字符串结尾。

---

这个例子足够简单，但覆盖了正则的核心功能：**字符范围限定**和**长度验证**。
