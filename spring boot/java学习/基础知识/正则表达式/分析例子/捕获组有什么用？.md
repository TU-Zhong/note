

捕获组（Capturing Group）是正则表达式中非常重要的功能，它允许你从匹配的文本中提取特定部分，或者在替换、反向引用中使用这些部分。以下是捕获组的核心用途和具体示例：

---

### **一、捕获组的作用**
1. **提取子字符串**  
   通过分组匹配，可以将正则表达式中的某一部分单独提取出来（例如提取日期中的年、月、日）。
   
2. **反向引用（Backreference）**  
   在正则表达式内部，可以通过 `\1`、`\2` 引用前面捕获组的内容（Java 中需转义为 `\\1`）。

3. **替换操作**  
   在字符串替换时，可以通过 `$1`、`$2` 引用捕获组内容，动态生成新字符串。

4. **逻辑分组**  
   将多个字符或子表达式视为一个整体，方便应用量词（如 `*`、`+` 等）。与普通分组的区别是捕获组会记录匹配内容。

---

### **二、语法与示例**
#### **1. 基本语法**  
- 普通捕获组：用 `()` 定义，例如 `(\d+)`。
- 反向引用：在正则中使用 `\1` 引用第一个捕获组，`\2` 引用第二个，依此类推。
- 替换操作：在替换字符串中用 `$1`、`$2` 指向捕获组。

---

#### **2. 示例分析**
##### **示例 1：提取日期中的年、月、日**
```java
String date = "2023-10-05";
Pattern pattern = Pattern.compile("(\\d{4})-(\\d{2})-(\\d{2})");
Matcher matcher = pattern.matcher(date);
if (matcher.find()) {
    String year = matcher.group(1);  // 2023
    String month = matcher.group(2); // 10
    String day = matcher.group(3);   // 05
}
```
- **`(\d{4})`**：捕获年（4位数字）。
- **`(\d{2})`**：捕获月和日（2位数字）。

---

##### **示例 2：替换字符串（调整日期格式）**
```java
String input = "2023-10-05";
String output = input.replaceAll("(\\d{4})-(\\d{2})-(\\d{2})", "$2/$3/$1");
// 结果："10/05/2023"
```
- **`$2`**：引用第二个捕获组（月）。
- **`$3`**：引用第三个捕获组（日）。
- **`$1`**：引用第一个捕获组（年）。

---

##### **示例 3：反向引用（匹配重复单词）**
```java
String regex = "(\\b\\w+\\b)\\s+\\1"; // 匹配重复的单词，如 "the the"
Pattern pattern = Pattern.compile(regex);
Matcher matcher = pattern.matcher("the the cat sat on the the mat");
while (matcher.find()) {
    System.out.println(matcher.group()); // "the the", "the the"
}
```
- **`(\b\w+\b)`**：捕获一个完整单词。
- **`\\1`**：要求后续内容与第一个捕获组完全相同。

---

##### **示例 4：验证对称格式（如HTML标签）**
```java
String html = "<div>content</div>";
boolean isValidTag = html.matches("<([a-zA-Z]+)>.*</\\1>"); // true
```
- **`([a-zA-Z]+)`**：捕获标签名（如 `div`）。
- **`</\\1>`**：要求闭合标签名与开始标签一致（`</div>`）。

---

### **三、捕获组 vs 非捕获组**
| 功能                | 捕获组 `()`        | 非捕获组 `(?:)`       |
|---------------------|--------------------|-----------------------|
| 是否记录匹配内容     | 是                 | 否                    |
| 是否占用内存         | 是                 | 否（更高效）          |
| 是否支持反向引用     | 是（`\1`, `$1`）  | 否                    |
| 典型场景            | 提取数据、替换操作 | 仅分组或应用量词      |

---

### **四、总结**
- **核心用途**：提取特定内容、动态替换、匹配对称结构。
- **常见场景**：日期解析、重复模式检测、模板替换、数据清洗。
- **注意性能**：如果不需要提取内容，优先使用非捕获组 `(?:)` 提升效率。

通过捕获组，你可以更灵活地操作正则匹配的结果，实现复杂文本处理需求。
