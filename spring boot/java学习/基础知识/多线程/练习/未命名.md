```java
public class RunnableTest {  
    public static void main(String[] args) {  
        Bank bank1=new Bank();  
        Bank bank2=new Bank();  
        Bank bank3 = new Bank();  
        Runnable t1=()->{  
              for(int i=0;i<5;i++){  
                  Bank.transferTo(bank1, bank2, 1000);  
                  try {  
                      Thread.sleep(3000);  
                  } catch (InterruptedException e) {  
                      throw new RuntimeException(e);  
                  }  
                  System.out.println("当前线程：" + Thread.currentThread() + "现在bank1的金额是：" + bank1.cathe + "当前bank2的金额是：" + bank2.cathe);  
              }  
        };  
        Runnable t2=()->{  
            for(int i=0;i<5;i++){  
                Bank.transferTo(bank2, bank3, 1000);  
                try {  
                    Thread.sleep(3000);  
                } catch (InterruptedException e) {  
                    throw new RuntimeException(e);  
                }  
                System.out.println("当前线程：" + Thread.currentThread() + "现在bank2的金额是：" + bank2.cathe + "当前bank3的金额是：" + bank3.cathe);  
            }  
        };  
        Thread thread1=new Thread(t1);  
        Thread thread2=new Thread(t2);  
        thread1.start();  
        thread2.start();  
    }  
}  
class Bank{  
    public int cathe=10000;  
    public static synchronized void transferTo(Bank bank1,Bank bank2,int cash){  
        bank2.cathe+=cash;  
        bank1.cathe-=cash;  
    }  
}
```
1. 多线程debug, 只有所有的断点都设置为Thread,才能进入多线程断点调试，然后所有线程启动完成后，点击
	Thread & Variables 面板的最上面的选项框进入具体的线程，并可以进行切换
	
2. 保持数据一致性，通过在transferTo方法上添加synchronized关键字，或者使用Reentrantlock显示锁等形式保持数据一致性。
3. yield()方法
