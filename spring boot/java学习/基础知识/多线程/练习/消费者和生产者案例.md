```java
import java.util.LinkedList;
import java.util.Queue;

public class ProducerConsumerExample {

    // 共享队列，最大容量为2（方便观察等待效果）
    private static final Queue<Integer> queue = new LinkedList<>();
    private static final int MAX_SIZE = 2;

    public static void main(String[] args) {
        // 创建2个生产者和3个消费者
        for (int i = 1; i <= 2; i++) {
            new Thread(new Producer(), "生产者-" + i).start();
        }
        for (int i = 1; i <= 3; i++) {
            new Thread(new Consumer(), "消费者-" + i).start();
        }
    }

    // 生产者
    static class Producer implements Runnable {
        private int product = 0;

        @Override
        public void run() {
            while (true) {
                synchronized (queue) {
                    // 队列满时等待
                    while (queue.size() == MAX_SIZE) {
                        try {
                            System.out.println(Thread.currentThread().getName() + ": 队列已满，等待...");
                            queue.wait();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    }
                    // 生产商品
                    queue.offer(++product);
                    System.out.println(Thread.currentThread().getName() + "生产了: " + product + "，队列剩余: " + queue.size());
                    queue.notifyAll(); // 唤醒所有等待的消费者
                }
                sleep(1000); // 模拟生产耗时
            }
        }
    }

    // 消费者
    static class Consumer implements Runnable {
        @Override
        public void run() {
            while (true) {
                synchronized (queue) {
                    // 队列空时等待
                    while (queue.isEmpty()) {
                        try {
                            System.out.println(Thread.currentThread().getName() + ": 队列空了，等待...");
                            queue.wait();
                        } catch (InterruptedException e) {
                            e.printStackTrace();
                        }
                    }
                    // 消费商品
                    int product = queue.poll();
                    System.out.println(Thread.currentThread().getName() + "消费了: " + product + "，队列剩余: " + queue.size());
                    queue.notifyAll(); // 唤醒所有等待的生产者
                }
                sleep(1500); // 模拟消费耗时
            }
        }
    }

    // 辅助方法：线程休眠
    private static void sleep(long millis) {
        try {
            Thread.sleep(millis);
        } catch (InterruptedException e) {
            e.printStackTrace();
        }
    }
}
```

**运行效果示例：**
```
生产者-1生产了: 1，队列剩余: 1
生产者-2生产了: 1，队列剩余: 2
生产者-1: 队列已满，等待...
生产者-2: 队列已满，等待...
消费者-1消费了: 1，队列剩余: 1
消费者-2消费了: 1，队列剩余: 0
消费者-3: 队列空了，等待...
生产者-1生产了: 2，队列剩余: 1
生产者-2生产了: 2，队列剩余: 2
...
```

**代码核心点：**
1. `synchronized` 保证对队列的原子操作
2. `wait()` 让线程在条件不满足时等待
3. `notifyAll()` 唤醒其他等待线程
4. 使用 `while` 检查条件（避免虚假唤醒）
5. 队列容量控制线程协作节奏

运行后会看到生产者和消费者交替工作的日志，队列会在0/1/2之间变化，完美体现多线程协作！
