

以下是 Java 中 Lambda 表达式的常用用法及解释，包含代码示例：

---

### 1. **替代匿名内部类**
```java
// 传统写法（匿名内部类）
Runnable r1 = new Runnable() {
    @Override
    public void run() {
        System.out.println("传统写法");
    }
};

// Lambda 写法
Runnable r2 = () -> System.out.println("Lambda 简化");
```
**解释**：适用于实现只有一个抽象方法的接口（函数式接口），如 `Runnable`、`Comparator` 等。

---

### 2. **集合遍历 (`forEach`)**
```java
List<String> list = Arrays.asList("A", "B", "C");
// Lambda 遍历
list.forEach(item -> System.out.println(item));
// 方法引用（更简洁）
list.forEach(System.out::println);
```
**解释**：`forEach` 方法接受一个 `Consumer` 函数式接口，Lambda 可替代循环代码。

---

### 3. **集合过滤与转换 (`Stream` 操作)**
```java
List<Integer> numbers = Arrays.asList(1, 2, 3, 4, 5);
// 过滤偶数
List<Integer> even = numbers.stream()
                            .filter(n -> n % 2 == 0)
                            .collect(Collectors.toList());
// 转为大写
List<String> upper = list.stream()
                         .map(s -> s.toUpperCase())
                         .collect(Collectors.toList());
```
**解释**：
- `filter(Predicate<T>)`：用 Lambda 定义过滤条件。
- `map(Function<T, R>)`：用 Lambda 转换元素。

---

### 4. **排序 (`Comparator`)**
```java
List<String> names = Arrays.asList("Bob", "Alice", "Charlie");
// 按长度排序（Lambda）
names.sort((s1, s2) -> s1.length() - s2.length());
// 方法引用写法
names.sort(Comparator.comparingInt(String::length));
```
**解释**：Lambda 替代了 `Comparator` 的 `compare` 方法，简化排序逻辑。

---

### 5. **多线程任务**
```java
// 使用 Lambda 启动线程
new Thread(() -> {
    System.out.println("线程任务执行");
}).start();
```
**解释**：直接向 `Thread` 类传递 Lambda，替代 `Runnable` 实现。

---

### 6. **函数式接口**
Java 内置的常用函数式接口与 Lambda 结合：
```java
// Predicate<T>（条件判断）
Predicate<Integer> isPositive = n -> n > 0;
System.out.println(isPositive.test(5)); // true

// Function<T, R>（转换）
Function<String, Integer> toLength = s -> s.length();
System.out.println(toLength.apply("Hello")); // 5

// Consumer<T>（消费）
Consumer<String> print = s -> System.out.println(s);
print.accept("Hello Lambda");

// Supplier<T>（生产）
Supplier<Double> random = () -> Math.random();
System.out.println(random.get());
```

---

### 7. **事件处理（如 GUI）**
```java
// Java Swing 按钮点击事件（Lambda 简化）
JButton button = new JButton("Click");
button.addActionListener(e -> System.out.println("按钮被点击"));
```

---

### 8. **条件延迟执行**
```java
public void logIf(boolean condition, Supplier<String> messageSupplier) {
    if (condition) {
        System.out.println(messageSupplier.get());
    }
}
// 调用（仅当条件为 true 时执行 Lambda）
logIf(true, () -> "耗时操作结果：" + expensiveOperation());
```
**解释**：利用 `Supplier` 延迟执行高消耗操作。

---

### Lambda 表达式的核心规则：
1. **语法**：`(参数列表) -> { 方法体 }`（单行可省略 `{}` 和 `return`）。
2. **目标类型**：必须匹配函数式接口的抽象方法签名。
3. **变量捕获**：Lambda 可访问外层 `final` 或事实 `final` 的局部变量。

---

通过 Lambda 表达式，Java 可以更简洁地实现函数式编程，大幅减少模板代码。建议结合 `java.util.function` 包中的内置函数式接口使用。
