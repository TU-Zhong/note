### *<mark style="background: #FFF3A3A6;">省流：继承全部抽象方法，默认方法，常量，不会继承静态方法（只属于接口）和私有方法（只能在类内部使用）</mark>*
---

### 1. **继承抽象方法**
子接口会继承父接口中的所有抽象方法。实现子接口的类必须实现这些抽象方法（除非类是抽象类）。

```java
interface Parent {
    void methodA();
}

interface Child extends Parent {
    void methodB();
}

class MyClass implements Child {
    @Override
    public void methodA() {
        System.out.println("Implemented methodA.");
    }

    @Override
    public void methodB() {
        System.out.println("Implemented methodB.");
    }
}
```

在上面的例子中，`Child` 接口继承了 `Parent` 接口的 `methodA()`，因此实现 `Child` 接口的类必须同时实现 `methodA()` 和 `methodB()`。

---

### 2. **继承默认方法**
子接口会继承父接口中的所有默认方法。实现类可以直接使用这些默认方法，也可以选择重写它们。

```java
interface Parent {
    default void defaultMethod() {
        System.out.println("Default method in Parent.");
    }
}

interface Child extends Parent {
    void methodB();
}

class MyClass implements Child {
    @Override
    public void methodB() {
        System.out.println("Implemented methodB.");
    }
}

public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.defaultMethod(); // 输出: Default method in Parent.
    }
}
```

在上面的例子中，`Child` 接口继承了 `Parent` 接口的 `defaultMethod()`，因此 `MyClass` 可以直接使用 `defaultMethod()`。

---

### 3. **继承静态方法**
静态方法不会被继承。静态方法属于接口本身，只能通过接口名调用。

```java
interface Parent {
    static void staticMethod() {
        System.out.println("Static method in Parent.");
    }
}

interface Child extends Parent {
    // 无法继承 staticMethod()
}

public class Main {
    public static void main(String[] args) {
        Parent.staticMethod(); // 输出: Static method in Parent.
        // Child.staticMethod(); // 错误：staticMethod() 未定义
    }
}
```

在上面的例子中，`Child` 接口无法继承 `Parent` 接口的 `staticMethod()`，只能通过 `Parent` 接口名调用。

---

### 4. **继承常量**
子接口会继承父接口中的所有常量（`public static final` 变量）。这些常量可以直接通过子接口或实现类访问。

```java
interface Parent {
    int CONSTANT_VALUE = 10;
}

interface Child extends Parent {
    void methodB();
}

class MyClass implements Child {
    @Override
    public void methodB() {
        System.out.println("Implemented methodB.");
    }
}

public class Main {
    public static void main(String[] args) {
        System.out.println(Child.CONSTANT_VALUE); // 输出: 10
        System.out.println(MyClass.CONSTANT_VALUE); // 输出: 10
    }
}
```

在上面的例子中，`Child` 接口继承了 `Parent` 接口的 `CONSTANT_VALUE`，因此可以通过 `Child` 接口或 `MyClass` 类访问该常量。

---

### 5. **继承私有方法**
私有方法不会被子接口继承。私有方法只能在定义它的接口内部使用。

```java
interface Parent {
    private void privateMethod() {
        System.out.println("Private method in Parent.");
    }

    default void defaultMethod() {
        privateMethod();
    }
}

interface Child extends Parent {
    // 无法继承 privateMethod()
}

class MyClass implements Child {
    // 可以直接使用 defaultMethod()
}

public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.defaultMethod(); // 输出: Private method in Parent.
    }
}
```

在上面的例子中，`Child` 接口无法继承 `Parent` 接口的 `privateMethod()`，但可以通过 `defaultMethod()` 间接使用它。

---

### 6. **多接口继承**
一个接口可以继承多个接口（多继承），此时子接口会继承所有父接口的成员。

```java
interface InterfaceA {
    void methodA();
}

interface InterfaceB {
    void methodB();
}

interface InterfaceC extends InterfaceA, InterfaceB {
    void methodC();
}

class MyClass implements InterfaceC {
    @Override
    public void methodA() {
        System.out.println("Implemented methodA.");
    }

    @Override
    public void methodB() {
        System.out.println("Implemented methodB.");
    }

    @Override
    public void methodC() {
        System.out.println("Implemented methodC.");
    }
}
```

在上面的例子中，`InterfaceC` 继承了 `InterfaceA` 和 `InterfaceB` 的所有方法，因此实现 `InterfaceC` 的类必须实现 `methodA()`、`methodB()` 和 `methodC()`。

---

### 总结
接口的继承会继承父接口的以下内容：
- **抽象方法**：子接口继承父接口的抽象方法，实现类必须实现这些方法。
- **默认方法**：子接口继承父接口的默认方法，实现类可以直接使用或重写。
- **常量**：子接口继承父接口的常量，可以直接通过子接口或实现类访问。
- **静态方法和私有方法不会被继承**：静态方法属于接口本身，私有方法只能在定义它的接口内部使用。

通过接口的继承，可以实现代码的复用和扩展，同时保持接口的灵活性和一致性。
