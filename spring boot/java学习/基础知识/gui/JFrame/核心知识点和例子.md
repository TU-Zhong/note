### **五、完整示例代码**
```java
import javax.swing.*;
import java.awt.*;
import java.awt.event.*;

public class JFrameExample extends JFrame {
    private JLabel label;
    private JTextField textField;

    public JFrameExample() {
        // 1. 基础设置
        setTitle("我的第一个JFrame窗口");
        setSize(400, 300);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        setLayout(new FlowLayout()); // 使用流式布局

        // 2. 添加组件
        label = new JLabel("输入内容：");
        textField = new JTextField(15);
        JButton button = new JButton("提交");

        // 3. 按钮点击事件
        button.addActionListener(e -> {
            String input = textField.getText();
            JOptionPane.showMessageDialog(this, "你输入了：" + input);
        });

        // 4. 将组件添加到窗口
        add(label);
        add(textField);
        add(button);

        // 5. 显示窗口
        setVisible(true);
    }

    public static void main(String[] args) {
        new JFrameExample();
    }
}
```

---

### **六、关键注意事项**
1. **必须调用 `setVisible(true)`**，否则窗口不可见。
2. **组件需添加到内容面板**：直接使用 `add()` 方法等效于 `getContentPane().add()`。
3. **线程安全**：Swing 组件应在 Event Dispatch Thread (EDT) 中创建和修改，可用 `SwingUtilities.invokeLater()` 启动程序。
4. **自定义关闭操作**：通过 `addWindowListener()` 监听窗口事件，定制关闭逻辑。
