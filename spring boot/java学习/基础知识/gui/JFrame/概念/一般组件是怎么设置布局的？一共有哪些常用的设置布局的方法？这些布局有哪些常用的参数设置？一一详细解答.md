### **一、设置组件布局的基本方法**
在 Java Swing 中，布局管理器通过容器的 `setLayout()` 方法设置。常用容器包括 `JFrame`、`JPanel` 等。

#### **通用步骤**
```java
// 1. 创建容器（如 JPanel 或 JFrame）
JPanel panel = new JPanel();

// 2. 设置布局管理器
panel.setLayout(new BorderLayout()); // 这里以 BorderLayout 为例

// 3. 添加组件时指定布局参数（视布局类型而定）
panel.add(component, BorderLayout.NORTH);
```

---

### **二、常用布局管理器及参数设置**

#### **1. BorderLayout（边框布局）**
- **核心规则**：将容器分为 `NORTH`, `SOUTH`, `EAST`, `WEST`, `CENTER` 五个区域。
- **常用方法**：
  ```java
  // 设置布局并指定间距（水平间距10px，垂直间距5px）
  setLayout(new BorderLayout(10, 5));
  
  // 添加组件时指定区域
  add(component, BorderLayout.NORTH); 
  ```
- **参数说明**：
  - 不设置区域默认为 `CENTER`。
  - 每个区域只能放置一个组件（可通过嵌套 `JPanel` 解决多组件问题）。
- **适用场景**：主窗口框架布局（如顶部工具栏、底部状态栏）。

---

#### **2. FlowLayout（流式布局）**
- **核心规则**：按组件添加顺序从左到右排列，自动换行。
- **常用方法**：
  ```java
  // 左对齐、水平间距10px、垂直间距5px
  setLayout(new FlowLayout(FlowLayout.LEFT, 10, 5));
  
  // 添加组件无需指定参数
  panel.add(button1);
  panel.add(button2);
  ```
- **参数说明**：
  - `alignment`: 对齐方式（`LEFT`/`CENTER`/`RIGHT`）。
  - `hgap`/`vgap`: 水平/垂直间距。
- **适用场景**：工具栏按钮、简单横向排列组件。

---

#### **3. GridLayout（网格布局）**
- **核心规则**：将容器划分为固定行列的网格，组件按顺序填充单元格。
- **常用方法**：
  ```java
  // 3行2列的网格，水平间距5px，垂直间距10px
  setLayout(new GridLayout(3, 2, 5, 10));
  
  // 添加组件顺序：第一行左到右，再第二行...
  panel.add(component1);
  panel.add(component2);
  ```
- **参数说明**：
  - `rows`/`cols`: 行数和列数（设为0表示不限，但两者不能同时为0）。
  - `hgap`/`vgap`: 单元格间距。
- **适用场景**：规整的表单布局、棋盘类界面。

---

#### **4. GridBagLayout（网格袋布局）**
- **核心规则**：通过 `GridBagConstraints` 精细控制组件的位置和大小。
- **常用参数**（`GridBagConstraints` 对象）：
  ```java
  GridBagConstraints gbc = new GridBagConstraints();
  gbc.gridx = 0;       // 起始列
  gbc.gridy = 0;       // 起始行
  gbc.gridwidth = 2;   // 占据2列
  gbc.gridheight = 1;  // 占据1行
  gbc.weightx = 1.0;   // 水平扩展权重
  gbc.weighty = 0.5;   // 垂直扩展权重
  gbc.fill = GridBagConstraints.HORIZONTAL; // 填充方式
  gbc.anchor = GridBagConstraints.NORTHWEST; // 对齐方式
  
  panel.add(component, gbc);
  ```
- **关键方法**：
  ```java
  panel.setLayout(new GridBagLayout());
  ```
- **适用场景**：复杂表单、需要跨行列或动态调整比例的布局。

---

#### **5. BoxLayout（盒式布局）**
- **核心规则**：沿单一方向（水平或垂直）排列组件。
- **常用方法**：
  ```java
  // 垂直排列
  panel.setLayout(new BoxLayout(panel, BoxLayout.Y_AXIS));
  
  // 添加组件
  panel.add(component1);
  panel.add(Box.createVerticalStrut(10)); // 添加10px垂直间距
  panel.add(component2);
  ```
- **参数说明**：
  - `BoxLayout.X_AXIS`：水平排列。
  - `Box.createHorizontalGlue()`：填充剩余空间。
- **适用场景**：垂直菜单、堆叠式组件（如设置对话框）。

---

#### **6. CardLayout（卡片布局）**
- **核心规则**：层叠多张卡片，同一时间只显示一张。
- **常用方法**：
  ```java
  CardLayout cardLayout = new CardLayout();
  panel.setLayout(cardLayout);
  
  panel.add(new JPanel(), "card1"); // 添加卡片并命名
  panel.add(new JPanel(), "card2");
  
  cardLayout.show(panel, "card2"); // 切换到指定卡片
  ```
- **适用场景**：向导界面、选项卡切换（需配合事件触发）。

### **三、综合示例：组合多种布局**
```java
import javax.swing.*;
import java.awt.*;

public class CombinedLayoutExample extends JFrame {
    public CombinedLayoutExample() {
        setTitle("组合布局示例");
        setSize(600, 400);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // 主面板使用 BorderLayout
        JPanel mainPanel = new JPanel(new BorderLayout(10, 10));
        add(mainPanel);

        // 1. 顶部（FlowLayout）
        JPanel topPanel = new JPanel(new FlowLayout(FlowLayout.LEFT));
        topPanel.add(new JButton("文件"));
        topPanel.add(new JButton("编辑"));
        mainPanel.add(topPanel, BorderLayout.NORTH);

        // 2. 中间（GridBagLayout）
        JPanel centerPanel = new JPanel(new GridBagLayout());
        GridBagConstraints gbc = new GridBagConstraints();
        gbc.insets = new Insets(5, 5, 5, 5); // 外边距

        // 添加标签和文本框
        gbc.gridx = 0; gbc.gridy = 0;
        centerPanel.add(new JLabel("用户名:"), gbc);
        gbc.gridx = 1;
        centerPanel.add(new JTextField(20), gbc);

        mainPanel.add(centerPanel, BorderLayout.CENTER);

        setVisible(true);
    }

    public static void main(String[] args) {
        new CombinedLayoutExample();
    }
}
```

### **四、布局选择与优化建议**
1. **优先嵌套布局**：使用多个 `JPanel` 嵌套不同的布局。
2. **响应式设计**：
   - 使用 `weightx`/`weighty`（GridBagLayout）或 `Box.createGlue()`（BoxLayout）分配剩余空间。
3. **默认布局特点**：
   - `JFrame` 默认布局为 `BorderLayout`。
   - `JPanel` 默认布局为 `FlowLayout`。
4. **调试技巧**：
   - 为不同面板设置背景色或边框，直观区分布局区域：
     ```java
     panel.setBackground(Color.LIGHT_GRAY);
     panel.setBorder(BorderFactory.createLineBorder(Color.RED));
     ```

通过合理选择布局管理器，可以高效构建结构清晰、适配性强的 Java GUI 应用。
