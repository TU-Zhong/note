### **完整代码示例**
```java
import javax.swing.*;
import java.awt.*;

public class GridBagLayoutExample extends JFrame {
    public GridBagLayoutExample() {
        setTitle("GridBagLayout 示例 - 登录界面");
        setSize(400, 200);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // 主面板使用 GridBagLayout
        JPanel panel = new JPanel(new GridBagLayout());
        add(panel);

        // 布局约束工具（想象成给每个组件的“说明书”）
        GridBagConstraints gbc = new GridBagConstraints();
        gbc.insets = new Insets(5, 5, 5, 5); // 组件四周的边距（上、左、下、右）

        // 1. 用户名标签（第0行，第0列）
        gbc.gridx = 0;       // 横坐标：第0列
        gbc.gridy = 0;       // 纵坐标：第0行
        gbc.anchor = GridBagConstraints.EAST; // 靠右对齐（标签紧贴右边的输入框）
        panel.add(new JLabel("用户名："), gbc);

        // 2. 用户名输入框（第0行，第1列，占2列宽度）
        JTextField userField = new JTextField(15);
        gbc.gridx = 1;       // 横坐标：第1列
        gbc.gridwidth = 2;   // 占据2列的宽度
        gbc.fill = GridBagConstraints.HORIZONTAL; // 水平填满空间
        panel.add(userField, gbc);

        // 3. 密码标签（第1行，第0列）
        gbc.gridx = 0;       // 重置横坐标到第0列
        gbc.gridy = 1;       // 纵坐标：第1行
        gbc.gridwidth = 1;   // 恢复占1列
        panel.add(new JLabel("密码："), gbc);

        // 4. 密码输入框（第1行，第1列，占2列宽度）
        JPasswordField passField = new JPasswordField(15);
        gbc.gridx = 1;
        gbc.gridwidth = 2;
        panel.add(passField, gbc);

        // 5. 登录按钮（第2行，第2列，靠右对齐）
        JButton loginButton = new JButton("登录");
        gbc.gridx = 2;       // 第2列（在输入框的右侧）
        gbc.gridy = 2;       // 第2行
        gbc.gridwidth = 1;   // 占1列
        gbc.anchor = GridBagConstraints.EAST; // 按钮靠右对齐
        panel.add(loginButton, gbc);

        // 6. 设置权重：让输入框所在列水平扩展（当窗口拉宽时，输入框变宽）
        gbc.gridx = 1;
        gbc.gridy = 0;
        gbc.weightx = 1.0;   // 列的水平扩展权重为1（最高优先级）
        panel.add(Box.createHorizontalGlue(), gbc); // 添加一个隐形占位组件

        setVisible(true);
    }

    public static void main(String[] args) {
        new GridBagLayoutExample();
    }
}
```

### **运行效果**
```
[用户名：][输入框            ][ ] <-- 输入框占2列
[密码：  ][输入框            ][ ]
[               ][登录按钮] <-- 按钮在第2行最右侧
```

- **窗口拉伸测试**：当横向拉伸窗口时，输入框会跟着变宽，但标签和按钮位置不变。
- **对齐效果**：标签自动靠右对齐输入框，按钮始终保持在最右侧。

---

### **关键参数总结**
| 参数                 | 作用                                                                 |
|----------------------|----------------------------------------------------------------------|
| `gridx/gridy`        | 组件左上角的列（x）和行（y）位置（从0开始）                          |
| `gridwidth/gridheight` | 组件占据的列数和行数                                                |
| `weightx/weighty`    | 分配剩余空间的比例（0表示不扩展）                                   |
| `anchor`             | 组件在格子中的对齐方式（如 `EAST` 靠右）                            |
| `fill`               | 组件填充方式（`HORIZONTAL`水平填满，`VERTICAL`垂直填满，`BOTH`全填满） |
| `insets`             | 组件四周的边距（类似 CSS 的 padding）                               |

### **类比现实场景**
想象你在用 Excel 表格设计界面：
- `gridx=1, gridy=0` → 把组件放在 B1 单元格。
- `gridwidth=2` → 合并 B1 和 C1 单元格。
- `weightx=1` → 当表格拉宽时，这一列比其他列优先变宽。
