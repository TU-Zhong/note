

以下是一个 `FlowLayout` 布局的完整示例代码，展示其默认行为和不同对齐方式的用法：

---

### **完整代码示例**
```java
import javax.swing.*;
import java.awt.*;

public class FlowLayoutExample extends JFrame {
    public FlowLayoutExample() {
        setTitle("FlowLayout 示例");
        setSize(400, 200);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // 1. 默认 FlowLayout（居中对齐）
        JPanel defaultPanel = new JPanel();
        defaultPanel.setBorder(BorderFactory.createTitledBorder("默认居中对齐"));
        defaultPanel.add(new JButton("按钮1"));
        defaultPanel.add(new JButton("按钮2"));
        defaultPanel.add(new JButton("按钮3"));

        // 2. 左对齐，设置间距（水平间距10px，垂直间距5px）
        JPanel leftPanel = new JPanel(new FlowLayout(FlowLayout.LEFT, 10, 5));
        leftPanel.setBorder(BorderFactory.createTitledBorder("左对齐"));
        leftPanel.add(new JButton("按钮A"));
        leftPanel.add(new JButton("按钮B"));
        leftPanel.add(new JButton("按钮C"));

        // 3. 右对齐
        JPanel rightPanel = new JPanel(new FlowLayout(FlowLayout.RIGHT));
        rightPanel.setBorder(BorderFactory.createTitledBorder("右对齐"));
        rightPanel.add(new JButton("按钮X"));
        rightPanel.add(new JButton("按钮Y"));

        // 主窗口使用垂直排列（BoxLayout）
        setLayout(new BoxLayout(getContentPane(), BoxLayout.Y_AXIS));
        add(defaultPanel);
        add(leftPanel);
        add(rightPanel);

        setVisible(true);
    }

    public static void main(String[] args) {
        new FlowLayoutExample();
    }
}
```

---

### **关键代码说明**
1. **默认行为**：
   ```java
   JPanel defaultPanel = new JPanel(); // 等价于 new FlowLayout(FlowLayout.CENTER)
   ```
   - 组件水平居中排列
   - 自动换行

2. **左对齐 + 间距**：
   ```java
   new FlowLayout(FlowLayout.LEFT, 10, 5) // 水平间距10px，垂直间距5px
   ```

3. **右对齐**：
   ```java
   new FlowLayout(FlowLayout.RIGHT)
   ```

---

### **运行效果**
1. **默认居中对齐**：
   ```
   [    按钮1    ][    按钮2    ][    按钮3    ]
   ```
2. **左对齐**：
   ```
   [按钮A][按钮B][按钮C]（左侧紧密排列）
   ```
3. **右对齐**：
   ```
                        [按钮X][按钮Y]
   ```

---

### **窗口调整测试**
- **收缩窗口**：当窗口宽度不足时，组件自动换行：
  ```
  [按钮1][按钮2]
  [    按钮3    ]  // 居中对齐换行
  ```
- **拉伸窗口**：组件重新排列为单行。

---

### **FlowLayout 特点总结**
| 特性              | 说明                                                                 |
|-------------------|----------------------------------------------------------------------|
| **默认对齐方式**  | `FlowLayout.CENTER`（水平居中）                                      |
| **自动换行**      | 当容器宽度不足时，自动换到下一行                                     |
| **间距控制**      | 可通过构造方法指定水平和垂直间距                                     |
| **适用场景**      | 工具栏、快速原型设计、简单横向排列                                   |
| **与其他布局对比**| 不同于 `GridLayout` 的严格网格，更灵活，但无法精确控制组件大小和位置 |

---

通过此示例，可以直观理解 `FlowLayout` 的组件排列逻辑。它是 Java Swing 中最简单的布局管理器之一，适合需要快速构建基础界面的场景。
