

以下是一个使用 `BorderLayout` 的详细示例，通过代码展示其五个区域（North/South/East/West/Center）的典型用法：

---

### **完整代码示例**
```java
import javax.swing.*;
import java.awt.*;

public class BorderLayoutExample extends JFrame {
    public BorderLayoutExample() {
        // 基础窗口设置
        setTitle("BorderLayout 示例");
        setSize(600, 400);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        
        // 设置布局为 BorderLayout（可指定间距）
        setLayout(new BorderLayout(10, 5)); // 水平间距10px，垂直间距5px

        // 1. North 区域（顶部）
        JPanel northPanel = new JPanel();
        northPanel.setBackground(Color.LIGHT_GRAY);
        northPanel.add(new JLabel("顶部工具栏"));
        northPanel.add(new JButton("文件"));
        northPanel.add(new JButton("编辑"));
        add(northPanel, BorderLayout.NORTH); // 添加至北部

        // 2. South 区域（底部）
        JLabel southLabel = new JLabel("状态栏：就绪");
        southLabel.setBorder(BorderFactory.createLineBorder(Color.DARK_GRAY));
        add(southLabel, BorderLayout.SOUTH);

        // 3. West 区域（左侧）
        JPanel westPanel = new JPanel();
        westPanel.setLayout(new GridLayout(3, 1)); // 3行1列的网格布局
        westPanel.add(new JButton("导航1"));
        westPanel.add(new JButton("导航2"));
        westPanel.add(new JButton("导航3"));
        add(westPanel, BorderLayout.WEST);

        // 4. East 区域（右侧）
        JTextArea eastText = new JTextArea("右侧日志区域");
        eastText.setBackground(new Color(240, 240, 240));
        add(new JScrollPane(eastText), BorderLayout.EAST); // 带滚动条

        // 5. Center 区域（中间）
        JTextArea centerText = new JTextArea("中央主内容区域");
        add(new JScrollPane(centerText), BorderLayout.CENTER);

        setVisible(true);
    }

    public static void main(String[] args) {
        new BorderLayoutExample();
    }
}
```

---

### **关键点说明**
1. **区域特性**：
   - **NORTH/SOUTH**：高度固定，宽度填满窗口。
   - **EAST/WEST**：宽度固定，高度填满剩余空间。
   - **CENTER**：自动扩展填充所有剩余空间。

2. **组件叠加规则**：
   - 每个区域只能放置一个组件（可通过嵌套 `JPanel` 解决）。
   - 如果某个区域未使用，其他区域会占据其空间（例如未设 `NORTH` 时，`CENTER` 向上扩展）。

3. **窗口拉伸效果**：
   - 调整窗口大小时，`CENTER` 区域内容会动态缩放。
   - 其他区域保持初始尺寸，仅调整位置。

---

### **运行效果**
1. **界面结构**：
   - 顶部灰色工具栏（`NORTH`）
   - 左侧垂直按钮导航（`WEST`）
   - 右侧灰色背景的日志区域（`EAST`）
   - 底部状态栏（`SOUTH`）
   - 中间占据最大区域的可滚动文本框（`CENTER`）

2. **行为验证**：
   - 删除 `add(..., BorderLayout.EAST)` 代码后，`CENTER` 区域会向右扩展填补空白。
   - 修改水平/垂直间距参数（如 `new BorderLayout(20, 10)`），观察组件间隔变化。

---

通过此示例，可以直观理解 `BorderLayout` 的分区管理逻辑，适合构建经典的“上-下-左-右-中”结构化界面。
