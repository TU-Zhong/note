

以下是 Java Swing 中常用布局管理器的详细知识点和一个综合应用实例：

---

### **一、核心布局管理器详解**

#### **1. BorderLayout（边框布局）**
[[spring boot/java学习/基础知识/gui/JFrame/布局/borderLayout/示例]]
<mark style="background: #FFF3A3A6;">构造方法，设置水平间距和垂直间距</mark>
- **区域划分**：`NORTH`, `SOUTH`, `EAST`, `WEST`, `CENTER`
- **特点**：  
  - 每个区域只能放置一个组件
  - `CENTER` 区域自动扩展填充剩余空间
- **常用方法**：
  ```java
  add(component, BorderLayout.NORTH); // 指定区域添加组件
  setHgap(int)/setVgap(int);          // 设置水平/垂直间距
  ```

#### **2. FlowLayout（流式布局）**
[[spring boot/java学习/基础知识/gui/JFrame/布局/FlowLayOut/示例|示例]]
- **排列方式**：按添加顺序从左到右排列，自动换行
- **对齐方式**：默认居中（可设置 `LEFT`/`RIGHT`）
- **常用方法**：
  ```java
  new FlowLayout(FlowLayout.LEFT, 10, 5); // 左对齐，水平间距10，垂直间距5
  ```

#### **3. GridLayout（网格布局）**
- **严格网格**：均匀划分行列，组件按顺序填充单元格
- **特点**：所有单元格大小相同
- **构造方法**：
  ```java
  new GridLayout(3, 2); // 3行2列
  new GridLayout(0, 4); // 不限行数，4列
  ```

#### **4. GridBagLayout（网格袋布局）**
[[傻子解释]]
- **最灵活但复杂**：通过 `GridBagConstraints` 精细控制组件位置和大小
- **核心参数**：
  ```java
  gridx/gridy      // 组件起始行列位置
  gridwidth/gridheight // 组件占据的列数/行数
  weightx/weighty  // 分配剩余空间的比例
  fill             // 填充方式（NONE, BOTH, HORIZONTAL, VERTICAL）
  anchor           // 组件对齐方式（CENTER, NORTH等）
  ```

#### **5. CardLayout（卡片布局）**
- **层叠卡片**：同一容器内可切换不同卡片（如向导界面）
- **常用方法**：
  ```java
  show(container, "cardName"); // 显示指定卡片
  next(container);             // 切换到下一张
  ```

### **二、综合示例：复杂布局嵌套**
```java
import javax.swing.*;
import java.awt.*;

public class ComplexLayoutExample extends JFrame {
    public ComplexLayoutExample() {
        setTitle("复杂布局示例");
        setSize(600, 400);
        setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

        // 主面板使用 BorderLayout
        JPanel mainPanel = new JPanel(new BorderLayout());
        add(mainPanel);

        // 1. 顶部工具栏（FlowLayout）
        JPanel topPanel = new JPanel(new FlowLayout(FlowLayout.LEFT));
        topPanel.add(new JButton("新建"));
        topPanel.add(new JButton("打开"));
        mainPanel.add(topPanel, BorderLayout.NORTH);

        // 2. 左侧导航栏（GridLayout）
        JPanel leftPanel = new JPanel(new GridLayout(4, 1, 5, 5));
        leftPanel.add(new JButton("主页"));
        leftPanel.add(new JButton("设置"));
        leftPanel.add(new JButton("帮助"));
        leftPanel.add(new JButton("退出"));
        mainPanel.add(leftPanel, BorderLayout.WEST);

        // 3. 中心区域（GridBagLayout）
        JPanel centerPanel = new JPanel(new GridBagLayout());
        GridBagConstraints gbc = new GridBagConstraints();
        gbc.insets = new Insets(5, 5, 5, 5); // 外边距

        // 第一行
        gbc.gridx = 0; gbc.gridy = 0;
        centerPanel.add(new JLabel("用户名:"), gbc);
        gbc.gridx = 1;
        centerPanel.add(new JTextField(15), gbc);

        // 第二行
        gbc.gridx = 0; gbc.gridy = 1;
        centerPanel.add(new JLabel("密码:"), gbc);
        gbc.gridx = 1;
        centerPanel.add(new JPasswordField(15), gbc);

        // 第三行（跨两列）
        gbc.gridx = 0; gbc.gridy = 2;
        gbc.gridwidth = 2;
        gbc.anchor = GridBagConstraints.CENTER;
        centerPanel.add(new JButton("登录"), gbc);

        mainPanel.add(centerPanel, BorderLayout.CENTER);

        // 4. 底部状态栏（BoxLayout）
        JPanel bottomPanel = new JPanel();
        bottomPanel.setLayout(new BoxLayout(bottomPanel, BoxLayout.X_AXIS));
        bottomPanel.add(new JLabel("状态: "));
        bottomPanel.add(Box.createHorizontalGlue()); // 弹性空间
        bottomPanel.add(new JLabel("就绪"));

        mainPanel.add(bottomPanel, BorderLayout.SOUTH);

        setVisible(true);
    }

    public static void main(String[] args) {
        new ComplexLayoutExample();
    }
}
```

### **三、布局选择技巧**
1. **优先使用嵌套**：通过 `JPanel` 分区域使用不同布局
2. **响应式设计**：
   - `BorderLayout.CENTER` + `GridBagLayout` 适应窗口大小变化
   - 使用 `weightx/weighty` 分配扩展比例
3. **快速原型**：
   - `GridLayout` 适合规整排列
   - `FlowLayout` 适合工具栏
4. **特殊需求**：
   - 绝对定位 `setLayout(null)` + `setBounds()`（慎用，破坏跨平台适配）

---

### **四、运行效果说明**
- **顶部工具栏**：按钮左对齐排列（FlowLayout）
- **左侧导航**：四个按钮垂直均分（GridLayout）
- **中心区域**：
  - 使用 `GridBagLayout` 实现标签-输入框对齐
  - 登录按钮跨两列居中
- **底部状态栏**：
  - 左侧状态标签与右侧信息自动分离（BoxLayout + Glue）

通过组合不同布局管理器，可实现复杂但结构清晰的 GUI 界面。
