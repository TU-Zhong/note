这里有idea的插件JFormDesigner可以直接用图形拖拽的方式来编写gui界面，然后你懂的。
这个插件生成的代码默认不含主题，所以在java代码那里要显示引入主题
```java
FlatAtomOneLightIJTheme.installLafInfo();  
try {  
    UIManager.setLookAndFeel( new FlatAtomOneLightIJTheme());  
} catch( Exception ex ) {  
    System.err.println( "Failed to initialize LaF" );  
}
```

FlatAtomOneLightIJTheme这个是可以去换的，然后这个和插件中的主题是一一对应的。
这个包要下载，有两个，在maven中心仓库可以找到
[Maven Repository: com.formdev » flatlaf-intellij-themes](https://mvnrepository.com/artifact/com.formdev/flatlaf-intellij-themes)

[Maven Repository: com.formdev » flatlaf](https://mvnrepository.com/artifact/com.formdev/flatlaf)


这个很重要，自己好好把握。
