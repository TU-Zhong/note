

在 Java 中实现数学表达式解析和计算（支持 `sin`, `cos` 等函数）需要 **词法分析、语法解析、优先级处理** 等步骤。以下是逐步解决方案和代码示例：

---

### **步骤 1：设计表达式规则**
明确支持的运算符、优先级和函数：
- **运算符优先级**（从高到低）：
  ```text
  1. 函数（sin, cos...）
  2. 括号 ()
  3. 负号 -
  4. 乘除 * /
  5. 加减 + -
  ```
- **函数支持**：`sin(x)`，`cos(x)` 等，需匹配函数名和参数。

---

### **步骤 2：拆分表达式为标记（Tokenize）**
将输入字符串拆分为 **数字、运算符、函数、括号** 等标记。  
使用正则表达式或手动遍历实现。

#### **正则表达式示例**：
```java
import java.util.ArrayList;
import java.util.List;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

public class Tokenizer {
    public static List<String> tokenize(String expression) {
        List<String> tokens = new ArrayList<>();
        String regex = "(\\d+\\.?\\d*)|([+\\-*/^()])|(sin|cos|tan)|([,])";
        Pattern pattern = Pattern.compile(regex, Pattern.CASE_INSENSITIVE);
        Matcher matcher = pattern.matcher(expression.replaceAll("\\s+", ""));

        while (matcher.find()) {
            String token = matcher.group(0);
            tokens.add(token);
        }
        return tokens;
    }
}
```

---

### **步骤 3：将中缀表达式转换为后缀表达式（逆波兰表示法）**
使用 **Shunting-Yard 算法** 处理优先级和括号，生成后缀表达式。  
**代码示例**：
```java
import java.util.Stack;
import java.util.List;
import java.util.ArrayList;

public class ShuntingYard {
    public static List<String> infixToPostfix(List<String> tokens) {
        List<String> output = new ArrayList<>();
        Stack<String> stack = new Stack<>();
        for (String token : tokens) {
            if (isNumber(token)) {
                output.add(token);
            } else if (isFunction(token)) {
                stack.push(token);
            } else if (",".equals(token)) {
                while (!stack.isEmpty() && !"(".equals(stack.peek())) {
                    output.add(stack.pop());
                }
            } else if ("(".equals(token)) {
                stack.push(token);
            } else if (")".equals(token)) {
                while (!stack.isEmpty() && !"(".equals(stack.peek())) {
                    output.add(stack.pop());
                }
                stack.pop(); // 弹出左括号
                if (!stack.isEmpty() && isFunction(stack.peek())) {
                    output.add(stack.pop()); // 添加函数
                }
            } else {
                // 处理运算符优先级
                while (!stack.isEmpty() && getPrecedence(token) <= getPrecedence(stack.peek())) {
                    output.add(stack.pop());
                }
                stack.push(token);
            }
        }
        while (!stack.isEmpty()) {
            output.add(stack.pop());
        }
        return output;
    }

    private static boolean isNumber(String token) {
        return token.matches("\\d+\\.?\\d*");
    }

    private static boolean isFunction(String token) {
        return token.matches("sin|cos|tan");
    }

    private static int getPrecedence(String op) {
        switch (op) {
            case "+":
            case "-":
                return 1;
            case "*":
            case "/":
                return 2;
            case "^":
                return 3;
            case "sin":
            case "cos":
                return 4; // 函数优先级最高
            default:
                return 0;
        }
    }
}
```

### **步骤 4：计算后缀表达式**
遍历后缀表达式，用栈计算结果（支持函数）：
```java
import java.util.List;
import java.util.Stack;

public class PostfixEvaluator {
    public static double evaluate(List<String> postfix) {
        Stack<Double> stack = new Stack<>();
        for (String token : postfix) {
            if (token.matches("\\d+\\.?\\d*")) {
                stack.push(Double.parseDouble(token));
            } else if (isFunction(token)) {
                double arg = stack.pop();
                stack.push(applyFunction(token, arg));
            } else {
                double b = stack.pop();
                double a = stack.pop();
                stack.push(applyOperator(token, a, b));
            }
        }
        return stack.pop();
    }

    private static double applyOperator(String op, double a, double b) {
        switch (op) {
            case "+": return a + b;
            case "-": return a - b;
            case "*": return a * b;
            case "/": return a / b;
            case "^": return Math.pow(a, b);
            default: throw new IllegalArgumentException("未知运算符: " + op);
        }
    }

    private static double applyFunction(String func, double arg) {
        switch (func.toLowerCase()) {
            case "sin": return Math.sin(Math.toRadians(arg));
            case "cos": return Math.cos(Math.toRadians(arg));
            case "tan": return Math.tan(Math.toRadians(arg));
            default: throw new IllegalArgumentException("未知函数: " + func);
        }
    }

    private static boolean isFunction(String token) {
        return token.matches("sin|cos|tan");
    }
}
```

---

### **步骤 5：整合所有步骤**
将 `Tokenizer`、`ShuntingYard` 和 `PostfixEvaluator` 整合到计算逻辑中：
```java
public class Calculator {
    public static double calculate(String expression) {
        List<String> tokens = Tokenizer.tokenize(expression);
        List<String> postfix = ShuntingYard.infixToPostfix(tokens);
        return PostfixEvaluator.evaluate(postfix);
    }

    public static void main(String[] args) {
        String input = "3 + 4 * sin(30)";
        double result = calculate(input);
        System.out.println(input + " = " + result); // 输出: 3 + 4 * sin(30) = 5.0
    }
}
```

---

### **补充说明**
1. **错误处理**：  
   添加对括号不匹配、非法字符的校验（例如 `tokenize` 阶段过滤无效字符）。
2. **更多函数支持**：  
   扩展 `applyFunction` 方法，添加 `log`, `sqrt` 等。
3. **变量支持**：  
   可增加变量替换功能（如 `x` 替换为具体值）。
