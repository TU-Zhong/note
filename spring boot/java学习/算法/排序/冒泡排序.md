好的，这是冒泡排序的 Java 代码示例：

Java

```java
import java.util.Arrays;

public class BubbleSort {

    public static void bubbleSort(int[] arr) {
        if (arr == null || arr.length == 0) {
            return; // 处理空数组或null数组的情况
        }

        int n = arr.length;
        boolean swapped; // 标记本轮是否有交换，用于优化

        // 外层循环控制排序的轮数
        // 每一轮会将当前未排序部分的最大元素“冒泡”到末尾
        for (int i = 0; i < n - 1; i++) {
            swapped = false; // 每轮开始前，假设没有发生交换

            // 内层循环进行相邻元素的比较和交换
            // arr.length - 1 - i 是因为每轮过后，末尾的 i 个元素已经是有序的了
            for (int j = 0; j < n - 1 - i; j++) {
                // 如果前一个元素大于后一个元素（升序排序）
                if (arr[j] > arr[j + 1]) {
                    // 交换 arr[j] 和 arr[j+1]
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                    swapped = true; // 标记发生了交换
                }
            }

            // 优化：如果在一轮内部比较中没有发生任何交换，
            // 说明数组已经完全排序，可以提前结束
            if (!swapped) {
                break;
            }
        }
    }

    public static void main(String[] args) {
        int[] data = {64, 34, 25, 12, 22, 11, 90};
        System.out.println("排序前的数组: " + Arrays.toString(data));

        bubbleSort(data);

        System.out.println("排序后的数组: " + Arrays.toString(data));

        int[] emptyArray = {};
        bubbleSort(emptyArray);
        System.out.println("排序后的空数组: " + Arrays.toString(emptyArray));

        int[] sortedArray = {1, 2, 3, 4, 5};
        bubbleSort(sortedArray);
        System.out.println("排序后的已排序数组: " + Arrays.toString(sortedArray));
    }
}
```

**代码解释：**

1. **`bubbleSort(int[] arr)` 方法：**
    
    - 接收一个整型数组 `arr` 作为参数。
    - 首先检查数组是否为 `null` 或长度为0，如果是则直接返回。
    - `n = arr.length;`：获取数组的长度。
    - `boolean swapped;`：声明一个布尔变量 `swapped`，用于标记在一轮比较中是否发生了元素交换。这是冒泡排序的一个常见优化，如果一轮下来没有任何元素交换，说明数组已经有序，可以提前终止排序。
2. **外层循环 `for (int i = 0; i < n - 1; i++)`：**
    
    - 这个循环控制排序的总轮数。对于一个长度为 `n` 的数组，最多需要 `n-1` 轮排序。
    - `i` 也代表了当前已经有多少个元素被放到了它们最终的有序位置（即数组的末尾部分）。
    - `swapped = false;`：在每一轮外层循环开始时，重置 `swapped` 为 `false`。
3. **内层循环 `for (int j = 0; j < n - 1 - i; j++)`：**
    
    - 这个循环负责在当前未排序的元素中进行相邻元素的比较和交换。
    - `n - 1 - i` 是内层循环的上界。减 `1` 是因为我们比较的是 `arr[j]` 和 `arr[j+1]`，所以 `j` 最多到 `n-2`。减 `i` 是因为每经过一轮外层循环，数组末尾的 `i` 个元素已经是排好序的了，无需再参与比较。
    - `if (arr[j] > arr[j + 1])`：比较相邻的两个元素。如果前面的元素大于后面的元素（这里是升序排序的逻辑），则执行交换。
    - **交换操作：**
        
        Java
        
        ```java
        int temp = arr[j];
        arr[j] = arr[j + 1];
        arr[j + 1] = temp;
        ```
        
        使用一个临时变量 `temp` 来完成两个元素的交换。
    - `swapped = true;`：如果发生了交换，将 `swapped` 标记为 `true`。
4. **优化判断 `if (!swapped) { break; }`：**
    
    - 在内层循环结束后，如果 `swapped` 仍然是 `false`，意味着在这一整轮的比较中没有发生任何元素的交换。这表明数组已经完全有序，因此可以提前 `break` 外层循环，结束排序过程。
5. **`main` 方法：**
    
    - 这是一个示例用法，创建了一个待排序的数组 `data`。
    - 调用 `bubbleSort(data)` 方法对数组进行排序。
    - 使用 `Arrays.toString(data)` 打印排序前后的数组内容，方便查看结果。
    - 还包含了对空数组和已排序数组的测试。

**冒泡排序的特点：**

- **简单易懂：** 逻辑非常直观。
- **稳定性：** 冒泡排序是一种稳定的排序算法，即相等元素的相对顺序在排序后不会改变。
- **时间复杂度：**
    - 最坏情况和平均情况：O(n2)，其中 n 是数组的长度。这是因为需要嵌套循环。
    - 最好情况（数组已经有序）：O(n)，如果加入了 `swapped` 标志进行优化，只需要遍历一遍即可确认。
- **空间复杂度：** O(1)，因为它是原地排序，只需要一个额外的临时变量用于交换。

由于其时间复杂度较高，冒泡排序通常不适用于对大规模数据进行排序，但在教学或处理小规模数据时仍有其价值。
