Java 泛型（Generics）是 Java 5 引入的一个重要特性，它允许在定义类、接口和方法时使用类型参数，从而提高代码的复用性和类型安全性。以下是 Java 泛型的主要知识点及对应示例：

### 1. 泛型类（Generic Class）
泛型类是指在定义类时使用类型参数，使得类可以处理多种数据类型。

```java
// 定义一个泛型类
public class Box<T> {
    private T item;

    public void setItem(T item) {
        this.item = item;
    }

    public T getItem() {
        return item;
    }
}

// 使用泛型类
Box<String> stringBox = new Box<>();
stringBox.setItem("Hello");
System.out.println(stringBox.getItem()); // 输出: Hello

Box<Integer> integerBox = new Box<>();
integerBox.setItem(123);
System.out.println(integerBox.getItem()); // 输出: 123
```

### 2. 泛型接口（Generic Interface）
泛型接口与泛型类类似，可以在定义接口时使用类型参数。

```java
// 定义一个泛型接口
public interface Pair<K, V> {
    K getKey();
    V getValue();
}

// 实现泛型接口
public class OrderedPair<K, V> implements Pair<K, V> {
    private K key;

        this.key = key;
        this.value = value;

    public V getValue() { return value; }

// 使用泛型接口

System.out.println(pair.getKey() + " -> " + pair.getValue()); // 输出: One -> 1
```

### 3. 泛型方法（Generic Method）
泛型方法是指在定义方法时使用类型参数，使得方法可以处理多种数据类型。

```java
// 定义一个泛型方法
public <T> void printArray(T[] array) {
    for (T element : array) {
        System.out.println(element);
    }
}

// 使用泛型方法
Integer[] intArray = {1, 2, 3};
String[] strArray = {"A", "B", "C"};

printArray(intArray); // 输出: 1 2 3
printArray(strArray); // 输出: A B C
```

### 4. 类型通配符（Wildcard）
类型通配符用于表示未知类型，通常用于泛型类或方法的参数中。

```java
// 使用通配符
public static void printList(List<?> list) {
    for (Object element : list) {
        System.out.println(element);
    }
}

// 使用通配符方法
List<Integer> intList = Arrays.asList(1, 2, 3);
List<String> strList = Arrays.asList("A", "B", "C");

printList(intList); // 输出: 1 2 3
printList(strList); // 输出: A B C
```

### 5. 上限通配符（Upper Bounded Wildcard）
上限通配符用于限制通配符的上限，表示通配符必须是某个类的子类。

```java
// 使用上限通配符
public static double sumOfList(List<? extends Number> list) {
    double sum = 0.0;
    for (Number number : list) {
        sum += number.doubleValue();
    }
    return sum;
}

// 使用上限通配符方法
List<Integer> intList = Arrays.asList(1, 2, 3);
List<Double> doubleList = Arrays.asList(1.1, 2.2, 3.3);

System.out.println(sumOfList(intList)); // 输出: 6.0
System.out.println(sumOfList(doubleList)); // 输出: 6.6
```

### 6. 下限通配符（Lower Bounded Wildcard）
下限通配符用于限制通配符的下限，表示通配符必须是某个类的父类。

```java
// 使用下限通配符
public static void addNumbers(List<? super Integer> list) {
    for (int i = 1; i <= 5; i++) {
        list.add(i);
    }
}

// 使用下限通配符方法
List<Number> numberList = new ArrayList<>();
addNumbers(numberList);
System.out.println(numberList); // 输出: [1, 2, 3, 4, 5]
```

### 7. 类型擦除（Type Erasure）
Java 泛型在编译时会进行类型擦除，即泛型类型信息在运行时会被擦除，替换为它们的上限类型（通常是 `Object`）。

```java
// 类型擦除示例
public class Box<T> {
    private T item;

    public void setItem(T item) {
        this.item = item;
    }

    public T getItem() {
        return item;
    }
}

// 编译后，Box 类中的 T 会被擦除为 Object
Box<String> stringBox = new Box<>();
stringBox.setItem("Hello");
String item = stringBox.getItem(); // 运行时，item 是 Object 类型，但编译器会插入类型转换
```

### 8. 泛型与继承（Generics and Inheritance）
泛型类或接口的继承关系与普通类或接口不同，泛型类型参数不会继承。

```java
// 泛型与继承示例
class Animal {}
class Dog extends Animal {}

List<Animal> animals = new ArrayList<>();
List<Dog> dogs = new ArrayList<>();

// animals = dogs; // 编译错误，List<Dog> 不是 List<Animal> 的子类
```

### 10. 泛型与反射（Generics and Reflection）
通过反射可以获取泛型类型信息，但需要注意类型擦除的影响。

```java
// 泛型与反射示例
import java.lang.reflect.ParameterizedType;
import java.lang.reflect.Type;

public class GenericType<T> {
    public void printType() {
        Type type = getClass().getGenericSuperclass();
        if (type instanceof ParameterizedType) {
            ParameterizedType pType = (ParameterizedType) type;
            Type[] actualTypeArguments = pType.getActualTypeArguments();
            for (Type t : actualTypeArguments) {
                System.out.println(t);
            }
        }
    }
}

class StringType extends GenericType<String> {}

// 使用反射获取泛型类型
new StringType().printType(); // 输出: class java.lang.String
```

### 11. 泛型与协变、逆变（Generics and Covariance, Contravariance）
Java 泛型是不变的，但可以通过通配符实现协变和逆变。

```java
// 协变示例
List<? extends Number> numbers = new ArrayList<Integer>(); // 协变

// 逆变示例
List<? super Integer> integers = new ArrayList<Number>(); // 逆变
```


### 14. 泛型与构造函数（Generics and Constructors）
构造函数可以使用泛型类型参数，但构造函数的泛型类型参数与类的泛型类型参数不同。

```java
// 泛型与构造函数示例
public class Box<T> {
    private T item;

    public <U> Box(U u) {
        // 构造函数的泛型类型参数 U 与类的泛型类型参数 T 不同
        System.out.println(u);
    }
}

// 使用泛型构造函数
Box<String> box = new Box<>(123); // 输出: 123
```

### 15. 泛型与匿名类（Generics and Anonymous Classes）
匿名类可以使用泛型类型参数，但需要注意类型擦除的影响。

```java
// 泛型与匿名类示例
List<String> list = new ArrayList<String>() {
    {
        add("A");
        add("B");
    }
};

System.out.println(list); // 输出: [A, B]
```

### 16. 泛型与 Lambda 表达式（Generics and Lambda Expressions）
Lambda 表达式可以与泛型方法结合使用，提高代码的简洁性和可读性。

```java
// 泛型与 Lambda 表达式示例
import java.util.function.Function;

public class Main {
    public static <T, R> R applyFunction(T input, Function<T, R> function) {
        return function.apply(input);
    }

    public static void main(String[] args) {
        String result = applyFunction(123, Object::toString);
        System.out.println(result); // 输出: 123
    }
}
```

### 17. 泛型与 Stream API（Generics and Stream API）
Stream API 可以与泛型结合使用，处理各种类型的数据流。

```java
// 泛型与 Stream API 示例
import java.util.Arrays;
import java.util.List;

public class Main {
    public static <T> void printList(List<T> list) {
        list.stream().forEach(System.out::println);
    }

    public static void main(String[] args) {
        List<String> strList = Arrays.asList("A", "B", "C");
        List<Integer> intList = Arrays.asList(1, 2, 3);

        printList(strList); // 输出: A B C
        printList(intList); // 输出: 1 2 3
    }
}
```

### 18. 泛型与集合框架（Generics and Collections Framework）
Java 集合框架广泛使用泛型，提供了类型安全的集合操作。

```java
// 泛型与集合框架示例
import java.util.ArrayList;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<String> list = new ArrayList<>();
        list.add("A");
        list.add("B");
        list.add("C");

        for (String s : list) {
            System.out.println(s); // 输出: A B C
        }
    }
}
```

### 19. 泛型与自定义类型（Generics and Custom Types）
泛型可以用于自定义类型，使得自定义类型可以处理多种数据类型。

```java
// 泛型与自定义类型示例
public class Pair<T, U> {
    private T first;
    private U second;

    public Pair(T first, U second) {
        this.first = first;
        this.second = second;
    }

    public T getFirst() { return first; }
    public U getSecond() { return second; }
}

// 使用自定义泛型类型
Pair<String, Integer> pair = new Pair<>("One", 1);
System.out.println(pair.getFirst() + " -> " + pair.getSecond()); // 输出: One -> 1
```

### 20. 泛型与递归类型（Generics and Recursive Types）
泛型可以用于定义递归类型，即类型参数可以引用自身。

```java
// 泛型与递归类型示例
public class TreeNode<T> {
    private T value;
    private List<TreeNode<T>> children = new ArrayList<>();

    public TreeNode(T value) {
        this.value = value;
    }

    public void addChild(TreeNode<T> child) {
        children.add(child);
    }

    public List<TreeNode<T>> getChildren() {
        return children;
    }

    public T getValue() {
        return value;
    }
}

// 使用递归泛型类型
TreeNode<String> root = new TreeNode<>("Root");
TreeNode<String> child1 = new TreeNode<>("Child1");
TreeNode<String> child2 = new TreeNode<>("Child2");

root.addChild(child1);
root.addChild(child2);

System.out.println(root.getValue()); // 输出: Root
System.out.println(root.getChildren().get(0).getValue()); // 输出: Child1
```

### 总结
Java 泛型提供了强大的类型安全机制，使得代码更加通用和可重用。通过泛型类、泛型接口、泛型方法、通配符等特性，可以编写出更加灵活和健壮的代码。理解泛型的类型擦除、协变、逆变等概念，可以帮助开发者更好地利用泛型特性。
