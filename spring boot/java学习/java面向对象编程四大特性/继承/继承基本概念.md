Java中的继承是面向对象编程（OOP）的核心概念之一。继承允许一个类（子类）继承另一个类（父类）的属性和方法，从而实现代码的重用和扩展。继承的主要目的是实现代码的复用，并且可以通过继承来建立类之间的层次关系。

### 继承的基本概念

1. **父类（Superclass）**：被继承的类称为父类或超类。
2. **子类（Subclass）**：继承父类的类称为子类或派生类。
3. **单继承**：Java不支持多继承，一个类只能直接继承一个父类。
4. **方法重写（Override）**：子类可以重写父类的方法，以提供不同的实现。
5. **`super`关键字**：用于调用父类的构造方法或访问父类的成员。

### 继承的语法

```java
class 父类 {
    // 父类的属性和方法
}

class 子类 extends 父类 {
    // 子类的属性和方法
}
```

### 继承的示例

#### 1. 简单的继承

```java
// 父类
class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }

    public void eat() {
        System.out.println(name + " is eating.");
    }
}

// 子类
class Dog extends Animal {
    public Dog(String name) {
        super(name); // 调用父类的构造方法
    }

    public void bark() {
        System.out.println(name + " is barking.");
    }
}

// 测试类
public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog("Buddy");
        dog.eat();  // 输出: Buddy is eating.
        dog.bark(); // 输出: Buddy is barking.
    }
}
```

在这个例子中，`Dog`类继承了`Animal`类。`Dog`类可以使用`Animal`类的`name`属性和`eat()`方法，并且还定义了自己的`bark()`方法。

#### 2. 方法重写

```java
// 父类
class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }

    public void makeSound() {
        System.out.println(name + " makes a sound.");
    }
}

// 子类
class Dog extends Animal {
    public Dog(String name) {
        super(name);
    }

    @Override
    public void makeSound() {
        System.out.println(name + " barks.");
    }
}

// 测试类
public class Main {
    public static void main(String[] args) {
        Animal animal = new Animal("Generic Animal");
        animal.makeSound(); // 输出: Generic Animal makes a sound.

        Dog dog = new Dog("Buddy");
        dog.makeSound(); // 输出: Buddy barks.
    }
}
```

在这个例子中，`Dog`类重写了`Animal`类的`makeSound()`方法，提供了不同的实现。

#### 3. 使用`super`关键字

```java
// 父类
class Animal {
    String name;

    public Animal(String name) {
        this.name = name;
    }

    public void eat() {
        System.out.println(name + " is eating.");
    }
}

// 子类
class Dog extends Animal {
    String breed;

    public Dog(String name, String breed) {
        super(name); // 调用父类的构造方法
        this.breed = breed;
    }

    public void display() {
        System.out.println(name + " is a " + breed + ".");
        super.eat(); // 调用父类的eat()方法
    }
}

// 测试类
public class Main {
    public static void main(String[] args) {
        Dog dog = new Dog("Buddy", "Golden Retriever");
        dog.display(); 
        // 输出:
        // Buddy is a Golden Retriever.
        // Buddy is eating.
    }
}
```

在这个例子中，`Dog`类使用`super`关键字调用父类的构造方法和`eat()`方法。

### 继承的特点

1. **代码复用**：子类可以直接使用父类的属性和方法，减少代码重复。
2. **扩展性**：子类可以在父类的基础上添加新的属性和方法，或者重写父类的方法。
3. **多态性**：通过继承，可以实现多态，即父类引用可以指向子类对象。

### 继承的注意事项

1. **构造方法**：子类在实例化时会自动调用父类的无参构造方法。如果父类没有无参构造方法，子类必须通过`super`显式调用父类的构造方法。
2. **访问权限**：子类不能直接访问父类的私有成员（`private`），但可以通过父类提供的公共方法（`public`或`protected`）来访问。
3. **单继承**：Java不支持多继承，即一个类不能同时继承多个类。但可以通过接口实现类似的功能。

### 继承与组合

继承并不是唯一实现代码复用的方式。在某些情况下，组合（将一个类的实例作为另一个类的成员）可能比继承更合适。组合提供了更高的灵活性和更松散的耦合。

```java
class Engine {
    public void start() {
        System.out.println("Engine started.");
    }
}

class Car {
    private Engine engine;

    public Car() {
        engine = new Engine();
    }

    public void start() {
        engine.start();
        System.out.println("Car started.");
    }
}

public class Main {
    public static void main(String[] args) {
        Car car = new Car();
        car.start();
        // 输出:
        // Engine started.
        // Car started.
    }
}
```

在这个例子中，`Car`类通过组合的方式使用了`Engine`类，而不是通过继承。

### 总结

继承是Java中实现代码复用和扩展的重要机制。通过继承，子类可以复用父类的属性和方法，并且可以通过重写父类的方法来提供不同的实现。然而，继承并不总是最佳选择，在某些情况下，组合可能更为合适。理解继承的优缺点，并根据具体需求选择合适的代码复用方式，是编写高质量Java代码的关键。
