您需要创建一个`BusinessException`类，这是一个自定义异常，用于处理业务逻辑异常。根据您的代码结构，我建议将它放在与其他异常类相同的包中。
```java
BusinessException.java

sky-common/src/main/java/com/sky/exception

Create

package com.sky.exception;

/**

 * 业务逻辑异常

 */

public class BusinessException extends BaseException {

    public BusinessException() {

    }

    public BusinessException(String msg) {

        super(msg);

    }

}

```

这个异常类继承自您已有的

`BaseException`

类，可以用于表示各种业务逻辑错误。

然后，您可以在

`GlobalExceptionHandler`

中添加对这个异常的处理：
```java
GlobalExceptionHandler.java

sky-server/src/main/java/com/sky/handler

Apply

/**

 * 处理业务逻辑异常

 * @param ex

 * @return

 */

@ExceptionHandler

public Result exceptionHandler(BusinessException ex){

    log.error("业务逻辑异常：{}", ex.getMessage());

    return Result.error(ex.getMessage());

}

```
