

以下是 HTML 中常用的行内元素（inline elements）和块级元素（block elements）的分类及典型示例：
[[注意事项]]
---

### **1. 行内元素（Inline Elements）**
**特点**：  
- 默认不换行，与其他行内元素共享一行。  
- **无法直接设置宽高**（需通过 `display: inline-block` 或转换为块级元素）。  
- 只能包含文本或其他行内元素（某些特殊元素例外，如 `<a>`）。

#### **常见行内元素**：
| 元素                | 说明                        | 示例                          |
|---------------------|----------------------------|-------------------------------|
| `<span>`            | 通用行内容器                | `<span>文本</span>`           |
| `<a>`               | 超链接                      | `<a href="#">链接</a>`        |
| `<img>`             | 图像（行内替换元素）        | `<img src="image.jpg">`       |
| `<strong>`          | 加粗文本（语义化强调）      | `<strong>重要内容</strong>`   |
| `<em>`              | 斜体文本（语义化强调）      | `<em>强调内容</em>`           |
| `<br>`              | 换行                        | `文本<br>换行`                |
| `<input>`           | 输入框（行内替换元素）      | `<input type="text">`         |
| `<label>`           | 表单标签                    | `<label for="name">姓名</label>` |
| `<code>`            | 代码片段                    | `<code>console.log()</code>`  |
| `<button>`          | 按钮（默认 `inline-block`） | `<button>点击</button>`       |

---

### **2. 块级元素（Block Elements）**
**特点**：  
- 默认独占一行（前后换行）。  
- 可以设置宽高（`width`, `height`）。  
- 可包含其他块级元素和行内元素。

#### **常见块级元素**：
| 元素                | 说明                        | 示例                          |
|---------------------|----------------------------|-------------------------------|
| `<div>`             | 通用块容器                  | `<div>内容</div>`             |
| `<p>`               | 段落                        | `<p>段落文本</p>`             |
| `<h1>`-`<h6>`       | 标题                        | `<h1>主标题</h1>`             |
| `<ul>`, `<ol>`      | 无序/有序列表               | `<ul><li>项目</li></ul>`      |
| `<li>`              | 列表项                      | `<li>列表项</li>`             |
| `<table>`           | 表格                        | `<table>...</table>`          |
| `<form>`            | 表单                        | `<form>...</form>`            |
| `<header>`          | 页眉（HTML5 语义化）        | `<header>头部内容</header>`   |
| `<footer>`          | 页脚（HTML5 语义化）        | `<footer>底部内容</footer>`   |
| `<section>`         | 内容区块（HTML5 语义化）    | `<section>...</section>`      |
| `<article>`         | 文章（HTML5 语义化）        | `<article>...</article>`      |
| `<nav>`             | 导航栏（HTML5 语义化）      | `<nav>导航链接</nav>`         |

---

### **3. 特殊元素类型**
#### **行内块元素（Inline-Block）**  
- **特点**：具有行内元素不换行的特性，但支持设置宽高。  
- **常见元素**：  
  - `<img>`（默认 `inline`，但表现类似 `inline-block`）  
  - `<button>`（默认 `inline-block`）  
  - 通过 CSS 转换：`display: inline-block`。

#### **替换元素（Replaced Elements）**  
- **特点**：内容由外部资源决定（如 `<img>`, `<input>`）。  
- 行为可能类似行内元素或块级元素，但可设置宽高。

---

### **4. 通过 CSS 修改显示类型**
可以通过 `display` 属性强制改变元素行为：  
```css
span {
  display: block; /* 将行内元素转为块级 */
}
div {
  display: inline; /* 将块级元素转为行内 */
}
div {
  display: inline-block; /* 混合特性（可设置宽高且不换行） */
}
```

---

### **总结**
- **行内元素**适用于文本修饰或小内容块（如链接、图标）。  
- **块级元素**适用于布局容器或大内容块（如段落、列表）。  
- 合理使用 `display` 属性可灵活控制布局（如 `flex`, `grid`）。
