

以下是常用HTML标签分类列表（按功能分类）：

---

### **一、基础结构标签**
1. `<!DOCTYPE>` - 声明文档类型
2. `<html>` - 文档根元素
3. `<head>` - 头部容器（元信息）
4. `<title>` - 页面标题
5. `<body>` - 主体内容容器

---

### **二、元信息标签**
1. `<meta>` - 页面元数据（字符集/视口设置等）
2. `<link>` - 外部资源链接（CSS/图标）
3. `<style>` - 内联CSS样式
4. `<script>` - JavaScript代码或引用

---

### **三、文本内容标签**
[[文本标签]]
1. 标题层级：`<h1>` ~ `<h6>`
2. 段落文本：`<p>`
3. 格式修饰：
   - `<strong>` 重要文本（加粗）
   - `<em>` 强调文本（斜体）
   - `<u>` 下划线
   - `<del>` 删除线
4. 特殊文本：
   - `<a>` 超链接
   - `<br>` 换行
   - `<hr>` 水平线
   - `<pre>` 预格式化文本
   - `<code>` 代码片段

---

### **四、媒体内容标签**
1. 图片：`<img>`
2. 视频：`<video>`
3. 音频：`<audio>`
4. 画布：`<canvas>`
5. 矢量图：`<svg>`

---

### **五、表单相关标签**
1. 表单容器：`<form>`
2. 输入控件：
   - `<input>`（文本/密码/单选/复选框等）
   - `<textarea>` 多行文本
   - `<select>` 下拉列表
   - `<button>` 按钮
3. 表单标签：`<label>`
4. 选项分组：`<optgroup>`
5. 数据列表：`<datalist>`

---

### **六、列表标签**
1. 无序列表：`<ul>` + `<li>`
2. 有序列表：`<ol>` + `<li>`
3. 自定义列表：`<dl>` + `<dt>` + `<dd>`

---

### **七、表格标签**
1. 表格容器：`<table>`
2. 表格行：`<tr>`
3. 表头单元格：`<th>`
4. 数据单元格：`<td>`
5. 结构分组：
   - `<thead>` 表头区域
   - `<tbody>` 表格主体
   - `<tfoot>` 表尾区域

---

### **八、HTML5语义化标签**
1. 页眉：`<header>`
2. 页脚：`<footer>`
3. 导航栏：`<nav>`
4. 文章内容：`<article>`
5. 独立区块：`<section>`
6. 侧边栏：`<aside>`
7. 主要内容：`<main>`
8. 详细信息：`<details>` + `<summary>`

---

### **九、交互组件标签**
1. 进度条：`<progress>`
2. 范围滑块：`<input type="range">`
3. 对话框：`<dialog>`
4. 折叠面板：`<details>`

---

### **注意事项**
1. **版本差异**：部分标签需HTML5支持（如`<canvas>`）
2. **语义化优先**：推荐使用`<header>`/`<nav>`等语义标签替代通用`<div>`
3. **过时标签**：避免使用`<font>`/`<center>`等废弃标签，改用CSS实现样式
4. **SEO优化**：合理使用`<meta>`标签和语义化结构
