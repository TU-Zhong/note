

以下是 HTML 中 `<ul>`（无序列表）和 `<ol>`（有序列表）的 **详细知识点和示例**，涵盖语法、属性、嵌套规则和 CSS 定制：

---

### **1. `<ul>` 无序列表**
**作用**：展示没有特定顺序的项目列表。  
**默认样式**：每个 `<li>` 前有 `•`（圆点）。

#### **核心知识点**：
- **标签结构**：
  ```html
  <ul>
    <li>项目1</li>
    <li>项目2</li>
  </ul>
  ```
- **已弃用属性**（不建议使用，推荐用 CSS）：
  - `type="disc|circle|square"`：设置项目符号（圆点、空心圆、方块）。
- **CSS 替代方案**：
  ```css
  ul {
    list-style-type: square; /* 定义符号 */
    list-style-image: url('icon.png'); /* 自定义图片符号 */
    padding-left: 20px; /* 控制缩进 */
  }
  ```

#### **示例**：
```html
<ul>
  <li>咖啡</li>
  <li>茶</li>
  <li>牛奶</li>
</ul>
```
**效果**：
- 咖啡
- 茶
- 牛奶

---

### **2. `<ol>` 有序列表**
**作用**：展示有明确顺序的项目列表（如步骤、排名）。  
**默认样式**：从 `1` 开始的数字编号。

#### **核心知识点**：
- **标签结构**：
  ```html
  <ol>
    <li>第一步</li>
    <li>第二步</li>
  </ol>
  ```
- **关键属性**：
  - `type="1|A|a|I|i"`：编号类型（数字、大写字母、小写字母、罗马数字）。
  - `start="n"`：起始编号（如 `start="3"` 表示从 3 开始）。
  - `reversed`：倒序显示编号（如 `3, 2, 1`）。
- **`<li>` 的特殊属性**：
  - `value="n"`：覆盖当前项的编号（仅对 `<ol>` 有效）。
- **CSS 替代方案**：
  ```css
  ol {
    list-style-type: upper-roman; /* 大写罗马数字 */
    list-style-position: inside; /* 符号在内容框内 */
  }
  ```

#### **示例**：
```html
<ol type="A" start="3">
  <li>准备材料</li>
  <li value="5">混合搅拌</li> <!-- 强制编号为 E -->
  <li>烘烤</li>
</ol>
```
**效果**：
C. 准备材料  
E. 混合搅拌  
F. 烘烤

---

### **3. 嵌套列表**
**规则**：在 `<li>` 中嵌套另一个 `<ul>` 或 `<ol>`，形成树形结构。

#### **示例**：
```html
<ul>
  <li>动物
    <ol>
      <li>狮子</li>
      <li>老虎</li>
    </ol>
  </li>
  <li>植物
    <ul>
      <li>松树</li>
      <li>玫瑰</li>
    </ul>
  </li>
</ul>
```
**效果**：
- 动物
  1. 狮子
  2. 老虎
- 植物
  - 松树
  - 玫瑰

---

### **4. 使用 CSS 自定义列表**
**常见需求**：移除默认符号、水平排列、自定义样式。

#### **示例**：水平导航菜单
```html
<style>
  ul.horizontal {
    list-style-type: none; /* 移除默认符号 */
    padding: 0;
    display: flex;
    gap: 20px; /* 项目间距 */
    background: #f0f0f0;
  }
  ul.horizontal li {
    padding: 10px;
  }
</style>

<ul class="horizontal">
  <li>首页</li>
  <li>产品</li>
  <li>关于我们</li>
</ul>
```
**效果**：  
[首页] [产品] [关于我们]（水平排列，无符号）

---

### **5. HTML5 特性**
- **`<ol>` 的 `reversed` 属性**：
  ```html
  <ol reversed>
    <li>第三名</li>
    <li>第二名</li>
    <li>第一名</li>
  </ol>
  ```
  **效果**：3. 第三名 → 2. 第二名 → 1. 第一名。
- **语义化**：`<ul>` 和 `<ol>` 默认具有 `list` 和 `listitem` 的 ARIA 角色，增强无障碍体验。

---

### **6. 注意事项**
1. **勿滥用**：列表应用于结构性内容，而非布局。
2. **替代方案**：复杂布局推荐使用 `<div>` + CSS Grid/Flexbox。
3. **兼容性**：`reversed` 属性在旧浏览器中可能失效。

---

### **总结**
| 特性          | `<ul>`               | `<ol>`               |
|---------------|----------------------|----------------------|
| **顺序**      | 无顺序               | 有顺序               |
| **默认符号**  | 圆点 (`•`)          | 数字 (`1.`)         |
| **嵌套支持**  | 支持                 | 支持                 |
| **关键属性**  | `type`（已弃用）     | `type`, `start`, `reversed` |

通过合理使用 `<ul>` 和 `<ol>`，可以清晰展示结构化内容；通过 CSS 可实现高度定制的视觉效果。
