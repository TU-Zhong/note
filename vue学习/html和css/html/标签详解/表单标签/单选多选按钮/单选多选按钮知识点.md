### **一、<mark style="background: #FFF3A3A6;">`label` 标签的作用**</mark>
- <mark style="background: #FFF3A3A6;"> **一句话解释**：`label` 是用来 **点文字也能选中按钮** 的！</mark>
- **具体好处**：
  1. 提升用户体验：不用非得点小圆圈/小方框，点旁边的文字也能选中。
  2. 对残障人士友好：屏幕阅读器能正确识别按钮的用途。

---

### **二、单选按钮（Radio）**
#### **特点**：
- **只能选一个**，适合互斥选项（如性别、单选题答案）。
- **同一组的单选按钮必须共享 `name` 属性**！

#### **代码示例**：
```html
<!-- 性别选择 -->
<label>
  <input type="radio" name="gender" value="male"> 男
</label>
<label>
  <input type="radio" name="gender" value="female"> 女
</label>
```

#### **关键点**：
- **设置默认选中**：用 `checked` 属性。
  ```html
  <input type="radio" name="gender" value="male" checked> 男
  ```
- **用 `for` 关联 `id`（推荐写法）**：  
  （适合需要精细化布局的情况）
  ```html
  <input type="radio" name="color" id="red" value="red">
  <label for="red">红色</label>

  <input type="radio" name="color" id="blue" value="blue">
  <label for="blue">蓝色</label>
  ```

---

### **三、多选按钮（Checkbox）**
#### **特点**：
- **可以选多个**，适合非互斥选项（如兴趣爱好）。
- 每个多选按钮的 `name` 可以相同或不同（通常相同）。

#### **代码示例**：
```html
<!-- 兴趣爱好 -->
<label>
  <input type="checkbox" name="hobby" value="music"> 音乐
</label>
<label>
  <input type="checkbox" name="hobby" value="sports"> 运动
</label>
```

#### **关键点**：
- **设置默认选中**：同样用 `checked`。
  ```html
  <input type="checkbox" name="hobby" value="music" checked> 音乐
  ```
- **提交数据格式**：  
  后端收到的数据是数组形式（如 `hobby=music&hobby=sports`）。

---

### **四、可视化对比**
| 类型        | 外观        | 选中效果       | 代码区别             |
|------------|------------|--------------|---------------------|
| 单选按钮    | ⚪️         | 只能选一个     | `type="radio"`      |
| 多选按钮    | ✅          | 可以选多个     | `type="checkbox"`   |

---

### **五、完整示例**
```html
<form>
  <!-- 单选按钮组 -->
  <fieldset>
    <legend>性别</legend>
    <label>
      <input type="radio" name="gender" value="male" checked> 男
    </label>
    <label>
      <input type="radio" name="gender" value="female"> 女
    </label>
  </fieldset>

  <!-- 多选按钮组 -->
  <fieldset>
    <legend>兴趣爱好</legend>
    <label>
      <input type="checkbox" name="hobby" value="music" checked> 音乐
    </label>
    <label>
      <input type="checkbox" name="hobby" value="sports"> 运动
    </label>
    <label>
      <input type="checkbox" name="hobby" value="reading"> 阅读
    </label>
  </fieldset>

  <button type="submit">提交</button>
</form>
```

---

### **六、注意事项**
1. **`name` 必须一致**（单选按钮），否则无法互斥。
2. **`value` 要明确**：表单提交时发送的值。
3. **不要用 `label` 包裹其他复杂元素**（如 div），可能导致点击失效。
4. **样式美化**：用 CSS 隐藏原生样式，自定义按钮外观。
