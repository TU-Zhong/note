### **练习题：设计一个披萨点单表单**

#### **需求描述**
用户需要在线点一份披萨，具体要求如下：
1. **主菜（单选）**：必选1种披萨类型（玛格丽特、海鲜至尊、素食主义）。
2. **配料（多选）**：可选多种额外配料（芝士加倍、蘑菇、火腿、橄榄）。
3. **饮料（多选）**：可选多种饮料（可乐、雪碧、柠檬茶）。
4. **甜点（单选）**：可选1种甜点（提拉米苏、冰淇淋、无甜点）。
5. **备注**：允许用户输入特殊要求（如少盐、不要洋葱）。

---

### **需要实现的功能**
1. 使用 `<input type="radio">` 和 `<input type="checkbox">`。
2. 所有选项必须关联 `<label>`，且文本可点击。
3. 主菜和甜点为必选项（添加 `required` 属性）。
4. 表单提交时，验证是否选择了必填项。

### **测试方法**
1. 点击各选项的文本，确认可以选中。
2. 尝试不选主菜或甜点，提交时应有浏览器验证提示。
3. 提交后，查看后端收到的数据格式是否符合预期（如多选值以数组形式提交）。
