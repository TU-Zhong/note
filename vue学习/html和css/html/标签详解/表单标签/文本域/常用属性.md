

以下是 `<textarea>` 标签的常用属性及示例，涵盖基础功能、输入限制和样式控制：

---
[[如何禁止用户拖拽文本域使其放大]]
### **常用属性列表**

| 属性              | 类型/值          | 说明                                                                 |
|-------------------|-----------------|--------------------------------------------------------------------|
| `rows`            | 数字（如 `5`）    | **可见行数**：设置文本域的默认显示高度（行数）。                                |
| `cols`            | 数字（如 `30`）   | **可见列数**：设置文本域的默认显示宽度（字符数）。                              |
| `name`            | 字符串（如 `bio`）| **字段名称**：表单提交时用于标识该字段（必填）。                               |
| `placeholder`     | 字符串           | **提示文本**：输入为空时显示的灰色提示。                                   |
| `maxlength`       | 数字（如 `200`）  | **最大输入长度**：限制用户输入的字符总数。                                  |
| `minlength`       | 数字（如 `10`）   | **最小输入长度**：输入字符数必须≥该值（HTML5）。                             |
| `required`        | -               | **必填字段**：提交前必须填写内容（HTML5验证）。                              |
| `readonly`        | -               | **只读**：内容不可修改（但会被提交）。                                    |
| `disabled`        | -               | **禁用**：内容不可修改且不会被提交。                                      |
| `autofocus`       | -               | **自动聚焦**：页面加载时自动将光标定位到该文本域。                             |
| `wrap`            | `soft`/`hard`   | **换行处理**：`soft`（默认，不强制换行）、`hard`（提交时保留换行）。               |
| `autocomplete`    | `on`/`off`      | **自动填充**：控制浏览器是否自动填充历史输入（如 `off` 用于敏感内容）。              |
| `form`            | 表单ID           | **关联表单**：指定所属的表单（即使不在 `<form>` 内）。                        |
| `spellcheck`      | `true`/`false`  | **拼写检查**：是否启用浏览器的拼写检查（默认跟随浏览器设置）。                      |

---

### **示例代码**

#### 1. 基础文本域
```html
<textarea rows="5" cols="30" name="message" placeholder="请输入你的留言..."></textarea>
```

#### 2. 带输入限制和验证
```html
<textarea 
  name="bio" 
  rows="4" 
  cols="50"
  placeholder="个人简介（至少20字）"
  minlength="20" 
  maxlength="200"
  required
></textarea>
```

#### 3. 禁用或只读状态
```html
<!-- 禁用 -->
<textarea rows="3" cols="40" name="comment" disabled>该评论已关闭</textarea>

<!-- 只读 -->
<textarea rows="3" cols="40" name="terms" readonly>用户协议内容...</textarea>
```

#### 4. 自动聚焦与换行控制
```html
<textarea 
  name="content" 
  rows="6" 
  cols="60" 
  autofocus 
  wrap="hard"
  placeholder="输入内容（换行将被保留）"
></textarea>
```

---

### **样式增强（CSS）**
```css
/* 禁用默认的拖放调整大小 */
textarea {
  resize: none;
}

/* 自定义样式 */
textarea.custom {
  width: 100%;
  padding: 10px;
  border: 2px solid #ccc;
  border-radius: 8px;
  font-family: Arial;
}
```

---

### **关键注意事项**
1. **`name` 属性必填**：否则表单提交时无法获取该字段的值。
2. **输入长度验证**：`maxlength` 和 `minlength` 需结合后端验证（前端可绕过）。
3. **换行处理**：`wrap="hard"` 需同时设置 `cols` 才能生效。
4. **禁用默认调整**：通过 `resize: none` 禁止用户拖放改变文本域大小。

通过上述属性和示例，你可以灵活控制 `<textarea>` 的输入行为和样式！
