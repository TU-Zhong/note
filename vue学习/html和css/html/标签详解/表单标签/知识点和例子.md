

以下是 HTML 表单的详细知识点和示例，涵盖常用标签、属性及表单验证等内容：

---

### 一、表单基础
#### **`<form>` 标签**
- 用于创建表单，所有表单元素需包裹在 `<form>` 标签内。
- 核心属性：
  - `action`: 表单提交的 URL（后端接口地址）。
  - `method`: 数据提交方式（`GET` 或 `POST`）。
  - `enctype`: 编码方式（如上传文件时需要 `enctype="multipart/form-data"`）。

```html
<form action="/submit" method="POST">
  <!-- 表单元素在此 -->
</form>
```

---

### 二、表单元素
[[常用属性列表和含义]]
#### **1. 输入框 `<input>`**
- 通过 `type` 属性定义不同类型：
  - **文本输入**：`text`, `password`, `email`, `tel`, `search`
    ```html
    <input type="text" name="username" placeholder="请输入用户名">
    <input type="password" name="password">
    <input type="email" name="email" required>
    ```
  - **数字输入**：`number`, `range`
    ```html
    <input type="number" name="age" min="18" max="100">
    <input type="range" name="volume" min="0" max="100">
    ```
  - **日期时间**：`date`, `time`, `datetime-local`
    ```html
    <input type="date" name="birthday">
    ```
  - **选择文件**：`file`
    ```html
    <input type="file" name="avatar" accept="image/*">
    ```
  - **隐藏字段**：`hidden`
    ```html
    <input type="hidden" name="token" value="abc123">
    ```

---

#### **2. 文本域 `<textarea>`**
- 多行文本输入。
  ```html
  <textarea name="message" rows="5" cols="30" placeholder="输入留言"></textarea>
  ```

---

#### **3. 下拉列表 `<select>`**
- 与 `<option>` 和 `<optgroup>` 配合使用：
  ```html
  <select name="country">
    <option value="">请选择国家</option>
    <option value="cn">中国</option>
    <option value="us">美国</option>
  </select>

  <!-- 分组 -->
  <select name="car">
    <optgroup label="德系车">
      <option value="bmw">宝马</option>
      <option value="benz">奔驰</option>
    </optgroup>
    <optgroup label="日系车">
      <option value="toyota">丰田</option>
    </optgroup>
  </select>
  ```

---

#### **4. 单选/多选按钮**
- **单选按钮（Radio）**：同一组需共享 `name`。
  ```html
  <input type="radio" name="gender" value="male" id="male">
  <label for="male">男</label>

  <input type="radio" name="gender" value="female" id="female">
  <label for="female">女</label>
  ```
- **复选框（Checkbox）**：
  ```html
  <input type="checkbox" name="hobby" value="music" id="music">
  <label for="music">音乐</label>
  <input type="checkbox" name="hobby" value="sports" id="sports">
  <label for="sports">运动</label>
  ```

---

#### **5. 按钮**
- `<button>` 或 `<input type="button">`：
  ```html
  <button type="submit">提交</button>
  <button type="reset">重置</button>
  <button type="button" onclick="alert('点击')">普通按钮</button>

  <!-- 等价于 -->
  <input type="submit" value="提交">
  <input type="reset" value="重置">
  ```

---

### 三、表单分组与标签
#### **1. `<label>` 标签**
- 关联表单元素，提高可访问性。
  ```html
  <label for="username">用户名：</label>
  <input type="text" id="username" name="username">
  ```

#### **2. `<fieldset>` 和 `<legend>`**
- 分组表单并添加标题：
  ```html
  <fieldset>
    <legend>用户信息</legend>
    <input type="text" name="name">
    <input type="email" name="email">
  </fieldset>
  ```

---

### 四、HTML5 新增功能
#### **1. 输入类型增强**
- `color`, `url`, `datetime`, `month`, `week` 等：
  ```html
  <input type="color" name="favcolor">
  <input type="url" name="website">
  ```

#### **2. 表单验证**
- 通过 HTML5 属性实现基础验证：
  - `required`: 必填字段。
  - `min`/`max`: 数值/日期范围。
  - `maxlength`/`minlength`: 文本长度限制。
  - `pattern`: 正则表达式验证。
  ```html
  <input type="text" name="phone" pattern="[0-9]{11}" required>
  <input type="number" name="score" min="0" max="100">
  ```

---

### 五、综合示例
```html
<form action="/submit" method="POST">
  <fieldset>
    <legend>注册表单</legend>
    
    <label for="username">用户名：</label>
    <input type="text" id="username" name="username" required minlength="4">

    <label for="email">邮箱：</label>
    <input type="email" id="email" name="email" required>

    <label for="password">密码：</label>
    <input type="password" id="password" name="password" required>

    <label for="age">年龄：</label>
    <input type="number" id="age" name="age" min="18" max="100">

    <label>性别：</label>
    <input type="radio" name="gender" value="male" id="male" required>
    <label for="male">男</label>
    <input type="radio" name="gender" value="female" id="female" required>
    <label for="female">女</label>

    <label for="bio">个人简介：</label>
    <textarea id="bio" name="bio" rows="4"></textarea>

    <button type="submit">提交</button>
    <button type="reset">重置</button>
  </fieldset>
</form>
```

---

### 六、关键注意事项
1. **`name` 属性**：表单提交时，数据以 `name=value` 的形式发送。
2. **`id` 属性**：与 `<label>` 的 `for` 属性关联，提升可访问性。
3. **表单验证**：结合 HTML5 验证和 JavaScript 增强用户体验。
4. **安全性**：敏感数据建议用 `POST` 方法提交，服务端需做数据过滤。

通过以上知识点和示例，你可以构建功能完整且符合标准的 HTML 表单！
