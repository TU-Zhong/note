

以下是 HTML 中常用的 **媒体内容标签** 的详细知识点和示例，适用于图片、音视频、画布等场景：

---

### **1. 图片 `<img>`**
**作用**：嵌入图像。  
**常用属性**：
- `src`：图片路径（必填）。
- `alt`：替代文本（图片无法显示时展示，必填）。
- `width`/`height`：设置宽高（建议用 CSS 替代）。
- `loading`：懒加载（`lazy` 延迟加载，提升性能）。

```html
<img src="cat.jpg" alt="一只可爱的猫" width="300" loading="lazy">
```

---

### **2. 音频 `<audio>`**
**作用**：嵌入音频文件。  
**常用属性**：
- `src`：音频路径。
- `controls`：显示播放控件。
- `autoplay`：自动播放（可能被浏览器阻止）。
- `loop`：循环播放。
- `muted`：静音。

```html
<audio controls src="music.mp3">
  Your browser does not support the audio tag.
</audio>
```

**多格式支持**：
```html
<audio controls>
  <source src="music.mp3" type="audio/mpeg">
  <source src="music.ogg" type="audio/ogg">
  浏览器不支持音频播放。
</audio>
```

---

### **3. 视频 `<video>`**
**作用**：嵌入视频文件。  
**常用属性**：
- `src`：视频路径。
- `controls`：显示播放控件。
- `width`/`height`：设置尺寸。
- `poster`：视频封面图。
- `preload`：预加载（`auto`/`metadata`/`none`）。

```html
<video controls width="600" poster="video-poster.jpg">
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.webm" type="video/webm">
  浏览器不支持视频播放。
</video>
```

---

### **4. 画布 `<canvas>`**
**作用**：通过 JavaScript 绘制图形、动画。  
**常用属性**：
- `width`/`height`：设置画布区域尺寸。

```html
<canvas id="myCanvas" width="200" height="100"></canvas>
<script>
  const canvas = document.getElementById('myCanvas');
  const ctx = canvas.getContext('2d');
  ctx.fillStyle = 'red';
  ctx.fillRect(10, 10, 150, 80);
</script>
```

---

### **5. 矢量图形 `<svg>`**
**作用**：嵌入矢量图形，适合图标、图表。  
**示例**：
```html
<svg width="100" height="100">
  <circle cx="50" cy="50" r="40" fill="blue" />
</svg>
```

---

### **6. 外部内容 `<iframe>`**
**作用**：嵌入其他网页或第三方内容（如地图、视频）。  
**常用属性**：
- `src`：外部资源 URL。
- `width`/`height`：尺寸。
- `frameborder`：边框（通常设为 `0`）。
- `allowfullscreen`：允许全屏。

```html
<iframe 
  src="https://example.com" 
  width="600" 
  height="400" 
  frameborder="0"
  allowfullscreen>
</iframe>
```

---

### **7. 响应式图片 `<picture>`**
**作用**：根据设备条件加载不同图片（响应式设计）。  
**常见场景**：
- 不同屏幕尺寸加载不同分辨率图片。
- 支持新格式（如 WebP）但提供回退。

```html
<picture>
  <source media="(min-width: 1200px)" srcset="large.jpg">
  <source media="(min-width: 800px)" srcset="medium.jpg">
  <img src="small.jpg" alt="响应式图片">
</picture>
```

---

### **8. 多媒体资源 `<source>`**
**作用**：在 `<audio>`、`<video>`、`<picture>` 中定义多种资源格式。  
**常用属性**：
- `src`：资源路径。
- `type`：媒体类型（如 `video/mp4`）。

```html
<video controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.ogg" type="video/ogg">
  浏览器不支持视频播放。
</video>
```

---

### **9. 字幕/说明 `<track>`**
**作用**：为视频或音频添加字幕、说明（如 `.vtt` 文件）。  
**常用属性**：
- `src`：字幕文件路径。
- `kind`：类型（`subtitles`/`captions`/`descriptions`）。
- `srclang`：语言（如 `en`、`zh`）。
- `label`：标题（如“中文字幕”）。

```html
<video controls>
  <source src="movie.mp4" type="video/mp4">
  <track src="subs-chinese.vtt" kind="subtitles" srclang="zh" label="中文">
</video>
```

---

### **10. 传统媒体嵌入 `<embed>` 和 `<object>`**
**作用**：嵌入外部应用或插件内容（如 Flash），已逐渐被淘汰。  
**示例**：
```html
<embed src="game.swf" type="application/x-shockwave-flash">
<object data="chart.pdf" type="application/pdf" width="400" height="300"></object>
```

---

### **最佳实践**
- **性能优化**：使用 `loading="lazy"` 延迟加载非首屏图片。
- **无障碍性**：为 `<img>` 添加 `alt`，为视频添加 `<track>`。
- **响应式设计**：优先使用 `<picture>` 和 `srcset`。
- **格式兼容**：为音视频提供多种格式的 `<source>`。

通过这些标签，可以高效地在网页中集成丰富的媒体内容！ 🎥🎵
