

以下是 HTML 表格的详细知识点和示例，涵盖基础结构、属性、合并单元格及最佳实践：

---

### **1. 表格基础结构**


| 标签       | 作用                                                                 |
|------------|--------------------------------------------------------------------|
| `<table>`  | 定义表格容器                                                          |
| `<tr>`     | 表格行（Table Row）                                                  |
| `<th>`     | 表头单元格（Header Cell），默认加粗居中                                   |
| `<td>`     | 表格数据单元格（Data Cell）                                             |

**示例：基础表格**  
```html
<table>
  <tr>
    <th>姓名</th>
    <th>年龄</th>
  </tr>
  <tr>
    <td>张三</td>
    <td>25</td>
  </tr>
</table>
```

---

### **2. 表格头部、主体和底部**
[[在  tbody  中的元素占据两行似乎不能拿  tfoot  中的元素的格子，怎么解释这个事实？]]

| 标签          | 作用                                                              |
|---------------|-----------------------------------------------------------------|
| `<thead>`     | 定义表格头部（包裹表头行）                                            |
| `<tbody>`     | 定义表格主体（包裹数据行）                                            |
| `<tfoot>`     | 定义表格底部（如汇总行）                                               |

**示例：分组结构**  
```html
<table>
  <thead>
    <tr>
      <th>商品</th>
      <th>价格</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>苹果</td>
      <td>5元</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>总计</td>
      <td>5元</td>
    </tr>
  </tfoot>
</table>
```

### **4. 表格标题与样式控制**
#### **标题**  
用 `<caption>` 添加表格标题：
```html
<table>
  <caption>学生成绩表</caption>
  <!-- 表格内容 -->
</table>
```

#### **样式属性（建议用 CSS 替代）**  
| 属性              | 作用               |
|-------------------|--------------------|
| `border="1"`      | 边框（直接写在 `<table>` 上） |
| `width`/`height`  | 宽高               |
| `cellspacing`     | 单元格间距          |
| `cellpadding`     | 单元格内边距        |

**示例：简单样式**  
```html
<table border="1" cellspacing="0" cellpadding="8">
  <!-- 表格内容 -->
</table>
```

---

### **5. 列分组与样式**
| 标签          | 作用                             |
|---------------|----------------------------------|
| `<colgroup>`  | 定义表格列的分组（包裹 `<col>` 标签）   |
| `<col>`       | 定义列样式（如宽度、背景色）           |

**示例：设置列宽度**  
```html
<table>
  <colgroup>
    <col style="width: 100px"> <!-- 第一列宽 100px -->
    <col style="width: 200px"> <!-- 第二列宽 200px -->
  </colgroup>
  <tr>
    <th>姓名</th>
    <th>地址</th>
  </tr>
  <tr>
    <td>李四</td>
    <td>上海</td>
  </tr>
</table>
```

---

### **6. 最佳实践**
1. **可访问性**：为表头添加 `scope` 属性：
   ```html
   <th scope="col">价格</th> <!-- 关联列 -->
   <th scope="row">商品</th> <!-- 关联行 -->
   ```

2. **响应式表格**：为移动端添加水平滚动容器：
   ```html
   <div style="overflow-x: auto;">
     <table>
       <!-- 宽表格内容 -->
     </table>
   </div>
   ```

3. **避免复杂嵌套**：表格中不要嵌套表格或过多 `<div>`。

### **完整示例**
```html
<table border="1" cellspacing="0" cellpadding="8">
  <caption>员工信息表</caption>
  <colgroup>
    <col style="background: #f0f0f0">
    <col>
  </colgroup>
  <thead>
    <tr>
      <th scope="col">姓名</th>
      <th scope="col">部门</th>
    </tr>
  </thead>
  <tbody>
    <tr>

      <td rowspan="2">技术部</td>
    </tr>
    <tr>
      <td>赵六</td>
    </tr>
  </tbody>
</table>
```
