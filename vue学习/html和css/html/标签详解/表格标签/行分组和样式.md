

以下是 HTML 表格中 **行分组** 和 **样式控制** 的详细知识点及示例，帮助你更灵活地管理表格结构和外观：

---

### **行分组标签**
HTML 提供了以下标签对表格行进行逻辑分组：

| 标签          | 作用                                                                 |
|---------------|--------------------------------------------------------------------|
| `<thead>`     | 定义表格的 **头部行组**（通常包含表头 `<th>`）。                        |
| `<tbody>`     | 定义表格的 **主体行组**（包含数据行 `<td>`）。                           |
| `<tfoot>`     | 定义表格的 **底部行组**（常用于汇总行）。                                |

**示例：基础分组**  
```html
<table border="1">
  <thead>
    <tr>
      <th>姓名</th>
      <th>分数</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>张三</td>
      <td>90</td>
    </tr>
    <tr>
      <td>李四</td>
      <td>85</td>
    </tr>
  </tbody>
  <tfoot>
    <tr>
      <td>平均分</td>
      <td>87.5</td>
    </tr>
  </tfoot>
</table>
```

---

### **样式控制**
通过 CSS 可为不同行组或行添加样式，提升可读性和视觉效果。

#### **1. 行组整体样式**
为 `<thead>`、`<tbody>`、`<tfoot>` 设置不同背景色或字体：
```html
<style>
  thead { background: #f0f0f0; }     /* 表头灰色背景 */
  tbody { font-family: Arial; }      /* 主体字体 */
  tfoot { font-weight: bold; }       /* 底部加粗 */
</style>
```

---

#### **2. 交替行颜色（斑马纹）**  
使用 `:nth-child(even)` 或 `:nth-child(odd)` 选择器：
```css
tbody tr:nth-child(even) { background: #f8f8f8; }  /* 偶数行浅灰背景 */
```

---

#### **3. 悬停高亮**  
为行添加悬停效果：
```css
tbody tr:hover { background: #e0e0ff; }  /* 鼠标悬停时蓝色背景 */
```

---

#### **4. 多 `<tbody>` 分组**  
通过多个 `<tbody>` 实现更细分的行组：
```html
<table>
  <thead>...</thead>
  <!-- 分组1 -->
  <tbody class="group1">
    <tr><td>分组1-行1</td></tr>
    <tr><td>分组1-行2</td></tr>
  </tbody>
  <!-- 分组2 -->
  <tbody class="group2">
    <tr><td>分组2-行1</td></tr>
    <tr><td>分组2-行2</td></tr>
  </tbody>
</table>
```

**样式示例**：
```css
.group1 { background: #fff0f0; }  /* 粉色背景 */
.group2 { background: #f0fff0; }  /* 绿色背景 */
```

---

#### **5. 特定行样式**  
通过类名或属性标记特定行：
```html
<tr class="highlight">
  <td>特殊数据</td>
  <td>100</td>
</tr>
```

```css
.highlight { background: gold; }  /* 高亮为金色 */
```

---

### **最佳实践**
1. **语义化**：始终使用 `<thead>`、`<tbody>`、`<tfoot>` 分组，提升可访问性。
2. **响应式优化**：为移动端添加水平滚动容器：
   ```html
   <div style="overflow-x: auto;">
     <table>...</table>
   </div>
   ```
3. **避免复杂嵌套**：不要在行内嵌套表格或过多 `<div>`。
4. **跨浏览器兼容**：测试不同浏览器下的样式表现，避免使用过于现代的 CSS 属性。

---

### **完整示例**
```html
<table border="1" style="width: 100%;">
  <thead style="background: #e0e0e0;">
    <tr>
      <th>产品</th>
      <th>价格</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>苹果</td>
      <td>5元</td>
    </tr>
    <tr style="background: #f8f8f8;">
      <td>香蕉</td>
      <td>3元</td>
    </tr>
  </tbody>
  <tfoot style="font-weight: bold;">
    <tr>
      <td>总计</td>
      <td>8元</td>
    </tr>
  </tfoot>
</table>
```
