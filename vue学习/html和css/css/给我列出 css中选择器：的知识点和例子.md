

以下是 **CSS 选择器** 的详细分类、知识点和示例，涵盖基本选择器、组合选择器、属性选择器、伪类和伪元素：

---

### **一、基本选择器**
#### **1. 元素选择器**
- **作用**：选择指定 HTML 元素。
- **语法**：`元素名 { ... }`
- **示例**：
  ```css
  p { color: red; } /* 所有段落文字变红 */
  ```

#### **2. 类选择器**
- **作用**：选择具有相同 `class` 属性的元素。
- **语法**：`.类名 { ... }`
- **示例**：
  ```css
  .text-bold { font-weight: bold; } /* class="text-bold" 的元素加粗 */
  ```

#### **3. ID 选择器**
- **作用**：选择具有指定 `id` 属性的元素（页面唯一）。
- **语法**：`#id名 { ... }`
- **示例**：
  ```css
  #header { background: blue; } /* id="header" 的元素背景蓝色 */
  ```

#### **4. 通配符选择器**
- **作用**：选择页面所有元素。
- **语法**：`* { ... }`
- **示例**：
  ```css
  * { margin: 0; } /* 清除所有元素的外边距 */
  ```

---

### **二、组合选择器**
#### **1. 后代选择器**
- **作用**：选择某个元素内的所有指定后代元素。
- **语法**：`父元素 后代元素 { ... }`
- **示例**：
  ```css
  div p { color: green; } /* div 内所有段落文字变绿 */
  ```

#### **2. 子元素选择器**
- **作用**：选择直接子元素（不包含孙子元素）。
- **语法**：`父元素 > 子元素 { ... }`
- **示例**：
  ```css
  ul > li { list-style: none; } /* 仅清除 ul 的直接子 li 的列表样式 */
  ```

#### **3. 相邻兄弟选择器**
- **作用**：选择紧跟在某元素后的第一个兄弟元素。
- **语法**：`前元素 + 相邻兄弟元素 { ... }`
- **示例**：
  ```css
  h1 + p { margin-top: 0; } /* h1 后的第一个段落移除上边距 */
  ```

#### **4. 通用兄弟选择器**
- **作用**：选择某元素后的所有兄弟元素。
- **语法**：`前元素 ~ 兄弟元素 { ... }`
- **示例**：
  ```css
  h2 ~ p { color: gray; } /* h2 后的所有段落文字变灰 */
  ```

#### **5. 群组选择器**
- **作用**：同时选择多个元素。
- **语法**：`元素1, 元素2, ... { ... }`
- **示例**：
  ```css
  h1, h2, h3 { font-family: Arial; } /* 所有 h1-h3 的字体设为 Arial */
  ```

---

### **三、属性选择器**
#### **1. 存在性选择器**
- **作用**：选择包含指定属性的元素。
- **语法**：`[属性名] { ... }`
- **示例**：
  ```css
  [target] { color: orange; } /* 所有带有 target 属性的元素文字橙色 */
  ```

#### **2. 精确匹配选择器**
- **作用**：选择属性值等于指定值的元素。
- **语法**：`[属性名="值"] { ... }`
- **示例**：
  ```css
  [type="text"] { border: 1px solid #ccc; } /* type="text" 的输入框加边框 */
  ```

#### **3. 子字符串匹配选择器**
- **语法**：
  - `[属性名^="前缀"]`：属性值以指定前缀开头。
  - `[属性名$="后缀"]`：属性值以指定后缀结尾。
  - `[属性名*="包含"]`：属性值包含指定字符串。
- **示例**：
  ```css
  a[href^="https"] { color: green; }       /* 链接以 https 开头 */
  img[src$=".png"] { border: 1px solid; }  /* src 以 .png 结尾的图片 */
  [class*="btn"] { padding: 8px; }         /* class 包含 "btn" 的元素 */
  ```

---

### **四、伪类选择器**
#### **1. 常用伪类**
- **`:hover`**：鼠标悬停时的样式。
  ```css
  a:hover { text-decoration: underline; } /* 鼠标悬停时链接显示下划线 */
  ```
- **`:active`**：元素被激活（如点击瞬间）的样式。
  ```css
  button:active { transform: scale(0.95); } /* 按钮点击时缩小 */
  ```
- **`:focus`**：元素获得焦点时的样式。
  ```css
  input:focus { outline: 2px solid blue; } /* 输入框聚焦时蓝色轮廓 */
  ```
- **`:nth-child(n)`**：选择父元素的第 `n` 个子元素。
  ```css
  li:nth-child(2n) { background: #f0f0f0; } /* 偶数行列表项灰色背景 */
  ```

#### **2. 表单相关伪类**
- **`:checked`**：选中状态的复选框或单选按钮。
  ```css
  input:checked + label { font-weight: bold; } /* 选中时关联标签加粗 */
  ```
- **`:disabled`**：禁用的表单元素。
  ```css
  input:disabled { opacity: 0.5; } /* 禁用输入框半透明 */
  ```

---

### **五、伪元素选择器**
#### **1. `::before` 和 `::after`**
- **作用**：在元素内容前/后插入虚拟内容。
- **语法**：
  ```css
  元素::before { content: "内容"; ... }
  元素::after { content: "内容"; ... }
  ```
- **示例**：
  ```css
  .price::before { content: "￥"; color: red; }  /* 价格前加红色¥符号 */
  .tooltip::after { content: "?"; margin-left: 5px; } /* 提示文字后加问号 */
  ```

#### **2. `::first-line` 和 `::first-letter`**
- **作用**：选择元素的第一行或首字母。
- **示例**：
  ```css
  p::first-line { font-weight: bold; }     /* 段落第一行加粗 */
  p::first-letter { font-size: 2em; }      /* 段落首字母放大 */
  ```

#### **3. `::selection`**
- **作用**：选择用户高亮（选中）的文本。
- **示例**：
  ```css
  ::selection { background: yellow; color: black; } /* 选中文本黄底黑字 */
  ```

---

### **六、优先级总结**
| 选择器类型       | 示例                | 优先级权重 |
|------------------|---------------------|------------|
| `内联样式`        | `style="..."`       | 1000       |
| `ID 选择器`       | `#header`           | 100        |
| `类/属性/伪类`    | `.btn`, `:hover`    | 10         |
| `元素/伪元素`     | `div`, `::before`   | 1          |

---

### **完整示例**
```html
<style>
  /* 后代选择器 */
  div span { color: blue; }

  /* 属性选择器 */
  a[target="_blank"] { color: purple; }

  /* 伪类 */
  button:hover { background: #eee; }

  /* 伪元素 */
  .quote::before { content: "“"; font-size: 24px; }
</style>

<div>
  <span>这段文字是蓝色的</span>
  <a href="#" target="_blank">紫色链接（新窗口打开）</a>
  <button>悬停我变灰</button>
  <p class="quote">这是一段引用</p>
</div>
```
