### **一、Flexbox 例子：导航栏**
**目标**：水平排列导航链接，间距均匀，居中对齐。


```html
<nav class="navbar">
  <a href="#">首页</a>
  <a href="#">产品</a>
  <a href="#">关于</a>
  <a href="#">联系</a>
</nav>

<style>
.navbar {
  display: flex;                /* 启用 Flexbox */
  justify-content: center;      /* 水平居中 */
  gap: 20px;                    /* 链接间距 */
  background: #333;
  padding: 15px;
}

.navbar a {
  color: white;
  text-decoration: none;
}
</style>
```

### <mark style="background: #FFF3A3A6;">**假设场景**</mark>

- **容器宽度**: `600px`
- **3 个元素**，每个 `80px`
- **`gap: 20px`**
- **`justify-content: space-around]]`**

<mark style="background: #FFF3A3A6;">如果代码中同时有gap和space-around,那么首先把gap的值添加到两个元素之间，然后全部宽度-所有gap的值-元素本身的宽度 /[（元素数量-1）*2+2]*2就是元素之间的距离
其中元素数量-1是元素之间的间隙数，*2是因为space-around属性值两边元素靠墙的距离是两个元素之间距离的1/2，所以方便计算把两边元素靠墙的那个距离设为1，这也是为什么之后+2，这是总份数，然后除总份数就是每份的多少，
然后再乘2就是两个元素之间的距离。</mark>


---

**计算逻辑**

1. **总固定占用空间**：
    
    - 元素宽度总和：`3 × 80px = 240px`
    - `gap` 总和：`2 × 20px = 40px`
    - **已用空间** = `240px + 40px = 280px`
2. **剩余空间** = `600px - 280px = 320px`
    
3. **按 6 份分配剩余空间**（两端各 1 份，中间各 2 份）：
    
    - **每份** = `320px / 6 ≈ 53.33px`
    - **两端间隙** = `53.33px`
    - **中间间隙** = `53.33px × 2 ≈ 106.67px`
4. **最终间隙**：
	- **左边第一个元素与容器左端** = `53.33px`
	- **元素之间** = `20px（gap） + 106.67px（space-around） ≈ 126.67px`
    - **右边最后一个元素与容器右端** = `53.33px`
**效果**：  
四个链接水平居中排列，间距 `20px`，背景深灰色。

### **二、Flexbox 例子：垂直居中卡片**
**目标**：在屏幕中央垂直和水平居中一个卡片。
[[align-items的常用值]]
[[align-items(只适用于flex和grid布局)]]
<mark style="background: #FFF3A3A6;">(包含了height:100vh的讲解)</mark>
```html
<div class="container">
  <div class="card">我是卡片</div>
</div>

<style>
.container {
  display: flex;
  justify-content: center;     /* 水平居中 */
  align-items: center;         /* 垂直居中 */
  height: 100vh;               /* 容器高度占满整个视口 */
}

.card {
  width: 200px;
  height: 200px;
  background: lightblue;
  text-align: center;
  line-height: 200px;
}
</style>
```

**效果**：  
浅蓝色卡片在页面正中央。

### **三、Grid 例子：网格图库**
**目标**：创建 3 列等宽网格，自适应间距。

```html
<div class="gallery">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>

<style>
.gallery {
  display: grid;
  grid-template-columns: repeat(3, 1fr);  /* 3 列等宽 */
  gap: 10px;                              /* 网格间距 */
}

.item {
  background: lightcoral;
  padding: 20px;
  text-align: center;
}
</style>
```

**效果**：  
6 个方块排列成 3 列，自适应宽度，间距 `10px`。

### **四、Grid 例子：复杂网页布局**
**目标**：创建头部、侧边栏、主体内容、底部的布局。

```html
<div class="layout">
  <header>头部</header>
  <aside>侧边栏</aside>
  <main>主体内容</main>
  <footer>底部</footer>
</div>

<style>
.layout {
  display: grid;
  grid-template-columns: 200px 1fr;      /* 侧边栏 200px，主体自适应 */
  grid-template-rows: 80px 1fr 100px;    /* 头部 80px，主体自适应，底部 100px */
  height: 100vh;
  gap: 10px;
}

header {
  grid-column: 1 / -1;   /* 横跨所有列 */
  background: lightblue;
}

aside {
  background: lightgreen;
}

main {
  background: lightyellow;
}

footer {
  grid-column: 1 / -1;   /* 横跨所有列 */
  background: lightpink;
}
</style>
```

### **总结对比**
| **功能**          | **Flexbox**                          | **Grid**                              |
|-------------------|--------------------------------------|---------------------------------------|
| **方向**          | 一次处理一行或一列（一维）           | 同时处理行和列（二维）                |
| **适用场景**      | 导航栏、卡片列表、垂直居中           | 网页整体布局、复杂网格                |
| **关键属性**      | `flex-direction`, `justify-content`  | `grid-template-columns`, `grid-gap`   |
| **合并单元格**    | 不支持                               | 支持（`grid-column`/`grid-row`）      |
