### **`justify-content` 是干啥的？**
想象你有一张桌子（容器），上面要摆几个杯子（子元素）。  
**`justify-content`** 就是决定这些杯子在桌子上 **横着怎么摆** 的魔法指令！

---

### **常用指令（属性值）**  
| 指令                | 效果                               | 示意图                          |
|---------------------|----------------------------------|--------------------------------|
| `flex-start`        | 杯子都挤到 **左边**（默认）       | [🍵][🍵][🍵]___________         |
| `flex-end`          | 杯子都挤到 **右边**               | ___________[🍵][🍵][🍵]         |
| `center`            | 杯子挤在 **正中间**               | ______[🍵][🍵][🍵]______        |
| `space-between`     | 杯子 **均匀分开**，头尾贴边       | [🍵]____[🍵]____[🍵]           |
| `space-around`      | 杯子周围留 **等距空间**           | _[🍵]__[🍵]__[🍵]_             |
| `space-evenly`      | 所有空隙 **完全相等**（包括头尾） | _[🍵]_[_🍵]_[_🍵]_             |

### **代码例子**  
```html
<div class="桌子">
  <div class="杯子">1</div>
  <div class="杯子">2</div>
  <div class="杯子">3</div>
</div>

<style>
.桌子 {
  display: flex;
  justify-content: center; /* 改这里看效果！ */
  border: 2px solid black;
  width: 300px;
}
.杯子 {
  width: 50px;
  height: 50px;
  background: lightblue;
}
</style>
```

---

### **一句话总结**  
`justify-content` 就是控制你的杯子（子元素）在桌子（容器）里 **横着怎么摆** 的开关！
