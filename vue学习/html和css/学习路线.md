

以下是一份详细、分阶段的HTML5+CSS从入门到精通学习路线，包含核心知识点、工具和实践建议：

---
### **第一阶段：基础入门（2-4周）**
**目标**：掌握HTML/CSS基础语法，能完成简单静态页面开发

1. **HTML基础**
   - HTML文档结构：`<!DOCTYPE>`、`<html>`、`<head>`、`<body>`
   - 常用标签：
     - 文本标签：`<h1>-<h6>`, `<p>`, `<span>`, `<em>`, `<strong>`
     - 列表：`<ul>`、`<ol>`、`<li>`
     - 媒体：`<img>`（srcset/sizes属性）、`<video>`、`<audio>`
     - 表单：`<form>`、`<input>`（类型扩展）、`<textarea>`、`<label>`
   - 语义化标签：`<header>`, `<nav>`, `<article>`, `<section>`, `<footer>`

2. **CSS基础**
   - 选择器：基本/组合/属性选择器、伪类（`:hover`）、伪元素（`::before`）
   - 盒模型：`width/height`、`padding`、`margin`、`border`
   - 常用属性：
     - 文本：`color`、`font-size`、`line-height`、`text-align`
     - 背景：`background-image`、`background-size`、渐变
     - 布局：`display`（block/inline/inline-block）

3. **实践项目**
   - 创建个人简介页（包含图文排版）
   - 制作简单的联系表单
   - 实现新闻列表页

**推荐资源**：
- MDN Web Docs - HTML Basics
- 《HTML与CSS设计与构建网站》（Jon Duckett）
- freeCodeCamp响应式网页设计认证（前4个模块）

---
### **第二阶段：布局进阶（3-5周）**
**目标**：掌握现代网页布局技术，实现响应式设计

1. **Flexbox布局**
   - 容器属性：`flex-direction`、`justify-content`、`align-items`
   - 项目属性：`flex-grow`、`flex-shrink`、`order`

2. **CSS Grid**
   - 网格容器：`grid-template-columns`、`grid-template-rows`
   - 网格项：`grid-column`、`grid-row`
   - 高级特性：auto-fit/minmax()、网格命名区域

3. **响应式设计**
   - 媒体查询：`@media`（分辨率、设备方向）
   - 移动优先原则
   - 相对单位：`rem`、`vw/vh`、`clamp()`

4. **实践项目**
   - 实现圣杯布局（Holy Grail Layout）
   - 开发响应式产品展示页
   - 创建杂志式多栏排版

**推荐工具**：
- Chrome DevTools布局调试
- CSS Grid Generator（可视化生成器）
- Responsively（多设备预览工具）

---
### **第三阶段：高级技巧（4-6周）**
**目标**：掌握CSS高级特性和工程化实践

1. **CSS预处理器**
   - Sass基础：变量、嵌套、mixin、模块化
   - 自动化编译配置（Webpack/Gulp）

2. **动画与过渡**
   - `transition`：贝塞尔曲线、多属性过渡
   - `@keyframes`动画：复杂时间轴控制
   - CSS性能优化：`will-change`、硬件加速

3. **现代CSS特性**
   - 自定义属性（CSS Variables）
   - 函数：`calc()`、`var()`、`rgb()`
   - 过滤器：`backdrop-filter`、`clip-path`

4. **浏览器兼容性**
   - 特性检测：`@supports`
   - 渐进增强策略
   - 使用PostCSS/Autoprefixer

**实践项目**：
- 实现交互式产品轮播图
- 创建带动画效果的仪表盘
- 开发主题切换功能（深色/浅色模式）

---
### **第四阶段：工程化实战（4-8周）**
**目标**：掌握大型项目开发流程和优化技巧

1. **CSS架构**
   - BEM命名规范
   - CSS方法论：OOCSS、ITCSS
   - CSS-in-JS（可选学习）

2. **性能优化**
   - 关键CSS提取
   - 字体优化：`font-display`、子集化
   - 图片优化：WebP格式、懒加载

3. **工具链**
   - 构建工具：Webpack/Vite配置
   - CSS模块化：CSS Modules
   - 代码检查：Stylelint配置

4. **实践项目**
   - 重构复杂电商首页
   - 开发响应式框架（类似Bootstrap简化版）
   - 完成Web性能审计（使用Lighthouse）

**推荐资源**：
- 《CSS权威指南》（Eric Meyer）
- CSS-Tricks完整教程
- Google Web Fundamentals - CSS篇

---
### **第五阶段：持续精通**
1. **深入浏览器原理**
   - 渲染流水线：重绘 vs 重排
   - 合成层优化
   - 新一代布局：Houdini API

2. **前沿技术探索**
   - 容器查询（Container Queries）
   - 层叠上下文（Layer）
   - 逻辑属性（Logical Properties）

3. **参与社区**
   - 关注W3C规范草案
   - 参与开源CSS框架贡献
   - 定期阅读：Smashing Magazine、CSS Weekly

**持续练习建议**：
- 每周模仿一个优秀网站（如Awwwards获奖作品）
- 参与FreeCodeCamp等平台的挑战任务
- 构建个人组件库（记录可复用样式方案）

---
### **学习工具推荐**
- **编辑器**：VS Code（安装Emmet、Prettier插件）
- **练习平台**：CodePen、JSFiddle
- **调试工具**：Chrome DevTools（重点关注：Computed Styles、Grid/Flex Inspector）
- **设计协同**：Figma（学习设计稿转CSS实现）

---
坚持每天2小时系统学习+实战编码，约4-6个月可达高级水平。建议从第二个月开始参与真实项目（如GitHub开源项目），通过实际问题驱动深入学习。
