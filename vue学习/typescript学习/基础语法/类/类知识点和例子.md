

以下是 TypeScript 中 **类（Class）** 的详细知识点和示例，涵盖基础语法、继承、访问控制等核心概念：

---

### **1. 类的基本定义**
类用于定义对象的模板，包含属性和方法。
```typescript
class Person {
  // 属性
  name: string;
  age: number;

  // 构造函数
  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }

  // 方法
  greet() {
    console.log(`Hello, I'm ${this.name}!`);
  }
}

// 创建实例
const alice = new Person("Alice", 25);
alice.greet(); // "Hello, I'm Alice!"
```

---

### **2. 类的继承（`extends`）**
子类继承父类的属性和方法，并可通过 `super` 调用父类构造函数和方法。
```typescript
class Student extends Person {
  grade: number;

  constructor(name: string, age: number, grade: number) {
    super(name, age); // 调用父类构造函数
    this.grade = grade;
  }

  study() {
    console.log(`${this.name} is studying in grade ${this.grade}.`);
  }
}

const bob = new Student("Bob", 18, 12);
bob.greet(); // "Hello, I'm Bob!"（继承自父类）
bob.study(); // "Bob is studying in grade 12."
```

---

### **3. 访问修饰符**
控制属性和方法的可访问性：

| 修饰符       | 说明                                                                 |
|--------------|----------------------------------------------------------------------|
| `public`     | 默认值，可在任何地方访问                                             |
| `private`    | 仅能在类内部访问                                                     |
| `protected`  | 可在类内部和子类中访问                                               |
| `readonly`   | 只读属性，初始化后不可修改（可结合其他修饰符使用，如 `public readonly`） |

#### **示例**
```typescript
class Animal {
  public name: string;
  private secret: string = "I'm a secret!";
  protected age: number;

  constructor(name: string, age: number) {
    this.name = name;
    this.age = age;
  }

  // 公共方法访问私有属性
  revealSecret() {
    console.log(this.secret);
  }
}

class Dog extends Animal {
  bark() {
    console.log(`${this.name} is barking!`);
    // console.log(this.secret); // ❌ 错误：private 属性无法在子类访问
    console.log(this.age); // ✅ 允许访问 protected 属性
  }
}

const dog = new Dog("Buddy", 3);
dog.revealSecret(); // "I'm a secret!"
```

### **4. 参数属性**
在构造函数参数中直接定义并初始化属性（简化代码）。
```typescript
class User {
  constructor(
    public name: string,
    private age: number,
    readonly id: string
  ) {
    // 参数属性自动初始化
  }
}

const user = new User("Alice", 25, "A001");
console.log(user.name); // "Alice"
```

---

### **5. 静态成员**
静态成员属于类本身，而非实例。通过 `static` 关键字定义。
```typescript
class MathUtils {
  static PI = 3.14159;

  static circleArea(radius: number) {
    return this.PI * radius * radius;
  }
}

console.log(MathUtils.PI); // 3.14159
console.log(MathUtils.circleArea(5)); // 78.53975
```

---

### **6. 抽象类（`abstract`）**
抽象类不能被实例化，只能被继承。抽象方法必须由子类实现。
```typescript
abstract class Shape {
  abstract area(): number;

  printArea() {
    console.log(`Area: ${this.area()}`);
  }
}

class Circle extends Shape {
  constructor(private radius: number) {
    super();
  }

  // 必须实现抽象方法
  area(): number {
    return Math.PI * this.radius * this.radius;
  }
}

const circle = new Circle(5);
circle.printArea(); // "Area: 78.53981633974483"
```

### **7. Getter 和 Setter**
通过 `get` 和 `set` 控制对属性的访问。
```typescript
class Temperature {
  private _celsius: number = 0;

  get celsius(): number {
    return this._celsius;
  }

  set celsius(value: number) {
    if (value < -273.15) {
      throw new Error("Invalid temperature!");
    }
    this._celsius = value;
  }

  get fahrenheit(): number {
    return this._celsius * 1.8 + 32;
  }
}

const temp = new Temperature();
temp.celsius = 25; // 调用 setter
console.log(temp.fahrenheit); // 77（调用 getter）
```

---

### **8. 类实现接口**
类可通过 `implements` 实现接口的约束。
```typescript
interface Runnable {
  run(): void;
}

class Robot implements Runnable {
  run() {
    console.log("Robot is running...");
  }
}

const robot = new Robot();
robot.run(); // "Robot is running..."
```

---

### **9. 类与类型别名/接口的区别**
| **特性**       | **类**                          | **接口/类型别名**               |
|----------------|--------------------------------|--------------------------------|
| 实例化         | 可以实例化（`new`）             | 不能实例化                     |
| 构造函数       | 支持                           | 不支持                         |
| 运行时存在     | 是                             | 否（编译后消失）               |
| 方法实现       | 可包含具体实现                 | 仅定义结构                     |

### **总结**
| **核心概念**      | **说明**                                      | **示例**                              |
|-------------------|----------------------------------------------|---------------------------------------|
| 类定义            | 使用 `class` 关键字                           | `class Person { ... }`                |
| 继承              | `extends` + `super`                          | `class Student extends Person { ... }`|
| 访问修饰符        | `public`、`private`、`protected`             | `private secret: string;`             |
| 参数属性          | 简化属性初始化                               | `constructor(public name: string) { ... }` |
| 静态成员          | `static` 定义类级成员                        | `static PI = 3.14;`                   |
| 抽象类            | `abstract` 定义抽象方法和类                  | `abstract class Shape { ... }`        |
| Getter/Setter     | 控制属性访问逻辑                             | `get celsius() { ... }`               |
| 类实现接口        | `implements` 实现接口约束                    | `class Robot implements Runnable { ... }` |

通过合理使用类，可以构建结构清晰、复用性强的代码！ 🚀
