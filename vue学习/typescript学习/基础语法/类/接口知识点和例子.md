以下是 TypeScript 中 **接口（Interface）** 的详细知识点和示例，涵盖定义、扩展、实现等核心概念：

---

### **1. 接口的基本定义**
接口用于定义对象的形状（属性和方法）。
```typescript
interface Person {
  name: string;
  age: number;
  greet(): void; // 方法
}

const alice: Person = {
  name: "Alice",
  age: 25,
  greet() {
    console.log(`Hello, I'm ${this.name}!`);
  },
};

alice.greet(); // "Hello, I'm Alice!"
```

---

### **2. 可选属性**
使用 `?` 定义可选属性。
```typescript
interface User {
  name: string;
  email?: string; // 可选属性
}

const user1: User = { name: "Alice" };
const user2: User = { name: "Bob", email: "bob@example.com" };
```

---

### **3. 只读属性**
使用 `readonly` 定义只读属性，初始化后不可修改。
```typescript
interface Point {
  readonly x: number;
  readonly y: number;
}

const p: Point = { x: 10, y: 20 };
p.x = 5; // ❌ 错误：x 是只读属性
```

---

### **4. 函数类型**
定义函数的形状（参数和返回值）。
<mark style="background: #FFF3A3A6;">这是函数式接口</mark>
[[函数式接口]]
```typescript
interface AddFunction {
  (a: number, b: number): number;
}

const add: AddFunction = (a, b) => a + b;
console.log(add(2, 3)); // 5
```

---

### **5. 索引签名**
定义动态属性名的类型。
```typescript
interface Dictionary {
  [key: string]: number;
}

const scores: Dictionary = {
  math: 90,
  english: 85,
};
```

---

### **6. 接口继承**
使用 `extends` 继承其他接口。
```typescript
interface Person {
  name: string;
  age: number;
}

interface Employee extends Person {
  salary: number;
}

const employee: Employee = {
  name: "Alice",
  age: 25,
  salary: 50000,
};
```

---

### **7. 类实现接口**
使用 `implements` 实现接口的约束。
```typescript
interface Runnable {
  run(): void;
}

class Robot implements Runnable {
  run() {
    console.log("Robot is running...");
  }
}

const robot = new Robot();
robot.run(); // "Robot is running..."
```

---

### **8. 混合类型**
定义对象同时包含函数和属性。
```typescript
interface Counter {
  (start: number): void;
  count: number;
}

function createCounter(): Counter {
  let counter = function (start: number) {
    counter.count = start;
  } as Counter;
  counter.count = 0;
  return counter;
}

const counter = createCounter();
counter(10);
console.log(counter.count); // 10
```

---

### **9. 接口与类型别名**
#### **接口**
- 主要用于定义对象的形状。
- 支持声明合并（同名接口自动合并）。
```typescript
interface User {
  name: string;
}

interface User {
  age: number;
}

const user: User = { name: "Alice", age: 25 };
```

#### **类型别名**
- 可定义任意类型（联合、交叉、基本类型等）。
- 不支持声明合并。
```typescript
type User = {
  name: string;
  age: number;
};
```

---

### **10. 接口的默认值**
接口本身不支持默认值，但可通过类实现默认值。
```typescript
interface Config {
  theme: string;
}

class DefaultConfig implements Config {
  theme = "light"; // 默认值
}

const config = new DefaultConfig();
console.log(config.theme); // "light"
```

---

### **11. 接口的泛型**
使用泛型增强接口的灵活性。
```typescript
interface Box<T> {
  value: T;
}

const numberBox: Box<number> = { value: 10 };
const stringBox: Box<string> = { value: "Hello" };
```

---

### **12. 接口的应用场景**
#### **(1) 定义 API 响应结构**
```typescript
interface ApiResponse<T> {
  status: number;
  data: T;
}

const response: ApiResponse<{ id: number }> = {
  status: 200,
  data: { id: 1 },
};
```

#### **(2) 定义函数参数**
```typescript
interface Options {
  timeout: number;
  retries: number;
}

function fetchData(options: Options) {
  // ...
}

fetchData({ timeout: 5000, retries: 3 });
```
