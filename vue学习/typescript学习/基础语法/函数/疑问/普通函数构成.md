### **普通函数的构成**
一个普通函数由以下部分组成：

1. **`function` 关键字**：用于声明一个函数。
2. **函数名**：函数的标识符，用于调用函数。
3. **参数列表**：函数接收的输入，用括号 `()` 包裹，多个参数用逗号分隔。
4. **参数类型注解**：为参数声明类型（TypeScript 特有）。
5. **返回值类型注解**：为函数返回值声明类型（TypeScript 特有）。
6. **函数体**：用花括号 `{}` 包裹，包含具体的代码逻辑。
7. **`return` 语句**：返回函数的结果（可选，如果函数没有返回值，可以省略）。

---

### **示例代码**
```typescript
function add(a: number, b: number): number {
  return a + b;
}
```

---

### **详细解释**

#### **1. `function` 关键字**
- 用于声明一个函数。
```typescript
function
```

#### **2. 函数名**
- 函数的标识符，用于调用函数。
```typescript
add
```

#### **3. 参数列表**
- 函数接收的输入，用括号 `()` 包裹。
```typescript
(a, b)
```

#### **4. 参数类型注解**
- 为参数声明类型（TypeScript 特有）。
```typescript
a: number, b: number
```

#### **5. 返回值类型注解**
- 为函数返回值声明类型（TypeScript 特有）。
```typescript
: number
```

#### **6. 函数体**
- 用花括号 `{}` 包裹，包含具体的代码逻辑。
```typescript
{
  return a + b;
}
```

#### **7. `return` 语句**
- 返回函数的结果（可选，如果函数没有返回值，可以省略）。
```typescript
return a + b;
```

---

### **完整示例**
```typescript
function add(a: number, b: number): number {
  return a + b;
}

// 调用函数
const result = add(3, 5); // result 的值为 8
```

---

### **无返回值函数**
如果函数没有返回值，可以使用 `void` 类型。
```typescript
function logMessage(message: string): void {
  console.log(message);
}
```

---

### **总结**
普通函数的基本结构如下：
```typescript
function 函数名(参数1: 类型, 参数2: 类型): 返回值类型 {
  // 函数体
  return 返回值;
}
```

通过以上内容，你可以清晰地理解 TypeScript 中普通函数的构成！
