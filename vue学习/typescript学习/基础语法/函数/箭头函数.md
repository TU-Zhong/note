### **1. 基本语法**
箭头函数用 `=>` 定义，语法简洁。

#### **无参数**
```typescript
const greet = (): string => {
  return "Hello!";
};
console.log(greet()); // 输出: Hello!
```

#### **单参数（可省略括号）**
```typescript
const square = (n: number): number => n * n;
console.log(square(3)); // 输出: 9
```

#### **多参数**
```typescript
const add = (a: number, b: number): number => a + b;
console.log(add(2, 3)); // 输出: 5
```

---

### **2. 类型注解**
#### **显式声明参数和返回值类型**
```typescript
const multiply: (a: number, b: number) => number = (a, b) => a * b;
// 或者更简洁的写法
const multiply = (a: number, b: number): number => a * b;
```

#### **自动类型推断**
如果省略类型注解，TypeScript 会根据上下文自动推断类型：
```typescript
const isEven = (n: number) => n % 2 === 0; // 推断返回值为 boolean
```

---

### **3. 函数体与大括号**
#### **单行返回值（隐式返回）**
如果函数体只有一行，可以省略 `return` 和大括号：
```typescript
const double = (n: number): number => n * 2;
```

#### **多行代码（显式返回）**
需要大括号和 `return` 关键字：
```typescript
const logAndDouble = (n: number): number => {
  console.log("Input:", n);
  return n * 2;
};
```

---

### **4. 作用域特性**
#### **箭头函数没有自己的 `this`**
箭头函数的 `this` 继承自外层作用域（与普通函数的关键区别）。

```typescript
class Counter {
  count = 0;
  
  // 普通函数的 this 指向实例
  increment() {
    setTimeout(function() {
      this.count++; // 错误！此时的 this 指向 window（或严格模式下为 undefined）
    }, 1000);
  }

  // 箭头函数的 this 继承自外层
  correctIncrement() {
    setTimeout(() => {
      this.count++; // 正确！this 指向 Counter 实例
    }, 1000);
  }
}
```

---

### **5. 参数默认值**
箭头函数也支持默认参数：
```typescript
const greet = (name: string = "Guest"): string => `Hello, ${name}!`;
console.log(greet()); // 输出: Hello, Guest!
```

---

### **6. 可选参数**
```typescript
const createUser = (name: string, age?: number): string => {
  return age ? `${name} (${age}岁)` : name;
};
console.log(createUser("Alice"));      // 输出: Alice
console.log(createUser("Bob", 25));    // 输出: Bob (25岁)
```

---

### **7. 与普通函数的区别**
| 特性                | 箭头函数                     | 普通函数                 |
|---------------------|-----------------------------|-------------------------|
| `this` 绑定          | 继承外层作用域               | 动态绑定（由调用者决定） |
| 构造函数            | 不能用作构造函数（无 `new`） | 可以                    |
| `arguments` 对象     | 没有 `arguments` 对象        | 有                      |
| 简写语法            | 支持单行隐式返回             | 需要显式 `return`       |

---

### **8. 适用场景**
#### **回调函数**
```typescript
const numbers = [1, 2, 3];
const doubled = numbers.map(n => n * 2); // [2, 4, 6]
```

#### **简化高阶函数**
```typescript
const createAdder = (x: number) => (y: number) => x + y;
const add5 = createAdder(5);
console.log(add5(3)); // 8
```

#### **事件处理**
```typescript
button.addEventListener("click", () => {
  console.log("Button clicked!");
});
```

---

### **9. 泛型箭头函数**
如果需要泛型，需在参数前声明：
```typescript
const identity = <T>(arg: T): T => arg;
console.log(identity<string>("Hello")); // 输出: Hello
```

---

### **总结**
箭头函数是 TypeScript/JavaScript 中的核心语法，具有以下优势：
1. **简洁性**：单行隐式返回可大幅简化代码。
2. **`this` 安全性**：避免 `this` 绑定问题。
3. **函数式编程**：适合与 `map`、`filter`、`reduce` 等方法配合使用。

通过灵活使用箭头函数，可以编写更简洁、更安全的代码！
