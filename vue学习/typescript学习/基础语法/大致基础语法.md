

以下是 TypeScript 的基础语法及示例（不涉及类型系统等语言特性）：

---

### **1. 变量声明**
- `let` 声明可变量，`const` 声明常量。

Q: let age = 25;是不是可以不用分号？
[[let age = 25;是不是可以不用分号？]]

<mark style="background: #FFF3A3A6;">一般为了可读性还是加一下好一点</mark>
```typescript
let age = 25;          // 可重新赋值
const name = "Alice";  // 不可重新赋值
```

---

### **2. 数据类型**
<mark style="background: #FFF3A3A6;">如果变量的初始值足够明确，TypeScript 会自动推断类型，此时可以省略类型声明。</mark>
[[什么时候需要显示声明变量类型]]
- **基本类型**：`string`, `number`, `boolean`, `null`, `undefined`。

```typescript
let title: string = "Hello";
let count: number = 10;
let isActive: boolean = true;
```

---

### **3. 函数定义**
- 常规函数和箭头函数。
[[如何调用已经写好的函数]]
[[错误1]]
[[箭头函数]]
```typescript
// 普通函数
function add(a: number, b: number): number {
  return a + b;
}

// 箭头函数
const multiply = (a: number, b: number): number => a * b;
```

---

### **4. 条件语句**
- `if-else` 和 `switch`。
```typescript
// if-else
if (age > 18) {
  console.log("Adult");
} else {
  console.log("Child");
}

// switch
switch (day) {
  case "Monday":
    console.log("工作日");
    break;
  default:
    console.log("休息日");
}
```

---

### **5. 循环语句**
- `for`, `while`, `do-while`。
```typescript
// for 循环
for (let i = 0; i < 5; i++) {
  console.log(i);
}

// while 循环
let j = 0;
while (j < 3) {
  console.log(j);
  j++;
}
```

---

### **8. 类**
- 使用 `class` 定义。
```typescript
class Animal {
  name: string;

  constructor(name: string) {
    this.name = name;
  }

  speak() {
    console.log(`${this.name} makes a noise.`);
  }
}

const dog = new Animal("Dog");
dog.speak(); // 输出: Dog makes a noise.
```

---

### **9. 模块化**
- 使用 `export` 和 `import`。
```typescript
// math.ts
export function square(n: number): number {
  return n * n;
}

// main.ts
import { square } from "./math";
console.log(square(4)); // 输出 16
```

### **11. 模板字符串**


### **总结**
这些基础语法与 JavaScript 几乎一致，TypeScript 在此基础上增加了静态类型（此处已按需求省略）。通过练习以上示例，可快速掌握基本用法！
