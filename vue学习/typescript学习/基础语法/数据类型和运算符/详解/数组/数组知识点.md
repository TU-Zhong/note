### **一、数组基础**

#### **1. 数组定义**
- **语法**：`元素类型[]` 或 `Array<元素类型>`
<mark style="background: #FFF3A3A6;">这两种数组声明方式的遍历方法是完全通用的！它们只是 TypeScript 中定义数组的两种不同语法，底层都是 JavaScript 数组，因此所有数组方法和遍历方式都可以互换使用。</mark>性能表现也完全一致

```typescript
// 基础类型数组
let numbers: number[] = [1, 2, 3];
let strings: Array<string> = ["A", "B", "C"];

// 混合类型数组（联合类型）
let mixed: (number | string)[] = [1, "Two", 3];
```

#### **2. 空数组与初始化**
```typescript
let empty: number[] = []; // 空数组
let initialized = [1, 2, 3]; // TypeScript 自动推断为 number[]
```

#### **3. 多维数组**
```typescript
let matrix: number[][] = [
  [1, 2],
  [3, 4],
```

### **二、数组操作**

#### **1. 访问元素**
```typescript
let fruits: string[] = ["Apple", "Banana", "Cherry"];
console.log(fruits[0]); // "Apple"（索引从 0 开始）
console.log(fruits[fruits.length - 1]); // "Cherry"（最后一个元素）
```

---

#### **2. 修改元素**
```typescript
fruits[1] = "Blueberry"; // 修改索引 1 的元素
console.log(fruits); // ["Apple", "Blueberry", "Cherry"]
```

---

### **三、常用数组方法**

#### **1. 添加/删除元素**
| 方法                  | 功能             | 示例                                |
| ------------------- | -------------- | --------------------------------- |
| `push(...items)`    | 在末尾添加元素，返回新长度  | `fruits.push("Orange");` → 长度 4   |
| `pop()`             | 删除最后一个元素，返回该元素 | `fruits.pop();` → 返回 "Cherry"     |
| `unshift(...items)` | 在开头添加元素，返回新长度  | `fruits.unshift("Mango");` → 长度 4 |
| `shift()`           | 删除第一个元素，返回该元素  | `fruits.shift();` → 返回 "Apple"    |

---

#### **2. 合并/切割数组**
nums.concat(arr):arr加在nums的最后面
[[splice()函数]]介绍

| 方法                                     | 功能             | 示例                                                  |
| -------------------------------------- | -------------- | --------------------------------------------------- |
| `concat(arr)`                          | 合并数组，返回新数组     | `let newArr = fruits.concat(["Grape"]);`            |
| `slice(start, end)`                    | 截取子数组（含头不含尾）   | `fruits.slice(1, 3);` → ["Banana", "Cherry"]        |
| `splice(start, deleteCount, ...items)` | 删除/替换元素，返回被删元素 | `fruits.splice(1, 1, "Kiwi");` → 删除索引1的元素并插入 "Kiwi" |

#### **3. 遍历数组**
[[map函数]]
[[有没有可以查找特定索引位置的值的函数]]

| 方法                  | 功能                                      | 示例                                      |
|-----------------------|------------------------------------------|------------------------------------------|
| `forEach(callback)`   | 遍历数组元素                              | `fruits.forEach(fruit => console.log(fruit));` |
| `map(callback)`       | 对每个元素执行操作，返回新数组            | `let lengths = fruits.map(f => f.length);` → [5, 6, 6] |
| `filter(callback)`    | 过滤符合条件的元素，返回新数组            | `let longFruits = fruits.filter(f => f.length > 5);` → ["Banana", "Cherry"] |
| `find(callback)`      | 查找第一个符合条件的元素                  | `fruits.find(f => f.startsWith("B"));` → "Banana" |

---

#### **4. 搜索/判断**
[[有没有可以查找特定索引位置的值的函数]]

| 方法                  | 功能                                      | 示例                                      |
|-----------------------|------------------------------------------|------------------------------------------|
| `includes(item)`      | 检查是否包含某元素                        | `fruits.includes("Apple");` → true       |
| `indexOf(item)`       | 返回元素的首次出现索引（无则返回 -1）     | `fruits.indexOf("Banana");` → 1          |
| `some(callback)`      | 判断是否有元素符合条件                    | `fruits.some(f => f === "Apple");` → true|
| `every(callback)`     | 判断是否所有元素符合条件                  | `fruits.every(f => f.length > 3);` → true|

---

#### **5. 转换/计算**
| 方法                  | 功能                                      | 示例                                      |
|-----------------------|------------------------------------------|------------------------------------------|
| `join(separator)`     | 将数组转为字符串（默认逗号分隔）          | `fruits.join("-");` → "Apple-Banana-Cherry" |
| `reduce(callback, initialValue)` | 累积计算结果              | `let sum = [1, 2, 3].reduce((a, b) => a + b, 0);` → 6 |
| `sort(callback)`      | 排序数组（默认按 Unicode 排序）           | `[3, 1, 2].sort((a, b) => a - b);` → [1, 2, 3] |

### **四、TypeScript 特有功能**

#### **1. 类型注解的严格性**
- TypeScript 会检查数组元素的类型：
```typescript
let numbers: number[] = [1, 2, 3];
numbers.push("4"); // ❌ 错误：类型不匹配
```

---

#### **2. 只读数组（`ReadonlyArray`）**
- 禁止修改数组内容：
```typescript
const readOnlyNumbers: ReadonlyArray<number> = [1, 2, 3];
readOnlyNumbers.push(4); // ❌ 错误：push 不存在于 ReadonlyArray
```

---

#### **3. 泛型数组**
- 使用泛型定义复杂类型数组：
```typescript
type User = { name: string; age: number };
let users: User[] = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 },
];
```

### **五、数组 vs 元组**
| **特性**       | **数组**                  | **元组**                |
|----------------|--------------------------|-------------------------|
| 元素类型        | 必须相同                 | 可以不同                |
| 长度           | 可变                     | 固定                    |
| 类型检查        | 检查元素类型             | 检查元素类型和位置      |
| 适用场景        | 同类型数据集合           | 结构化的固定格式数据    |

### **六、实战示例**

#### **示例 1：过滤并转换数据**
```typescript
interface Product {
  id: number;
  name: string;
  price: number;
}

const products: Product[] = [
  { id: 1, name: "Laptop", price: 999 },
  { id: 2, name: "Phone", price: 699 },
];

// 过滤价格低于 700 的产品，并提取名称
const affordableNames = products
  .filter(p => p.price < 700)
  .map(p => p.name);

console.log(affordableNames); // ["Phone"]
```

---

#### **示例 2：求和与平均值**
```typescript
const scores: number[] = [85, 90, 78, 92, 88];
const total = scores.reduce((sum, score) => sum + score, 0);
const average = total / scores.length;

console.log(`总分：${total}，平均分：${average}`); // 总分：433，平均分：86.6
```
