标准化输出是指通过统一的方式格式化或呈现数据，使其易于理解、处理和分享。在 TypeScript/JavaScript 中，标准化输出通常涉及以下技术和方法：

---
[[如何输出不带回车（换行符）]]
### **1. 字符串格式化**
#### **模板字符串**
- 使用反引号 `` ` `` 和 `${}` 插入变量。
```typescript
let name = "Alice";
let age = 25;
console.log(`Name: ${name}, Age: ${age}`); // Name: Alice, Age: 25
```

#### **`padStart` 和 `padEnd`**
- 补齐字符串长度。
```typescript
let num = "5";
console.log(num.padStart(3, "0")); // "005"
console.log(num.padEnd(3, "0"));   // "500"
```

---

### **2. 数字格式化**
#### **`toFixed`**
- 保留指定位数的小数。
```typescript
let num = 3.14159;
console.log(num.toFixed(2)); // "3.14"
```

#### **`Intl.NumberFormat`**
- 国际化数字格式化（如货币、千位分隔符）。
```typescript
let num = 1234567.89;
let formatter = new Intl.NumberFormat('en-US', {
  style: 'currency',
  currency: 'USD',
});
console.log(formatter.format(num)); // "$1,234,567.89"
```

---

### **3. 日期格式化**
#### **`toLocaleDateString`**
- 本地化日期格式。
```typescript
let date = new Date();
console.log(date.toLocaleDateString("en-US")); // "2/26/2025"
console.log(date.toLocaleDateString("zh-CN")); // "2025/2/26"
```

#### **`Intl.DateTimeFormat`**
- 更灵活的日期时间格式化。
```typescript
let date = new Date();
let formatter = new Intl.DateTimeFormat('en-US', {
  year: 'numeric',
  month: 'long',
  day: 'numeric',
});
console.log(formatter.format(date)); // "February 26, 2025"
```

---

### **4. JSON 格式化**
#### **`JSON.stringify`**
- 将对象转换为 JSON 字符串。
```typescript
let data = { name: "Alice", age: 25 };
let jsonString = JSON.stringify(data, null, 2); // 2 表示缩进空格数
console.log(jsonString);
// 输出：
// {
//   "name": "Alice",
//   "age": 25
// }
```

#### **`JSON.parse`**
- 将 JSON 字符串解析为对象。
```typescript
let jsonString = '{"name":"Alice","age":25}';
let data = JSON.parse(jsonString);
console.log(data.name); // "Alice"
```

---

### **5. 日志格式化**
#### **`console` 方法**
- 使用 `console.log`、`console.table` 等格式化输出。
```typescript
let users = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 },
];
console.table(users);
// 输出表格：
// ┌─────────┬─────────┬─────┐
// │ (index) │  name   │ age │
// ├─────────┼─────────┼─────┤
// │    0    │ "Alice" │ 25  │
// │    1    │  "Bob"  │ 30  │
// └─────────┴─────────┴─────┘
```

---

### **6. 颜色格式化（终端）**
#### **ANSI 转义码**
- 在终端中为输出添加颜色。
```typescript
console.log("\x1b[36m%s\x1b[0m", "This is cyan text!"); // 青色文本
```

---

### **7. 表格格式化**
#### **`console.table`**
- 将数组或对象以表格形式输出。
```typescript
let data = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 },
];
console.table(data);
```

---

### **8. 自定义格式化函数**
- 根据需求编写自定义格式化函数。
```typescript
function formatUser(user: { name: string; age: number }): string {
  return `Name: ${user.name.padEnd(10)}, Age: ${user.age.toString().padStart(2)}`;
}

let user = { name: "Alice", age: 25 };
console.log(formatUser(user)); // "Name: Alice     , Age: 25"
```

---

### **总结**
| **方法**              | **用途**                          | **示例**                              |
|-----------------------|-----------------------------------|---------------------------------------|
| 模板字符串            | 动态插入变量                      | `` `Name: ${name}` ``                 |
| `toFixed`             | 保留指定位数小数                  | `3.14159.toFixed(2)` → `"3.14"`       |
| `Intl.NumberFormat`   | 国际化数字格式化                  | `$1,234,567.89`                       |
| `toLocaleDateString`  | 本地化日期格式                    | `"2/26/2025"`                         |
| `JSON.stringify`      | 对象转 JSON 字符串                | `{"name":"Alice","age":25}`           |
| `console.table`       | 表格形式输出                      | 输出表格                              |
| ANSI 转义码           | 终端颜色输出                      | `\x1b[36mCyan Text\x1b[0m`            |

通过以上方法，可以实现数据的标准化输出，使其更易读、易用！ 🚀
