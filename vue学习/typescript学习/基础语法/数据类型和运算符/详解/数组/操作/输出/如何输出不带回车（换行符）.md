在 TypeScript/JavaScript 中，默认的 `console.log` 会在输出后自动添加一个换行符。如果希望输出不带回车，可以使用以下方法：

---

### **1. 使用 `process.stdout.write`（Node.js 环境）**
- `process.stdout.write` 不会自动添加换行符。
```typescript
process.stdout.write("Hello, ");
process.stdout.write("World!");
// 输出：Hello, World!
```

---

### **2. 使用字符串拼接**
- 将所有内容拼接成一个字符串，然后一次性输出。
```typescript
let output = "Hello, " + "World!";
console.log(output); // Hello, World!
```

---

### **3. 使用模板字符串**
- 使用模板字符串将多行内容合并为一行。
```typescript
let name = "Alice";
let age = 25;
console.log(`Name: ${name}, Age: ${age}`); // Name: Alice, Age: 25
```

---

### <mark style="background: #FFF3A3A6;">**4. 使用 `Array.join`**</mark>
- <mark style="background: #FFF3A3A6;">将数组元素拼接成一个字符串。</mark>
<mark style="background: #FFF3A3A6;">join()函数里面的参数是数组中元素的分隔符，如果没有显示写明就是",",写了就是你写的</mark>
```typescript
let parts = ["Hello, ", "World!"];
console.log(parts.join("")); // Hello, World!
```

---

### **5. 使用 `\r` 覆盖当前行**
- 使用 `\r` 返回到行首，覆盖当前行内容。
```typescript
process.stdout.write("Loading...");
setTimeout(() => {
  process.stdout.write("\rDone!     \n"); // \n 用于换行
}, 1000);
```

---

### **6. 使用 `console.log` 的连续调用**
- 在 `console.log` 中传入多个参数，它们会在同一行输出。
```typescript
console.log("Hello,", "World!"); // Hello, World!
```

---

### **总结**
| **方法**                | **适用场景**                        | **示例**                              |
|-------------------------|-------------------------------------|---------------------------------------|
| `process.stdout.write`  | Node.js 环境，无换行输出            | `process.stdout.write("Hello")`       |
| 字符串拼接              | 简单场景，合并多行内容              | `"Hello, " + "World!"`                |
| 模板字符串              | 动态插入变量，合并多行内容          | `` `Hello, ${name}` ``                |
| `Array.join`            | 将数组元素拼接为一行                | `["Hello", "World"].join(" ")`        |
| `\r` 覆盖当前行         | 动态更新当前行内容                  | `process.stdout.write("\rDone!")`     |
| `console.log` 多参数    | 同一行输出多个值                    | `console.log("Hello,", "World!")`     |

通过以上方法，可以实现不带回车的输出！ 🚀
