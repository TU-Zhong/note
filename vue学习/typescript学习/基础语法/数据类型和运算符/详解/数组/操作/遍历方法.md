在 TypeScript 中，使用 `Array<元素类型>` 定义的数组可以通过多种方式进行遍历。以下是常见的遍历方法及示例：

---

### **1. `for` 循环**
- 使用传统的 `for` 循环遍历数组。
```typescript
let numbers: Array<number> = [1, 2, 3];

for (let i = 0; i < numbers.length; i++) {
  console.log(numbers[i]);
}
```

---

### **2. `for...of` 循环**
- 直接遍历数组元素。
```typescript
let fruits: Array<string> = ["Apple", "Banana", "Cherry"];

for (let fruit of fruits) {
  console.log(fruit);
}
```

---

### **3. `forEach` 方法**
- 使用数组的 `forEach` 方法遍历元素。
```typescript
let colors: Array<string> = ["Red", "Green", "Blue"];

colors.forEach((color) => {
  console.log(color);
});
```

---

### **4. `map` 方法**
- 遍历数组并对每个元素执行操作，返回新数组。
```typescript
let numbers: Array<number> = [1, 2, 3];

let doubled = numbers.map((num) => {
  return num * 2;
});

console.log(doubled); // [2, 4, 6]
```

---

### **5. `for...in` 循环**
- 遍历数组的索引（不推荐，主要用于对象）。
```typescript
let animals: Array<string> = ["Cat", "Dog", "Elephant"];

for (let index in animals) {
  console.log(animals[index]);
}
```

---

### **6. `reduce` 方法**
- 遍历数组并累积计算结果。
```typescript
let numbers: Array<number> = [1, 2, 3, 4];

let sum = numbers.reduce((total, num) => {
  return total + num;
}, 0);

console.log(sum); // 10
```

---

### **7. `some` 和 `every` 方法**
- **`some`**：检查是否有元素符合条件。
- **`every`**：检查是否所有元素符合条件。
```typescript
let scores: Array<number> = [85, 90, 78, 92];

let hasPassed = scores.some((score) => score >= 90); // true
let allPassed = scores.every((score) => score >= 60); // true
```

---

### **8. `filter` 方法**
- 遍历数组并过滤符合条件的元素。
```typescript
let numbers: Array<number> = [1, 2, 3, 4, 5];

let evenNumbers = numbers.filter((num) => num % 2 === 0);

console.log(evenNumbers); // [2, 4]
```

---

### **9. `find` 和 `findIndex` 方法**
- **`find`**：查找第一个符合条件的元素。
- **`findIndex`**：查找第一个符合条件的元素的索引。
```typescript
let users: Array<{ name: string; age: number }> = [
  { name: "Alice", age: 25 },
  { name: "Bob", age: 30 },
];

let user = users.find((u) => u.age > 25); // { name: "Bob", age: 30 }
let index = users.findIndex((u) => u.age > 25); // 1
```

---

### **总结**
| **方法**           | **特点**                                | **适用场景**                            |
|--------------------|----------------------------------------|----------------------------------------|
| `for` 循环         | 传统方式，适合需要索引的场景            | 需要精确控制遍历过程时                  |
| `for...of` 循环    | 直接遍历元素，语法简洁                  | 遍历数组元素时                         |
| `forEach`          | 函数式风格，适合简单的遍历操作          | 仅需遍历元素，无需返回值时              |
| `map`              | 对每个元素执行操作，返回新数组          | 需要转换数组元素时                     |
| `reduce`           | 累积计算结果                            | 需要计算总和、平均值等时               |
| `filter`           | 过滤符合条件的元素                      | 筛选数组元素时                         |
| `find`/`findIndex` | 查找符合条件的元素或索引                | 查找特定元素时                         |

通过灵活使用这些方法，可以高效地遍历和操作数组！ 🚀
