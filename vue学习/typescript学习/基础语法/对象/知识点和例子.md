以下是 TypeScript 中对象的详细知识点和示例：

---

### **1. 对象的定义**
- 对象是键值对的集合，键是字符串或 Symbol，值可以是任何类型。
```typescript
let user = {
  name: "Alice",
  age: 25,
  isAdmin: false,
};
```

---

### **2. 对象的类型注解**
- 使用接口（`interface`）或类型别名（`type`）定义对象类型。
```typescript
interface User {
  name: string;
  age: number;
  isAdmin?: boolean; // 可选属性
}

let user: User = {
  name: "Alice",
  age: 25,
};
```

---

### **3. 访问对象属性**
- 使用点语法 `obj.key` 或方括号 `obj["key"]` 访问属性。
```typescript
console.log(user.name); // "Alice"
console.log(user["age"]); // 25
```

---

### **4. 修改对象属性**
- 直接赋值修改属性。
```typescript
user.name = "Bob";
user["age"] = 30;
```

---

### **5. 添加新属性**
- 直接赋值添加新属性。
```typescript
user.isAdmin = true;
user["email"] = "bob@example.com";
```

---

### **6. 删除属性**
- 使用 `delete` 操作符删除属性。
```typescript
delete user.isAdmin;
```

---

### **7. 对象的遍历**
#### **`for...in` 循环**
```typescript
for (let key in user) {
  console.log(`${key}: ${user[key]}`);
}
```

#### **`Object.keys`、`Object.values`、`Object.entries`**
```typescript
console.log(Object.keys(user)); // ["name", "age"]
console.log(Object.values(user)); // ["Bob", 30]
console.log(Object.entries(user)); // [["name", "Bob"], ["age", 30]]
```

---

### **8. 对象的合并**
#### **`Object.assign`**
- 将多个对象合并到一个对象中。
```typescript
let obj1 = { a: 1 };
let obj2 = { b: 2 };
let merged = Object.assign({}, obj1, obj2);
console.log(merged); // { a: 1, b: 2 }
```

#### **扩展运算符 `...`**
```typescript
let merged = { ...obj1, ...obj2 };
console.log(merged); // { a: 1, b: 2 }
```

---

### **9. 对象的深拷贝与浅拷贝**
#### **浅拷贝**
- 只拷贝对象的第一层属性。
```typescript
let shallowCopy = { ...user };
```

#### **深拷贝**
- 递归拷贝对象的所有层级。
```typescript
let deepCopy = JSON.parse(JSON.stringify(user));
```

---

### **10. 对象的方法**
- 对象可以包含函数作为属性值。
```typescript
let calculator = {
  add(a: number, b: number): number {
    return a + b;
  },
};

console.log(calculator.add(2, 3)); // 5
```

---

### **11. 对象的只读属性**
- 使用 `readonly` 修饰符定义只读属性。
```typescript
interface Point {
  readonly x: number;
  readonly y: number;
}

let p: Point = { x: 10, y: 20 };
p.x = 5; // ❌ 错误：x 是只读属性
```

---

### **12. 对象的可选属性**
- 使用 `?` 定义可选属性。
```typescript
interface User {
  name: string;
  age?: number;
}

let user: User = { name: "Alice" }; // age 是可选的
```

---

### **13. 对象的索引签名**
- 允许对象具有动态属性名。
```typescript
interface Dictionary {
  [key: string]: number;
}

let scores: Dictionary = {
  math: 90,
  english: 85,
};
```

---

### **14. 对象的类型断言**
- 手动指定对象类型。
```typescript
let obj = { name: "Alice", age: 25 } as User;
let obj = <User>{ name: "Alice", age: 25 };
```

---

### **15. 对象的解构赋值**
- 从对象中提取属性并赋值给变量。
```typescript
let { name, age } = user;
console.log(name); // "Alice"
console.log(age); // 25
```

---

### **16. 对象的默认值**
- 在解构时为变量提供默认值。
```typescript
let { name, isAdmin = false } = user;
console.log(isAdmin); // false（如果 user.isAdmin 未定义）
```

---

### **17. 对象的 `this` 指向**
- 对象方法中的 `this` 指向调用该方法的对象。
```typescript
let user = {
  name: "Alice",
  greet() {
    console.log(`Hello, ${this.name}!`);
  },
};

user.greet(); // "Hello, Alice!"
```

---

### **18. 对象的原型**
- 使用 `Object.create` 创建基于原型的对象。
```typescript
let person = {
  greet() {
    console.log("Hello!");
  },
};

let user = Object.create(person);
user.greet(); // "Hello!"
```

---

### **总结**
| **特性**          | **说明**                                | **示例**                              |
|-------------------|----------------------------------------|---------------------------------------|
| 定义对象          | 使用 `{}` 或 `Object.create`            | `let obj = { key: value };`           |
| 类型注解          | 使用 `interface` 或 `type`              | `interface User { name: string; }`    |
| 访问属性          | 点语法或方括号                         | `obj.key` 或 `obj["key"]`             |
| 遍历对象          | `for...in`、`Object.keys` 等方法        | `for (let key in obj) { ... }`        |
| 合并对象          | `Object.assign` 或扩展运算符            | `let merged = { ...obj1, ...obj2 };`  |
| 解构赋值          | 提取对象属性                           | `let { name } = user;`                |
| 默认值            | 解构时为变量提供默认值                 | `let { isAdmin = false } = user;`     |
| 只读属性          | 使用 `readonly` 修饰符                  | `readonly x: number;`                 |
| 可选属性          | 使用 `?` 修饰符                        | `age?: number;`                       |
| 索引签名          | 允许动态属性名                         | `[key: string]: number;`              |

通过掌握这些知识点，可以更高效地使用 TypeScript 中的对象！ 🚀
