在 TypeScript 中，对象是一种键值对的集合，键是字符串或 Symbol，值可以是任何类型。TypeScript 允许你显式声明对象的数据类型，以确保对象的属性和值符合预期的类型。以下是对象的语法和显式声明数据类型的方法：

---

### **1. 对象的语法**
对象的基本语法是用花括号 `{}` 包裹键值对，键和值之间用冒号 `:` 分隔，键值对之间用逗号 `,` 分隔。
```typescript
let obj = {
  key1: value1,
  key2: value2,
};
```

---

### **2. 显式声明对象的数据类型**
TypeScript 提供了多种方式来显式声明对象的数据类型。

#### **(1) 使用接口（`interface`）**
- 接口是一种定义对象类型的方式。
```typescript
interface User {
  name: string;
  age: number;
  isAdmin?: boolean; // 可选属性
}

let user: User = {
  name: "Alice",
  age: 25,
};
```

#### **(2) 使用类型别名（`type`）**
- 类型别名也可以定义对象类型。
```typescript
type User = {
  name: string;
  age: number;
  isAdmin?: boolean; // 可选属性
};

let user: User = {
  name: "Alice",
  age: 25,
};
```

#### **(3) 内联类型注解**
- 直接在变量声明时定义对象类型。
```typescript
let user: { name: string; age: number } = {
  name: "Alice",
  age: 25,
};
```
