扩展运算符（`...`）是 JavaScript/TypeScript 中的一个强大语法，它可以将数组、对象或函数参数展开或合并。以下是它的详细用法和示例：

---

### **1. 扩展运算符的基本功能**
- **展开**：将数组或对象拆分为单个元素。
- **合并**：将多个数组或对象合并为一个。

---

### **2. 扩展运算符的用法**

#### **(1) 展开数组**
- 将数组展开为单独的元素。
```typescript
let numbers = [1, 2, 3];
let newNumbers = [...numbers, 4, 5];

console.log(newNumbers); // [1, 2, 3, 4, 5]
```

---

#### **(2) 合并数组**
- 将多个数组合并为一个。
```typescript
let arr1 = [1, 2];
let arr2 = [3, 4];
let merged = [...arr1, ...arr2];

console.log(merged); // [1, 2, 3, 4]
```

---

#### **(3) 展开对象**
- <mark style="background: #FFF3A3A6;">将对象展开为单独的键值对。</mark>
```typescript
let person = { name: "Alice", age: 25 };
let updatedPerson = { ...person, city: "New York" };

console.log(updatedPerson); // { name: "Alice", age: 25, city: "New York" }
```

---

#### **(4) 合并对象**
- 将多个对象合并为一个（同名属性会覆盖）。
```typescript
let obj1 = { a: 1, b: 2 };
let obj2 = { b: 3, c: 4 };
let merged = { ...obj1, ...obj2 };

console.log(merged); // { a: 1, b: 3, c: 4 }
```

---

#### **(5) 函数参数展开**
- 将数组展开为函数参数。
```typescript
function add(a: number, b: number, c: number) {
  return a + b + c;
}

let numbers = [1, 2, 3];
console.log(add(...numbers)); // 6
```

---

#### **(6) 剩余参数**
- 将函数参数收集为数组。
[[reduce()函数]]
```typescript
function sum(...numbers: number[]) {
  return numbers.reduce((total, num) => total + num, 0);
}

console.log(sum(1, 2, 3, 4)); // 10
```

---

### **3. 扩展运算符的注意事项**
1. **对象扩展是浅拷贝**：
   - 扩展运算符只会复制对象的第一层属性，嵌套对象仍然是引用。
   ```typescript
   let obj1 = { a: 1, nested: { b: 2 } };
   let obj2 = { ...obj1 };
   obj2.nested.b = 3;
   console.log(obj1.nested.b); // 3（obj1 也被修改了）
   ```
2. **数组扩展会创建新数组**：
   - 扩展运算符会创建新的数组实例。
   ```typescript
   let arr1 = [1, 2];
   let arr2 = [...arr1];
   arr2.push(3);
   console.log(arr1); // [1, 2]（arr1 未被修改）
   ```

---

### **4. 扩展运算符的常见场景**

#### **(1) 复制数组或对象**
```typescript
let arr = [1, 2, 3];
let copyArr = [...arr];

let obj = { a: 1, b: 2 };
let copyObj = { ...obj };
```

#### **(2) 添加新属性**
```typescript
let person = { name: "Alice" };
let updatedPerson = { ...person, age: 25 };
```

#### **(3) 覆盖属性**
```typescript
let defaults = { theme: "light", fontSize: 12 };
let userSettings = { ...defaults, fontSize: 14 };
```

#### **(4) 动态参数传递**
```typescript
function logArguments(...args: any[]) {
  console.log(args);
}

logArguments(1, "Hello", true); // [1, "Hello", true]
```

---

### **总结**
| **场景**             | **用法**                               | **示例**                              |
|----------------------|----------------------------------------|---------------------------------------|
| 展开数组             | `[...arr]`                            | `let newArr = [...arr];`              |
| 合并数组             | `[...arr1, ...arr2]`                  | `let merged = [...arr1, ...arr2];`    |
| 展开对象             | `{ ...obj }`                          | `let newObj = { ...obj };`            |
| 合并对象             | `{ ...obj1, ...obj2 }`                | `let merged = { ...obj1, ...obj2 };`  |
| 函数参数展开         | `func(...arr)`                        | `log(...args);`                       |
| 剩余参数             | `function func(...args)`              | `function sum(...numbers)`            |

扩展运算符是简化代码、提高开发效率的利器！ 🚀
