---

### **一、`reduce` 函数的基本参数**
```typescript
数组.reduce(回调函数, 初始值);
```
- **回调函数**：定义如何合并每个元素到最终结果。
- **初始值**：累加的起点值（如果不传初始值，会默认用数组第一个元素作为初始值）。

---

### **二、执行步骤分解**
#### **步骤 1：初始化**
- `初始值` 是 `0`，所以第一次循环开始前，`res = 0`。

| **循环次数** | **当前 res** | **当前 num** | **操作**          | **新的 res** |
|--------------|---------------|---------------|-------------------|-------------|
| 初始化       | `0`           | -             | -                 | -           |

---

#### **步骤 2：第一次循环**
- `num` 是数组第一个元素 `1`。
- 执行 `res + num` → `0 + 1 = 1`，新的 `res = 1`。

| **循环次数** | **当前 res** | **当前 num** | **操作**          | **新的 res** |
|--------------|---------------|---------------|-------------------|-------------|
| 第一次循环   | `0`           | `1`           | `0 + 1 = 1`       | `1`         |

---

#### **步骤 3：第二次循环**
- `num` 是数组第二个元素 `2`。
- 执行 `res + num` → `1 + 2 = 3`，新的 `res = 3`。

| **循环次数** | **当前 res** | **当前 num** | **操作**          | **新的 res** |
|--------------|---------------|---------------|-------------------|-------------|
| 第二次循环   | `1`           | `2`           | `1 + 2 = 3`       | `3`         |

---

#### **步骤 4：第三次循环**
- `num` 是数组第三个元素 `3`。
- 执行 `res + num` → `3 + 3 = 6`，新的 `res = 6`。

| **循环次数** | **当前 res** | **当前 num** | **操作**          | **新的 res** |
|--------------|---------------|---------------|-------------------|-------------|
| 第三次循环   | `3`           | `3`           | `3 + 3 = 6`       | `6`         |

---

#### **最终结果**
- 数组遍历完成，返回最终的 `res = 6`。

---

### **三、关键问题解答**
#### **1. `res` 会保存吗？**
- **是的！** `res` 是累加器（accumulator），每次循环的返回值会作为下一次循环的 `res`。
- 例如：
  - 第一次循环返回 `1` → 第二次循环的 `res` 是 `1`。
  - 第二次循环返回 `3` → 第三次循环的 `res` 是 `3`。

#### **2. `num` 会更新吗？**
- **不会！** `num` 不是变量，而是每次循环时当前处理的数组元素。
- `num` 的值来源于数组元素按顺序依次传递：
  - 第一次循环的 `num` → 第一个元素 `1`。
  - 第二次循环的 `num` → 第二个元素 `2`。
  - 依此类推。

#### **3. `reduce` 的执行过程**
- **遍历顺序**：严格按数组顺序从左到右处理元素。
- **执行次数**：如果有初始值，执行次数等于数组长度；如果没有初始值，执行次数为数组长度减一。

---

### **四、无初始值的特殊情况**
假设不传初始值：
```typescript
let numbers = [1, 2, 3];
let sum = numbers.reduce((res, num) => res + num); // 初始值为数组第一个元素 1
```
执行步骤：
| **循环次数** | **当前 res** | **当前 num** | **操作**          | **新的 res** |
|--------------|---------------|---------------|-------------------|-------------|
| 初始化       | `1`           | -             | -                 | -           |
| 第一次循环   | `1`           | `2`           | `1 + 2 = 3`       | `3`         |
| 第二次循环   | `3`           | `3`           | `3 + 3 = 6`       | `6`         |

最终结果相同，但空数组会报错！

---

### **五、注意事项**
1. **必须处理空数组**：
   - 如果数组为空且未提供初始值，`reduce` 会抛出错误。
   - 安全做法：始终提供初始值。
   ```typescript
   let emptyArray: number[] = [];
   let sum = emptyArray.reduce((res, num) => res + num, 0); // 返回 0，不会报错
   ```

2. **回调函数的返回值决定累加器类型**：
   - 如果初始值为 `number`，回调函数必须返回 `number`。
   - 例如，以下代码会报错：
     ```typescript
     [1, 2, 3].reduce((res, num) => res + num.toString(), 0); // ❌ 返回值变为 string
     ```

---

### **总结**
| **步骤**       | **`res`** | **`num`** | **操作**              |
|----------------|-----------|-----------|-----------------------|
| 初始化         | `0`       | -         | -                     |
| 第一次循环     | `0`       | `1`       | `0 + 1 = 1` → 新 `res`|
| 第二次循环     | `1`       | `2`       | `1 + 2 = 3` → 新 `res`|
| 第三次循环     | `3`       | `3`       | `3 + 3 = 6` → 最终结果|

通过 `reduce`，你可以灵活地将数组元素“折叠”成任意类型的值！ 🚀
