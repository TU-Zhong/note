在 TypeScript 中，调用函数的方式与 JavaScript 完全一致。以下是调用函数的基本方法和示例：

---

### **1. 调用普通函数**
- 直接使用函数名加上括号 `()` 来调用函数。
- 如果函数有参数，需要在括号内传入参数。

#### **示例**
```typescript
function greet(name: string): string {
  return `Hello, ${name}`;
}

// 调用函数
const message = greet("Alice");
console.log(message); // 输出: Hello, Alice
```

---

### **2. 调用箭头函数**
- 箭头函数的调用方式与普通函数相同。

#### **示例**
```typescript
const add = (a: number, b: number): number => a + b;

// 调用函数
const result = add(3, 5);
console.log(result); // 输出: 8
```

---

### **3. 调用类中的方法**
- 需要先创建类的实例，然后通过实例调用方法。

#### **示例**
```typescript
class Calculator {
  add(a: number, b: number): number {
    return a + b;
  }
}

// 创建实例并调用方法
const calc = new Calculator();
const result = calc.add(10, 20);
console.log(result); // 输出: 30
```

---

### **4. 调用静态方法**
- 静态方法属于类本身，可以直接通过类名调用，无需创建实例。

#### **示例**
```typescript
class MathUtils {
  static square(n: number): number {
    return n * n;
  }
}

// 调用静态方法
const result = MathUtils.square(4);
console.log(result); // 输出: 16
```

---

### **5. 调用匿名函数**
- 匿名函数可以立即调用（IIFE，立即调用函数表达式）。

#### **示例**
```typescript
const result = (function (a: number, b: number): number {
  return a + b;
})(3, 5);

console.log(result); // 输出: 8
```

---

### **6. 可选参数和默认参数**
- 如果函数有可选参数或默认参数，调用时可以根据需要传入参数。

#### **示例**
```typescript
function greet(name: string, message: string = "Hello"): string {
  return `${message}, ${name}`;
}

// 调用函数
console.log(greet("Alice")); // 输出: Hello, Alice
console.log(greet("Bob", "Hi")); // 输出: Hi, Bob
```

---

### **7. 调用返回函数的函数**
- 如果函数返回另一个函数，可以链式调用。

#### **示例**
```typescript
function createGreeter(message: string) {
  return function (name: string): string {
    return `${message}, ${name}`;
  };
}

// 调用函数
const greeter = createGreeter("Hi");
console.log(greeter("Alice")); // 输出: Hi, Alice
```

---

### **总结**
在 TypeScript 中，调用函数的方式非常简单：
1. 普通函数：`函数名(参数)`
2. 箭头函数：`函数名(参数)`
3. 类方法：`实例.方法名(参数)`
4. 静态方法：`类名.方法名(参数)`
5. 匿名函数：`(function() { ... })()`

希望这些示例能帮助你理解如何在 TypeScript 中调用函数！
