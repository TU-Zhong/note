### 问题 1: `Nums` 数组未初始化
**错误现象**: 在构造函数中直接访问 `this.Nums[i]`，但 `Nums` 数组未被初始化，导致 `TypeError: Cannot set property '0' of undefined`。

```typescript

    let sum = 0;
    for (let i = 0; i < nums.length; i++) {

    }
}
```

### 问题 2: `sumRange` 方法逻辑错误
**错误现象**: `return this.Nums[right] - this.Nums[right]` 导致结果始终为 `0`，这是明显的代码笔误。

**修复方法**: 正确计算区间和应使用前缀和公式：
```typescript
sumRange(left: number, right: number): number {
    const prefixLeft = left > 0 ? this.Nums[left - 1] : 0;
    return this.Nums[right] - prefixLeft;
}
```

**公式解释**：
- 如果 `left === 0`，直接返回 `this.Nums[right]`（因为前缀和到 `right` 的值就是总和）。
- 如果 `left > 0`，返回 `this.Nums[right] - this.Nums[left - 1]`（区间和等于右端点前缀和减去左端点前一位的前缀和）。
