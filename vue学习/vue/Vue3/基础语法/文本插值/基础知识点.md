### 一、基础用法
#### 1. 显示变量
[[这里凭什么可以写成{{}}里面这样的形式？]]
```html
<p>{{ message }}</p> <!-- 直接显示数据 -->
<script>
data() {
  return {
    message: 'Hello Vue3!'
  }
}
</script>
```

#### 2. 显示表达式
```html
<p>计算：{{ 5 + 3 }}</p> <!-- 数学运算 -->
<p>判断：{{ isActive ? '激活' : '未激活' }}</p> <!-- 三元表达式 -->
<p>反转：{{ message.split('').reverse().join('') }}</p> <!-- 字符串操作 -->
```

---

### 二、进阶用法
#### 1. 显示原始HTML
```html
<div v-html="rawHtml"></div> <!-- 注意XSS安全 -->
<script>
data() {
  return {
    rawHtml: '<span style="color:red">红字警告</span>'
  }
}
</script>
```

#### 2. 避免HTML转义
```html
<!-- 默认会自动转义HTML标签 -->
<p>{{ safeHtml }}</p> 
<!-- 输出：<em>斜体</em>（显示为纯文本） -->
```

---

### 三、特殊处理
#### 1. 处理未定义值
```html
<p>{{ user?.address?.street || '暂无地址' }}</p> <!-- 使用可选链操作符 -->
```

#### 2. <mark style="background: #FFF3A3A6;">一次插值（不更新）</mark>
```html
<span v-once>{{ initialValue }}</span> <!-- 后续数据变化不再更新 -->
```

### 四、实用技巧
#### 1. 格式化日期
<mark style="background: #FFF3A3A6;">bug2</mark>
[[2]]
```html
<p>{{ formatDate(timestamp) }}</p> <!-- 通过方法格式化 -->
<script>
methods: {
  formatDate(ts) {
    return new Date(ts).toLocaleDateString()
  }
}
</script>
```

#### 2. 货币格式化
```html
<p>价格：{{ price.toFixed(2) }}元</p> <!-- 保留两位小数 -->
```

### 五、注意事项
#### 1. 禁止使用语句
```html
<!-- 错误示例 -->
{{ var a = 1 }}     <!-- 声明变量 -->
{{ if (ok) { ... } }} <!-- 流程控制 -->
```

#### 2. <mark style="background: #FFF3A3A6;">避免复杂逻辑</mark>
```html
<!-- 不推荐 -->
{{ items.filter(item => item.active).map(item => item.name).join(', ') }}

<!-- 推荐使用计算属性 -->
{{ filteredNames }}
<script>
computed: {
  filteredNames() {
    return this.items.filter(...).map(...)
  }
}
</script>
```

### 六、对比Vue2变化
#### 1. 过滤器替代方案
```html
<!-- Vue2写法（已废弃） -->
<p>{{ price | currency }}</p>

<!-- Vue3替代方案 -->
<p>{{ formatCurrency(price) }}</p> <!-- 使用方法 -->
或
<p>{{ currencyComputed }}</p> <!-- 使用计算属性 -->
```

#### 2. v-text指令对比
```html
<!-- 等效写法 -->
<p v-text="message"></p> <!-- 完全替换元素内容 -->
<p>{{ message }}</p> <!-- 可混合静态内容 -->
```

### 七、实战示例
```html
<div id="app">
  <h1>{{ title }}</h1>
  <p>时间：{{ new Date().toLocaleTimeString() }}</p>
  <p>折扣价：¥{{ (price * 0.8).toFixed(2) }}</p>
  <div v-html="dynamicContent"></div>
  <p v-if="user">欢迎回来，{{ user.name || '游客' }}</p>
</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      title: '商品详情',
      price: 99.99,
      dynamicContent: '<small>限时优惠</small>',
      user: null
    }
  }
}).mount('#app')
</script>
```

### 📚 知识点总结表
| 场景                | 语法                     | 说明                          |
|--------------------|-------------------------|-----------------------------|
| 基本显示             | `{{ data }}`            | 显示文本内容                   |
| 表达式运算           | `{{ 1 + 1 }}`           | 支持JS表达式                   |
| HTML渲染            | `v-html`                | 渲染原始HTML（需谨慎使用）         |
| 一次性插值           | `v-once`                | 仅渲染初始值                    |
| 可选链操作           | `{{ obj?.prop }}`       | 防止访问不存在的属性              |
| 方法调用             | `{{ format(val) }}`     | 可在表达式中调用methods中的方法      |

建议将这些代码复制到HTML文件中实际运行，通过修改数据观察不同效果，可以快速掌握文本插值的各种用法！ 😊
