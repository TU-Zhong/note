

以下是 **Vue3 文本渲染核心知识点** 的详细说明和示例，适合新手快速掌握：

---

### 一、基础文本渲染
#### 1. **插值语法 (Mustache)**
[[基础知识点]]
```html
<p>{{ message }}</p> <!-- 显示 data 中的 message -->
```

#### 2. **v-text 指令**
```html
<p v-text="message"></p> <!-- 等效于 {{ message }} -->
```

**特点对比**：

| 方式        | 是否可以混合静态内容 | 是否自动转义 HTML |
|-----------|------------|-------------|
| `{{ }}`   | ✔️         | ✔️          |
| `v-text`  | ❌          | ✔️          |

---

### 二、动态HTML渲染
#### **v-html 指令**  
用于渲染原始HTML内容：
```html
<div v-html="rawHtml"></div>
```

**示例**：
```javascript
data() {
  return {
    rawHtml: '<span style="color:red">危险内容</span>'
  }
}
```

**注意事项**：
1. <mark style="background: #FFF3A3A6;">禁止渲染用户输入的 HTML（有 XSS 攻击风险）</mark>
2. <mark style="background: #FFF3A3A6;">必要时使用 DOMPurify 等库过滤内容
</mark>
---

### 三、一次性渲染
#### **v-once 指令**  
使元素内容只渲染一次，后续数据变化不再更新：
```html
<p v-once>初始值：{{ count }}</p>
```

**适用场景**：
- 显示初始化后不再变化的数据
- 优化性能（减少不必要的更新）

---

### 四、表达式渲染
#### 1. **支持 JavaScript 表达式**
```html
<p>{{ message.split('').reverse().join('') }}</p> <!-- 字符串反转 -->
<p>{{ user.age > 18 ? '成年' : '未成年' }}</p> <!-- 条件判断 -->
<p>总价：￥{{ (price * quantity).toFixed(2) }}</p> <!-- 计算金额 -->
```

#### 2. **访问全局对象**
```html
<p>当前时间：{{ new Date().toLocaleString() }}</p>
<p>圆周率：{{ Math.PI.toFixed(2) }}</p>
```

---

### 五、最佳实践
#### 1. **优先使用计算属性**
```html
<p>{{ reversedMessage }}</p>
```
```javascript
computed: {
  reversedMessage() {
    return this.message.split('').reverse().join('')
  }
}
```

#### 2. **复杂数据格式化**
```html
<p>{{ formatPrice(total) }}</p>
```
```javascript
methods: {
  formatPrice(value) {
    return '¥' + value.toFixed(2)
  }
}
```

---

### 六、特殊指令
#### **v-pre**  
跳过该元素及其子元素的编译：
```html
<div v-pre>
  {{ 这里的内容不会被编译 }}
  可以快速显示原始 Mustache 标签
</div>
```

---

### 七、综合示例
```html
<div id="app">
  <!-- 基础渲染 -->
  <h1>{{ title }}</h1>

  <!-- 动态HTML -->
  <div v-html="styledText"></div>

  <!-- 一次性渲染 -->
  <p v-once>初始用户数：{{ userCount }}</p>

  <!-- 表达式计算 -->
  <p>注册天数：{{ Math.floor((Date.now() - regTime)/86400000) }} 天</p>

  <!-- 方法格式化 -->
  <p>最近登录：{{ formatDate(lastLogin) }}</p>
</div>

<script>
const { createApp } = Vue

createApp({
  data() {
    return {
      title: '用户中心',
      styledText: '<b>欢迎您</b>',
      userCount: 1000,
      regTime: 1672502400000, // 2023-01-01
      lastLogin: Date.now()
    }
  },
  methods: {
    formatDate(ts) {
      return new Date(ts).toLocaleDateString()
    }
  }
}).mount('#app')
</script>
```

---

### 八、注意要点
1. **避免副作用操作**  
   不要在模板中修改数据（如：`{{ count++ }}`）

2. **表达式复杂度把控**  
   建议单行表达式不超过3个操作

3. **特殊字符转义**  
   使用 `{{ '{' }}` 显示原始大括号：`{{ '{{' }}` → 显示 `{{`

---

### 总结速查表
| 指令/语法          | 作用                  | 典型场景                  |
|------------------|---------------------|-----------------------|
| `{{ }}`          | 基础文本插值             | 显示简单变量、表达式            |
| `v-text`         | 元素文本替换             | 需要完全覆盖元素内容时           |
| `v-html`         | 渲染HTML             | 显示富文本内容（需谨慎）          |
| `v-once`         | 单次渲染              | 静态内容优化性能              |
| `v-pre`          | 跳过编译              | 展示原始模板语法              |
| 计算属性            | 复杂逻辑处理            | 数据转换、过滤、组合            |
| 方法调用            | 动态格式化数据           | 日期/货币格式化              |
