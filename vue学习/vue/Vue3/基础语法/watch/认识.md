watch 监视数据变化
监视 ref,reactive ,函数返回的值，包含上述数据的数组
<mark style="background: #FFF3A3A6;">深度监视</mark> deep
[[watch里面的深度监视是什么意思？]]
第一个参数传入对象时不同的传入方法
[[易错点 第一个参数传入对象时不同的传入方法]]
```typescript
const stopWatch=watch(sum,(newSum,oldValue)=>{  
  console.log('sum变化了', newSum, oldValue);  
  if(newSum>=10){  
    stopWatch();  
  }  
})
```
若修改的是ref定义的对象中的属性，newValue和oldValue都是新值，因为他们是同一个对象
若修改的整个ref定义的对象，newValue是新值，oldVaue是旧值，不是同一个对象了

而reactive包裹的全部都是修改的对象中的值，都是同一个对象。默认开启深度监视，不可关闭
