

以下是 Vue3 中事件处理的详细知识点及示例：

---

### 一、`v-on` 指令基础
**指令**：`v-on` 或简写 `@`
**作用**：监听 DOM 事件并执行代码

#### 1. 基本用法
```html
<template>
  <!-- 直接绑定方法名 -->
  <button v-on:click="handleClick">点击</button>
  
  <!-- 简写形式 -->
  <button @click="handleClick">点击</button>
  
  <!-- 内联表达式 -->
  <button @click="count++">Add Count: {{ count }}</button>
</template>

<script setup>
import { ref } from 'vue';
const count = ref(0);

const handleClick = () => {
  console.log('Button clicked!');
};
</script>
```

---

### 二、事件对象 `$event`
#### 1. 默认传递事件对象
```html
<button @click="handleEvent">传递事件对象</button>
```
```javascript
const handleEvent = (event) => {
  console.log(event.target); // 输出触发事件的 DOM 元素
};
```

#### 2. 显式传递参数 + 事件对象
```html
<button @click="handleArgs(123, $event)">带参数和事件对象</button>
```
```javascript
const handleArgs = (num, event) => {
  console.log(num, event.clientX); // 输出 123 和鼠标 X 坐标
};
```

---

### 三、事件修饰符
#### 1. 常用修饰符
```html
<!-- 阻止默认行为（如表单提交） -->
<form @submit.prevent="onSubmit"></form>

<!-- 阻止事件冒泡 -->
<div @click.stop="doThis"></div>

<!-- 修饰符可链式调用 -->
<a @click.stop.prevent="doThat"></a>

<!-- 只触发一次 -->
<button @click.once="doOnce">只触发一次</button>

<!-- 事件触发目标为元素自身时触发（非子元素） -->
<div @click.self="onSelfClick"></div>

<!-- 使用事件捕获模式（先触发外层元素） -->
<div @click.capture="captureClick"></div>

<!-- 滚动事件的默认行为立即触发（提升性能） -->
<div @scroll.passive="onScroll"></div>
```

---

### 四、按键修饰符
#### 1. 常用按键别名
```html
<!-- 回车键 -->
<input @keyup.enter="submitForm">

<!-- 删除/退格键 -->
<input @keyup.delete="clearInput">

<!-- 退出键（ESC） -->
<input @keyup.esc="closeModal">

<!-- 方向键 -->
<input @keyup.left="moveLeft">
```

#### 2. 系统修饰键（需按下修饰键）
```html
<!-- Ctrl + 点击 -->
<div @click.ctrl="ctrlClick">Ctrl + 点击</div>

<!-- Alt + Enter -->
<input @keyup.alt.enter="specialSubmit">

<!-- 精确匹配（仅按下 Ctrl 时触发） -->
<button @click.ctrl.exact="exactCtrl">精确 Ctrl</button>
```

---

### 五、鼠标按钮修饰符
```html
<!-- 左键点击 -->
<button @click.left="leftClick">左键</button>

<!-- 右键点击（会阻止默认右键菜单） -->
<button @click.right="rightClick">右键</button>

<!-- 中键点击 -->
<button @click.middle="middleClick">中键</button>
```

---

### 六、综合示例
```html
<template>
  <div>
    <!-- 阻止表单提交并记录输入值 -->
    <form @submit.prevent="handleSubmit">
      <input type="text" v-model="inputText" @keyup.enter="addItem">
      <button type="submit">添加</button>
    </form>

    <!-- 右键菜单示例 -->
    <div @click.right.prevent="showContextMenu($event)">
      右键显示菜单
    </div>
  </div>
</template>

<script setup>
import { ref } from 'vue';

const inputText = ref('');
const handleSubmit = () => {
  console.log('提交的值:', inputText.value);
};
const addItem = () => {
  console.log('回车添加:', inputText.value);
};
const showContextMenu = (event) => {
  console.log('鼠标位置:', event.clientX, event.clientY);
  // 自定义右键菜单逻辑
};
</script>
```

---

### 关键点总结
1. **`v-on` 是事件绑定核心指令**，简写 `@` 更常用
2. **事件对象**通过 `$event` 显式传递或默认自动传递
3. **修饰符链式调用**可实现复杂事件控制（如 `.stop.prevent`）
4. **系统修饰键组合**需注意按下顺序（如 `.ctrl.enter`）
5. **`.exact` 修饰符**用于精确控制组合按键
6. **Vue3 废弃了** `keyCode` 支持，推荐使用按键别名
