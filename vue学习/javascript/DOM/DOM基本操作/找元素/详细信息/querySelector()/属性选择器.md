

你提到的 `[checkbox]` 这种写法是 **无效的 CSS 选择器**！以下是关键知识点：

---
[[css选择器有多个匹配条件怎么办]]
### **错误写法解析**
#### **写法示例**：
```css
input[checkbox] {
  /* 样式 */
}
```
#### **浏览器理解**：
- 查找所有 `<input>` 元素，且 **带有名为 `checkbox` 的自定义属性**
- 类似这样：
  ```html
  <input checkbox> <!-- 实际不存在这种写法！ -->
  ```

---

### **正确写法对照**
#### 场景：匹配 `type="checkbox"` 的输入框
| 错误写法           | 正确写法                          | 作用差异                  |
|--------------------|----------------------------------|--------------------------|
| `input[checkbox]`  | `input[type="checkbox"]`         | 前者试图找自定义属性，后者找 type 属性 |
| `input[type=checkbox]` | ✅ 无需引号也可（但推荐加引号） | 匹配标准的复选框输入框     |

---

### **选择器核心规则**
1. **属性选择器必须包含属性名**  
   ```css
   [属性名="值"]
   ```
2. **特殊属性名必须正确对应**  
   - `type`、`id`、`class` 等是 HTML 的**标准属性**，必须明确指定
   - 不能直接用属性值代替属性名

---

### **实际案例验证**
```html
<input type="checkbox">
<input type="text" custom-attr="checkbox">
```

| 选择器                 | 匹配元素        | 原因                                                                 |
|------------------------|----------------|--------------------------------------------------------------------|
| `input[type="checkbox"]` | 第一个输入框     | 正确匹配 `type` 属性为 `checkbox` 的元素                              |
| `input[checkbox]`        | **无匹配**      | 查找的是 `checkbox` 属性，但两个输入框都没有这个属性                     |
| `input[custom-attr="checkbox"]` | 第二个输入框 | 正确匹配自定义属性 `custom-attr` 值为 `checkbox` 的元素                |

---

### **其他类似错误对比**
| 错误写法             | 正确写法                     | 作用差异                                           |
| ---------------- | ------------------------ | ---------------------------------------------- |
| `[submit]`       | `[type="submit"]`        | 前者找 `submit` 属性，后者找 `type` 属性为 `submit`        |
| `div[container]` | `div[class="container"]` | 前者找 `container` 属性，后者找 `class` 属性为 `container` |
| `a[https]`       | `a[href^="https"]`       | 前者找 `https` 属性，后者找 `href` 属性以 `https` 开头的链接    |

---

### **总结**
- **属性选择器的核心是 `属性名=值`**，不能省略属性名！
- `type="checkbox"` 应该用 `[type="checkbox"]` 匹配
- 直接写 `[checkbox]` 会变成匹配不存在的自定义属性 ✅

通过正确使用属性选择器，可以精准操作网页元素！ 🎯
