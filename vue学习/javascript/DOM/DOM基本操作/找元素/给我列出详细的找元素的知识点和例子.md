

以下是 JavaScript 中查找 DOM 元素的 **详细知识点和示例**，涵盖 9 种核心方法，按使用频率排序：

---

### **一、精准定位元素**
#### 1. **`getElementById()`**
- **作用**：通过 `id` 查找唯一元素
- **返回**：单个元素或 `null`
- **特点**：速度最快（浏览器优化）
```javascript
// 查找 id 为 header 的元素
const header = document.getElementById('header');
```

#### 2. **`querySelector()`**
[[querySelector()的参数是什么]]
- **作用**：<mark style="background: #FFF3A3A6;">通过 CSS 选择器查找 **第一个匹配元素**</mark>
- **返回**：单个元素或 `null`
- **灵活场景**：
```javascript
// 通过类名查找第一个匹配项
const btn = document.querySelector('.submit-btn'); 

// 通过属性选择器查找
const emailInput = document.querySelector('input[type="email"]'); 

// 找类名是list的ul的子元素li的第一个子元素
const item = document.querySelector('ul.list > li:first-child');
```

---

### **二、批量查找元素**
#### 3. **`getElementsByClassName()`**
- **作用**：通过类名查找多个元素
- **返回**：`HTMLCollection`（动态集合）
- **注意**：类名前不需要加 `.`
```javascript
// 查找所有有 'active' 类的元素
const activeItems = document.getElementsByClassName('active'); 

// 多类名查找（同时包含两个类）
const special = document.getElementsByClassName('btn primary'); 
```

#### 4. **`getElementsByTagName()`**
- **作用**：通过标签名查找元素
- **返回**：`HTMLCollection`
```javascript
// 查找所有 <img> 标签
const images = document.getElementsByTagName('img');
```

#### 5. **`querySelectorAll()`**
- **作用**：通过 CSS 选择器查找 **所有匹配元素**
- **返回**：`NodeList`（静态快照）
- **常用场景**：
[[querySelector()的参数是什么]]
```javascript
// 查找所有 data-* 属性的元素
const dataItems = document.querySelectorAll('[data-id]'); 

// 复杂选择器
const listItems = document.querySelectorAll('ul#mainList > li:not(.disabled)'); 
```

---

### **三、特殊场景查找**
#### 6. **通过表单属性查找**
```javascript
// 通过 name 属性查找
const username = document.getElementsByName('username')[0]; 

// 查找表单元素
const form = document.forms['loginForm']; // 通过 name
const email = form.elements.email;        // 通过表单内元素的 name
```

#### 7. **伪类选择器**
```javascript
// 查找第一个符合条件的元素
const firstLi = document.querySelector('li:first-child');

// 查找被选中的单选按钮
const checkedRadio = document.querySelector('input[type="radio"]:checked'); 

// 查找无效的表单元素
const invalidInput = document.querySelector('input:invalid'); 
```

#### 8. **相对查找（基于已有元素）**
```javascript
const container = document.getElementById('app');

// 在 container 内查找
const innerBtn = container.querySelector('.btn');
const allDivs = container.getElementsByTagName('div');
```

#### 9. **`matches()` 验证元素**
```javascript
const element = document.querySelector('.item');

// 检查元素是否匹配选择器
if (element.matches('.active.highlight')) { 
  console.log('元素符合条件');
}
```

### **四、集合处理技巧**
#### `HTMLCollection` vs `NodeList`
| 特性                | HTMLCollection                  | NodeList                       |
|---------------------|---------------------------------|--------------------------------|
| **动态性**          | 实时更新（随 DOM 变化）         | 静态快照（查询时状态）         |
| **转换数组**        | `Array.from(collection)`        | `Array.from(nodeList)`         |
| **遍历方法**        | 只能用 `for` 循环               | 支持 `forEach`                 |

#### 示例：转换数组操作
```javascript
// 将 HTMLCollection 转为数组
const btns = Array.from(document.getElementsByClassName('btn')); 

// 使用数组方法操作
btns.forEach(btn => {
  btn.classList.add('highlight');
});
```

---

### **五、选择器最佳实践**
| 场景                   | 推荐方法                                  | 原因                     |
|------------------------|-----------------------------------------|--------------------------|
| **精确查找唯一元素**   | `getElementById()`                     | 性能最优                 |
| **复杂条件查找**       | `querySelector()` / `querySelectorAll` | 选择器强大灵活           |
| **批量相同标签元素**   | `getElementsByTagName()`               | 简洁高效                 |
| **表单元素操作**       | `document.forms` + `elements`          | 语义清晰                 |
| **动态内容操作**       | `querySelectorAll()`                   | 避免动态集合的意外更新   |

---

### **六、高级选择器示例**
#### 1. **层级选择器**
```javascript
// 直接子元素
document.querySelectorAll('ul > li'); 

// 后代元素
document.querySelectorAll('div p'); 
```

#### 2. **属性选择器**
```javascript
// 查找包含 data-role 属性的元素
document.querySelectorAll('[data-role]'); 

// 精确匹配属性值
document.querySelector('[data-id="123"]'); 

// 模糊匹配（开头）
document.querySelector('[class^="btn-"]'); 
```

#### 3. **状态选择器**
```javascript
// 被禁用的按钮
document.querySelector('button:disabled'); 

// 获取焦点的输入框
document.querySelector('input:focus'); 

// 空内容的输入框
document.querySelector('input:placeholder-shown'); 
```
