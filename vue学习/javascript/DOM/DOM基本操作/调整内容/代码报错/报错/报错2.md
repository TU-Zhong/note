### **1. 模块导入路径错误**
**错误信息**：  
`Uncaught TypeError: Failed to resolve module specifier "dompurify". Relative references must start with either "/", "./", or "../".`  

**原因**：  
<mark style="background: #FFF3A3A6;">在浏览器中使用 ES6 模块时，直接通过 `import DOMPurify from "dompurify"` 导入 npm 包会失败，因为浏览器无法解析非相对路径的模块。</mark>

---

### **解决方案**
#### **方法一：使用 CDN 直接导入**
修改导入语句为 CDN 地址（例如通过 [Skypack](https://cdn.skypack.dev/)）：  
```javascript
// 修改导入语句
import DOMPurify from 'https://cdn.skypack.dev/dompurify';
```

#### **方法二：本地安装并打包**
1. 安装 `dompurify`：
   ```bash
   npm install dompurify
   ```
2. 使用构建工具（如 Webpack、Vite）打包代码，确保模块路径正确解析。

---

### **2. 按钮点击事件未正确绑定**
**问题代码**：  
```html
<button>按钮</button>
```
```javascript
addEventListener("click", OClick); // 点击页面任意位置都会触发
```

**问题**：  
事件监听器绑定到了全局 `window`，导致点击页面任何位置都会触发函数。

---

### **解决方案**
为按钮添加 `id` 并指定事件目标：
```html
<button id="myButton">按钮</button> <!-- 添加 id -->
```
```javascript
document.getElementById("myButton").addEventListener("click", OClick); // 仅按钮点击触发
```

#### **JavaScript (`FindByID.js`)**  
```javascript
// 使用 CDN 导入
import DOMPurify from 'https://cdn.skypack.dev/dompurify';

function OClick() {
    // 示例1：消毒后插入内容
    const userInput = '<img src="x" alt="aaa" onerror="alert(1)">';
    const cleanHTML = DOMPurify.sanitize(userInput); // 输出: <img src="x">
    const P0 = document.getElementById("hhh");
    P0.innerHTML = cleanHTML;

    // 示例2：直接插入安全内容
    const P = document.getElementById("test");
    P.innerHTML = "<strong>hello xtu!</strong>";

    // 示例3：插入列表
    const P2 = document.getElementById("test2");
    P2.innerHTML = `
        <ul>
            <li>你好</li>
            <li>你好</li>
            <li>你好</li>
            <li>你好</li>
            <li>好</li>
        </ul>
    `;
}

// 绑定按钮点击事件（非全局）
document.getElementById("myButton").addEventListener("click", OClick);
```
