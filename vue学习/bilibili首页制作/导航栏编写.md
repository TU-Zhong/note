1. 用div固定大小
2. ul 无序列表写导航栏项,默认清除li的样式,然后布局设置成灵活布局
```css
	ul{  
    display: flex;  
    list-style: none;  
    margin:0;  
    padding: 0;  
}
```
1. 里面的li设置成 grid布局，设置align-items : center,让里面的所有元素垂直居中，这个position设不设无所谓
```css
	li{  
    display: grid;  
    position: relative;  
    align-items: center;  
}
```
1. 然后li里面写a标签 ，
```html
a{  
/*display: block;*/  
padding-right: 15px;  
text-decoration: none;  
color: #ede9e9;  
}
```
