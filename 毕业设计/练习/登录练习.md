## 🚀 快速全栈Demo任务书：实现用户登录功能

### 1. 🎯 项目目标

搭建一个最简化的前后端分离项目，实现一个包含“用户名、密码、模拟验证码”的登录界面，并通过API与后端通信，后端连接SQL Server数据库进行验证。

### 2. 🛠️ 技术栈

- **前端：** Vue 3 (使用 Vite 构建) + Axios (用于API请求)
    
- **后端：** Spring Boot + Spring Data JPA
    
- **数据库：** SQL Server
    

### 3. 📅 任务分解

1. **安装工具：**

#### 阶段二：后端开发 (Spring Boot)

1. **创建项目：**
    
    - 访问 [Spring Initializr](https://start.spring.io/)。
        
    - **依赖 (Dependencies)：**
        
        - `Spring Web` (用于构建REST API)
            
        - `Spring Data JPA` (用于数据库操作)
            
        - `SQL Server Driver` (连接SQL Server)
            
        - `Lombok` (可选，简化代码)
            
    - 点击 "Generate" 下载项目压缩包，并用IDEA打开。
        
2. **配置数据库连接：**
    
    - 打开 `src/main/resources/application.properties` 文件。
        
    - 添加SQL Server的连接信息：
        
        Properties
        
        ```
        # SQL Server Connection
        spring.datasource.url=jdbc:sqlserver://localhost:1433;databaseName=login_demo_db;encrypt=false;trustServerCertificate=true;
        spring.datasource.username=sa  # 替换为你的SQL Server用户名
        spring.datasource.password=YourPassword123 # 替换为你的SQL Server密码
        spring.datasource.driver-class-name=com.microsoft.sqlserver.jdbc.SQLServerDriver
        
        # JPA Settings
        spring.jpa.hibernate.ddl-auto=update # 开发时用update，它会自动更新表结构
        spring.jpa.show-sql=true # 在控制台显示执行的SQL
        ```
        
3. **创建实体类 (Entity)：**
    
    - 创建 `com.example.demo.model` 包。
        
    - 创建 `User.java` 类：
        
        Java
        
        ```
        package com.example.demo.model;
        
        import jakarta.persistence.*;
        import lombok.Data;
        
        @Data
        @Entity
        @Table(name = "t_user")
        public class User {
            @Id
            @GeneratedValue(strategy = GenerationType.IDENTITY)
            private Integer id;
            private String username;
            private String password;
        }
        ```
        
4. **创建数据仓库 (Repository)：**
    
    - 创建 `com.example.demo.repository` 包。
        
    - 创建 `UserRepository.java` 接口：
        
        Java
        
        ```
        package com.example.demo.repository;
        
        import com.example.demo.model.User;
        import org.springframework.data.jpa.repository.JpaRepository;
        import java.util.Optional;
        
        public interface UserRepository extends JpaRepository<User, Integer> {
            // Spring Data JPA 会自动实现这个方法
            Optional<User> findByUsername(String username);
        }
        ```
        
5. **创建控制器 (Controller)：**
    
    - 创建 `com.example.demo.controller` 包。
        
    - 为了简化，我们跳过Service层，直接在Controller中处理逻辑（**注意：** 实际项目应有Service层）。
        
    - 创建 `LoginController.java` 类：
        
        Java
        
        ```
        package com.example.demo.controller;
        
        import com.example.demo.model.User;
        import com.example.demo.repository.UserRepository;
        import org.springframework.beans.factory.annotation.Autowired;
        import org.springframework.http.ResponseEntity;
        import org.springframework.web.bind.annotation.*;
        
        import java.util.Map;
        import java.util.Optional;
        
        @RestController
        @RequestMapping("/api")
        @CrossOrigin(origins = "http://localhost:5173") // 关键：允许前端(Vite默认端口)跨域访问
        public class LoginController {
        
            @Autowired
            private UserRepository userRepository;
        
            // 定义一个内部类或DTO来接收登录请求
            public static class LoginRequest {
                public String username;
                public String password;
            }
        
            @PostMapping("/login")
            public ResponseEntity<?> login(@RequestBody LoginRequest loginRequest) {
                Optional<User> userOptional = userRepository.findByUsername(loginRequest.username);
        
                if (userOptional.isEmpty()) {
                    // 404 Not Found (或 401 Unauthorized)
                    return ResponseEntity.status(401).body(Map.of("message", "用户不存在"));
                }
        
                User user = userOptional.get();
        
                // ！！重要安全提示：真实项目中密码不应明文存储和比较，应使用BCrypt等加密
                if (user.getPassword().equals(loginRequest.password)) {
                    // 200 OK
                    return ResponseEntity.ok(Map.of("message", "登录成功", "username", user.getUsername()));
                } else {
                    // 401 Unauthorized
                    return ResponseEntity.status(401).body(Map.of("message", "密码错误"));
                }
            }
        }
        ```
        
6. **运行后端：**
    
    - 运行 `DemoApplication.java` (主类)。
        
    - 你的后端服务将在 `http://localhost:8080` 启动。

#### 阶段三：前端开发 (Vue 3 + Vite)

1. **创建项目：**
    
    - 打开终端，运行Vite创建命令：
        
        Bash

    - 进入项目目录并安装依赖：
        
        Bash
        
        ```
        cd vue-login-demo
        npm install
        npm install axios # 安装axios用于API请求
        ```
        
2. **创建登录组件：**
    
    - 删除 `src/components/HelloWorld.vue`。
        
    - 创建 `src/components/LoginForm.vue` 文件。
        
    
    Code snippet
    
    ```
    <template>
      <div class="login-container">
        <img src="/logo.svg" alt="Logo" class="logo" />
        <h2>系统登录</h2>
    
        <form @submit.prevent="handleLogin" class="login-form">
          <div class="form-group">
            <label for="username">用户名:</label>
            <input type="text" id="username" v-model="form.username" required />
          </div>
          <div class="form-group">
            <label for="password">密 码:</label>
            <input type="password" id="password" v-model="form.password" required />
          </div>
    
          <div class="form-group captcha-group">
            <label for="captcha">验证码:</label>
            <input type="text" id="captcha" v-model="form.captchaInput" required />
            <span class="captcha-code" @click="generateCaptcha">{{ captchaCode }}</span>
          </div>
    
          <button type="submit" class="login-button">登 录</button>
    
          <p v-if="errorMessage" class="error-message">{{ errorMessage }}</p>
          <p v-if="successMessage" class="success-message">{{ successMessage }}</p>
        </form>
      </div>
    </template>
    
    <script setup>
    import { ref, onMounted } from 'vue';
    import axios from 'axios';
    
    // 响应式状态
    const form = ref({
      username: '',
      password: '',
      captchaInput: '',
    });
    const captchaCode = ref('');
    const errorMessage = ref('');
    const successMessage = ref('');
    
    // --- 验证码逻辑 ---
    const generateCaptcha = () => {
      // 生成一个4位随机数
      const randomNum = Math.floor(1000 + Math.random() * 9000);
      captchaCode.value = randomNum.toString();
    };
    
    // 组件挂载时生成第一个验证码
    onMounted(() => {
      generateCaptcha();
    });
    
    // --- 登录逻辑 ---
    const handleLogin = async () => {
      // 重置消息
      errorMessage.value = '';
      successMessage.value = '';
    
      // 1. 客户端验证码校验
      if (form.value.captchaInput !== captchaCode.value) {
        errorMessage.value = '验证码不正确！';
        generateCaptcha(); // 刷新验证码
        form.value.captchaInput = ''; // 清空输入
        return;
      }
    
      // 2. 调用后端API
      try {
        // 注意：这里的URL必须与你的Spring Boot后端地址一致
        const response = await axios.post('http://localhost:8080/api/login', {
          username: form.value.username,
          password: form.value.password,
        });
    
        // 登录成功
        successMessage.value = `${response.data.message}, 欢迎 ${response.data.username}!`;
        // 可以在这里做路由跳转等
    
      } catch (error) {
        // 登录失败
        if (error.response && error.response.data) {
          errorMessage.value = error.response.data.message; // 显示后端返回的错误
        } else {
          errorMessage.value = '登录失败，请检查网络或联系管理员。';
        }
        generateCaptcha(); // 失败后刷新验证码
        form.value.captchaInput = '';
      }
    };
    </script>

      border-radius: 8px;

    }

      display: flex;

    }

      color: red;

    }
    .success-message {
      color: green;
      margin-top: 15px;
    }
    </style>
    ```
