windows PowerShell命令行窗口

1. 通过docker 创建 vue3 项目命令 
	` docker run -it --rm -v "${pwd}:/app" -w /app node:latest npm init vue@latest`
2. 通过docker安装依赖
	`docker run -it --rm -v "${pwd}:/app" -w /app node:latest npm install`
3. 通过docker映射来启动当前的vue应用
	`docker run -it --rm -v "${pwd}:/app" -w /app -p 5173:5173 node:latest npm run dev -- --host`
