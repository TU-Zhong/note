这道题我为你设计了一个经典的**“员工入职信息录入”**场景。

它看似简单，但涵盖了 `v-model` 的 **5 种核心形态**（字符串、数字、布尔值、单选、多选数组）。如果你能把这道题做出来，`v-model` 这一章你就毕业了。

### 📝 题目：员工入职信息录入卡

需求描述：

请在一个页面上制作一个表单，收集新员工的信息。要求所有数据必须双向绑定，即：你在表单里修改，下方的一个 <pre> 区域要实时显示出最新的 JSON 数据。

**具体字段要求：**

1. **姓名 (Name)**: 文本输入框。
    
    - _进阶要求：_ 自动过滤首尾空格 (使用修饰符)。
        
2. **年龄 (Age)**: 数字输入框。
    
    - _进阶要求：_ 绑定的变量必须是 `Number` 类型，不能是 `String` (使用修饰符)。
        
3. **性别 (Gender)**: 单选按钮 (Radio)。
    
    - 选项：男 / 女 / 保密。
        
4. **是否远程办公 (Remote)**: 单个复选框 (Checkbox)。
    
    - 选中为 `true`，不选中为 `false`。
        
5. **掌握技能 (Skills)**: 多个复选框 (Checkbox Group)。
    
    - 选项：Java, Vue, Docker, Linux。
        
    - **难点：** 选中的技能要自动变成一个数组，例如 `['Java', 'Docker']`。
        
6. **所属部门 (Department)**: 下拉菜单 (Select)。
    
    - 选项：研发部, 市场部, 运维部。

### ✅ 参考答案 (代码实现)

如果你卡住了，或者写完了想对答案，请看下面。

Code snippet

```
<script setup>
import { ref } from 'vue'

// 定义一个对象，类似于 Java 中的 UserDTO
const employee = ref({
  name: '',
  age: 25,
  gender: '保密',      // 对应 Radio 的 value
  isRemote: false,    // 对应单个 Checkbox (boolean)
  skills: [],         // ⚠️ 重点：多选框必须绑定到数组上
  department: ''      // 对应 Select 的 value
})
</script>

<template>
  <div class="container">
    <h2>员工入职登记表</h2>

    <div class="form-item">
      <label>姓名：</label>
      <input type="text" v-model.trim="employee.name" placeholder="请输入姓名">
    </div>

    <div class="form-item">
      <label>年龄：</label>
      <input type="number" v-model.number="employee.age">
    </div>

    <div class="form-item">
      <label>性别：</label>
      <label><input type="radio" v-model="employee.gender" value="男"> 男</label>
      <label><input type="radio" v-model="employee.gender" value="女"> 女</label>
      <label><input type="radio" v-model="employee.gender" value="保密"> 保密</label>
    </div>

    <div class="form-item">
      <label>工作性质：</label>
      <label>
        <input type="checkbox" v-model="employee.isRemote"> 申请远程办公
      </label>
    </div>

    <div class="form-item">
      <label>掌握技能：</label>
      <label><input type="checkbox" v-model="employee.skills" value="Java"> Java</label>
      <label><input type="checkbox" v-model="employee.skills" value="Vue"> Vue</label>
      <label><input type="checkbox" v-model="employee.skills" value="Docker"> Docker</label>
      <label><input type="checkbox" v-model="employee.skills" value="Linux"> Linux</label>
    </div>

    <div class="form-item">
      <label>部门：</label>
      <select v-model="employee.department">
        <option disabled value="">请选择部门</option>
        <option value="研发部">研发部</option>
        <option value="市场部">市场部</option>
        <option value="运维部">运维部</option>
      </select>
    </div>

    <div class="preview">
      <h3>实时数据模型 (JSON):</h3>
      <pre>{{ employee }}</pre>
    </div>
  </div>
</template>

<style scoped>
/* 简单的排版，不用太在意 */

```

### 🎯 知识点复盘 (考点解析)

1. **`v-model.trim`**:
    
    - 如果你不加 `.trim`，用户输入 "Tom " (后面带空格)，你的变量就是 "Tom "。
        
    - 加了之后，Vue 自动帮你执行 `String.trim()`。
        
2. **`v-model.number`**:
    
    - HTML 的 `<input type="number">` 实际上返回的是字符串 "25"。
        
    - 加了 `.number`，Vue 帮你自动 `parseFloat()` 转成数字类型。这对 Java 后端接收数据很重要。
        
3. **多选框魔法 (`skills`)**:
    
    - 这是最神奇的地方。如果 `v-model` 绑定的是一个**数组** (`[]`)，Vue 会自动把“勾选”变成 `add`，把“取消勾选”变成 `remove`。
        
    - 如果绑定的不是数组而是布尔值，它就变成了开关（如上面的 `isRemote`）。
