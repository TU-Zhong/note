<script setup>
import { ref } from 'vue'

// 1. 定义一个响应式变量，用来存放输入框的内容

### 🧐 发生了什么？(Java 视角解析)

#### 1. `v-model="name"` 是什么？

它是 Vue 最著名的语法糖。

在 Java Swing 或 JavaFX 这种桌面开发中，你想把一个输入框 (TextField) 和一个 String 变量绑定，通常需要写监听器 (Listener)。

但在 Vue 里，`v-model` 把这两件事合二为一了：

1. **View -> Model**: 当用户在 `<input>` 里打字时，Vue 自动把最新的字赋值给 `name.value`。
    
2. **Model -> View**: 当你在代码里改了 `name.value = "xxx"`，Vue 自动把 `<input>` 里的字改成 "xxx"。
    

#### 2. 怎么验证是“双向”的？

- **正向验证**：你在输入框里打字，下面的 `<span class="highlight">` 会跟着变。
    
- **反向验证**：点击那个“重置为 TuZhong”的按钮。你会发现，虽然你没操作 DOM 也没操作 input 元素，但因为 `name.value` 变了，输入框里的字自己就变了。
