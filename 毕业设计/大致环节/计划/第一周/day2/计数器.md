你可以把文件里**原本的内容全部删掉**，然后复制粘贴下面的代码。

### 核心代码 (计数器)

Code snippet

```vue
<script setup>
// 1. 引入 ref 工具
// (就好比 Java 里 import java.util.List)
import { ref } from 'vue'

// 2. 定义一个响应式变量 count
// ref(0) 相当于把 0 包装成了一个能被监控的对象
const count = ref(0)

// 3. 定义修改数据的方法
function increment() {
  // ⚠️ 重点：在 script 里修改 ref 变量，必须加 .value
  count.value++
  console.log("当前值:", count.value)
}

function decrement() {
  count.value--
}
</script>

<template>
  <div class="container">
    <h1>计数器</h1>
    
    <p class="number">{{ count }}</p>

    <div class="buttons">
      <button @click="decrement">- 减少</button>
      <button @click="increment">+ 增加</button>
    </div>
  </div>
</template>

<style scoped>
/* 5. 简单的样式美化 (可选) */
.container {
  text-align: center;
  margin-top: 50px;
}
.number {
  font-size: 50px;
  font-weight: bold;
  color: #42b883;
}
button {
  margin: 0 10px;
  padding: 10px 20px;
  font-size: 16px;
  cursor: pointer;
}
</style>
```

---

### 知识点拆解（必读）

1. **`import { ref } from 'vue'`**:
    
    - 这是 Vue 的核心魔法。不用 `ref` 定义的变量只是普通变量，变了页面也不会动。用了 `ref`，Vue 才会盯着它。
        
2. **`count.value` (脚本中)**:
    
    - 在 `<script>` 标签里写逻辑时，`count` 其实是一个对象（RefImpl），它的**真值**藏在 `.value` 属性里。所以必须写 `count.value`。
        
3. **`{{ count }}` (模板中)**:
    
    - 在 `<template>` (HTML) 里，Vue 帮你自动“拆包”了，所以**千万不要**写成 `{{ count.value }}`，直接写变量名就行。
        
4. **`@click`**:
    
    - 这是 `v-on:click` 的缩写。意思是“当点击时，执行后面的函数”。
