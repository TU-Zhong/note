正确答案是 **B：128.202.0.0**。

**解析过程如下：**

1. **判断 IP 地址分类**：在标准的“有类网络”中，IP 地址是根据第一段数字来划分的：
    
    - **A 类**：1 ~ 126
        
    - **B 类**：128 ~ 191
        
    - **C 类**：192 ~ 223
        
        题目中的 IP 地址第一段是 `128`，正好落在 128 ~ 191 的区间内，因此它属于 **B 类地址**。
        
2. **确定默认子网掩码**：
    
    - B 类地址的默认子网掩码是 `255.255.0.0`（也就是说，它的前两个字节代表“网络号”，后两个字节代表“主机号”）。
        
3. **计算网络地址**：
    
    计算所属网络的最直接方法，就是将 IP 地址与其子网掩码进行二进制的“按位与”（AND）运算：
    
    - IP 地址：`128.202.99.65`
        
    - 子网掩码：`255.255.0.0`
        
    - **网络地址：`128.202.0.0`** （掩码为 255 的部分保留原值，掩码为 0 的部分清零）
        

在计算机网络基础的考察中，这种“有类 IP 地址”的题目非常经典。
